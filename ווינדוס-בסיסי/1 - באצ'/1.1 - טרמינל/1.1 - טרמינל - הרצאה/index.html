    1.1 - טרמינל - הרצאה

    עץ התיקיות¶

    מהו עץ תיקיות?¶

    במחשב אנו שומרים מידע בקבצים, ותיקיות מאפשרות לנו לארגן ולנהל אותם. כל תיקיה יכולה להכיל מספר קבצים וגם תיקיות נוספות, הנקראות תיקיות משנה (Subdirectories).

    למעשה, כל מערכת הקבצים במחשב מסודרת בצורה היררכית שמזכירה מבנה של עץ:

    • שורש העץ (Root) – התקייה הראשית שממנה הכול מתחיל, ב-Windows מסומן כ- C:\.
    • ענפי העץ – תיקיות בתוך תיקיות שמתחילות מהתקייה הראשית.
    • עלי העץ – הקבצים עצמם.

    כך נוצר מבנה של עץ תיקיות שמאפשר סדר וארגון.

    סייר הקבצים¶

    סייר הקבצים (File Explorer) הוא הכלי הגרפי שמאפשר לנו לגלוש בין הקבצים והתיקיות בצורה נוחה.

    איך לפתוח את סייר הקבצים?¶

    ניתן לפתוח את סייר הקבצים בקלות:

    • לחצו על מקש WinKey + E במקלדת.
    • או חפשו "סייר הקבצים" בתפריט ההתחלה (Start Menu).

    פתחו את סייר הקבצים ועברו בין התיקיות כדי להכיר את מבנה הקבצים במחשב שלכם.

    הטרמינל¶

    מהו הטרמינל?¶

    הטרמינל הוא כלי שמאפשר לנו לנהל את המחשב באמצעות פקודות טקסטואליות במקום interface גרפי. דרכו ניתן לבצע פעולות מתקדמות, כמו ניוד בין תיקיות, ניהול קבצים, הרצת סקריפטים, ועוד.

    פתיחת הטרמינל (CMD - Command Prompt)¶

    1. לחצו על WinKey והקלידו cmd.
    2. לחצו על Enter.

    ייפתח בפניכם המסך הבא:

    זהו הטרמינל, כאן ניתן להקליד פקודות ולבצע פעולות שונות במחשב.

    דוגמה לפקודה בסיסית¶

    פקודת whoami תציג את שם המשתמש המחובר כרגע למחשב:

    whoami

    כאן ניתן לראות שאני מחובר כמשתמש amitp.

    ניוד בין תיקיות בטרמינל¶

    איך לדעת באיזה תיקייה אנחנו נמצאים?¶

    בכל רגע נתון הטרמינל מציג לנו את הנתיב (Path) שבו אנחנו נמצאים בצד שמאל:

    בדוגמה זו, הנתיב הנוכחי הוא:

    C:\Users\amitp

    משמעותו: אנו נמצאים בתוך התיקייה amitp שבתוך Users, שבתוך C:\.

    איך לראות את הקבצים שבתיקייה הנוכחית?¶

    ניתן להשתמש בפקודה dir כדי לראות את כל הקבצים והתיקיות שבתוך הנתיב הנוכחי:

    dir

    איך להיכנס לתיקייה מסוימת?¶

    כדי להיכנס לתיקייה בתוך הנתיב הנוכחי, נשתמש בפקודה cd (Change Directory):

    cd dev

    ואז נריץ שוב dir כדי לראות מה יש בפנים:

    איך לחזור תיקייה אחת אחורה?¶

    אם נרצה לחזור לתיקייה הקודמת, נשתמש ב:

    cd ..

    הinterface טקסטואלי (TUI) מול interface גרפי (GUI)¶

    בטרמינל אין interface גרפי כמו בסייר הקבצים, אלא הכול מבוסס על טקסט. צורת עבודה זו נקראת TUI - Text User Interface (interface משתמש טקסטואלי). לעומת זאת, סייר הקבצים משתמש ב-GUI - Graphical User Interface (interface משתמש גרפי).

    ההבדלים בין TUI ל-GUI¶

    תכונה TUI (טרמינל) GUI (סייר הקבצים)
    דרך שימוש פקודות טקסט לחיצות עכבר ותפריטים
    מהירות פעולה מהיר (למי שמיומן) איטי יותר אך נוח
    גמישות גבוהה מאוד מוגבלת לפיצ'רים הקיימים
    התאמה אישית ניתן להריץ סקריפטים ואוטומציות מוגבל להגדרות קיימות

    חזרה קצרה¶

    • הטרמינל הוא תוכנה שמאפשרת לנו להריץ פקודות לניהול המחשב.
    • נתיב (Path) מראה לנו באיזו תיקייה אנו נמצאים כרגע.
    • cd מאפשר לנו לנוע בין תיקיות.
    • dir מציג את הקבצים שבתיקייה.
    • TUI ו-GUI הם שני סוגים שונים של interfaces משתמש – טקסטואלי לעומת גרפי.

    ככל שתתרגלו יותר שימוש בטרמינל, תוכלו לבצע פעולות מתקדמות במהירות וביעילות!
