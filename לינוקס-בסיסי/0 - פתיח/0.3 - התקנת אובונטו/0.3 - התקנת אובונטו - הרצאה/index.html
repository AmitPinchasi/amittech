    0.3 - התקנת אובונטו - הרצאה

    איך נתקין אובונטו?¶

    • יש לנו כמה אפשרויות להתקין אובונטו, על המחשב האישי שלנו או על מכונה וירטואלית במחשב שלנו.
    • מכיוון שלהתקין אובונטו ישירות על המחשב האישי שלנו, במקום המערכת ההפעלה הנוכחית שלנו יכול להיות מורכב- נעדיף להשתמש באופציה של מכונה וירטואלית.
    • מנהל מכונות וירטואליות היא תוכנה שאנחנו יכולים להתקין במערכת ההפעלה הנוכחית שלנו ודרכה להריץ ״מחשב״ וירטואלי. במכונה הוירטואלית שלנו נתקין אובונטו.
    • אחד ממנהלי המכונות הוירטואליות הפופלרי ביותר הוא Vmware Workstation, או למק: Vmware player. נתקין אותו, ניצור מכונה וירטואלית חדשה ונתקין עלייה Ubuntu

    מדריך להתקנת Ubuntu על VMware Workstation¶

    הנה מדריך מפורט להתקנת Ubuntu על VMware Workstation.

    שלב 1: התקנת VMware Workstation¶

    1. הורדת VMware Workstation:
    2. עבור לאתר הרשמי של VMware כאן.
    3. הורד את גרסת VMware Workstation Community Edition.

    4. אם יש לכם מק, הורידו את vmware player או את virtual box.

    5. התקנת VMware Workstation:

    6. לאחר ההורדה, פתח את קובץ ההתקנה.
    7. לאחר ההתקנה, הפעל את VMware Workstation.

    שלב 2: התקנת Ubuntu על VMware Workstation¶

    1. הורדת קובץ התקנת Ubuntu (ISO):
    2. גש לאתר הרשמי של Ubuntu כאן.
    3. הורד את גרסת ה-ISO העדכנית של Ubuntu. גרסה מומלצת היא Ubuntu Desktop (לשימוש כללי).

    1. יצירת מכונה וירטואלית חדשה:
    2. פתח את VMware Workstation.
    3. לחץ על "Create a New Virtual Machine" במסך הבית.
    4. בחר באפשרות "Typical (recommended)" ולחץ על "Next".
    5. בחר באפשרות "Installer disc image file (iso)" והזן את הנתיב לקובץ ה-ISO שהורדת. לחץ על "Next".


    6. הזן פרטים כגון שם משתמש וסיסמה (שתצטרך להשתמש בהם לאחר התקנת Ubuntu). לחץ על "Next".
    7. בחר שם ותיקייה לאחסון המכונה הווירטואלית ולחץ על "Next".
    8. בחר את גודל הדיסק הקשיח הווירטואלי (מומלץ להקצות לפחות 25 GB) ולחץ על "Next".

    9. בסוף, תוכל לעבור על כל ההגדרות, ולחץ על "Finish" להשלמת התהליך.

    10. התקנת Ubuntu:

    11. לאחר לחיצה על "Finish", המכונה הווירטואלית תתחיל לפעול והתקנת Ubuntu תתחיל.
    12. עקוב אחר ההנחיות על המסך כדי להתקין את Ubuntu במכונה הווירטואלית.
    13. בתהליך ההתקנה, תוכל לבחור את שפת המערכת, לאשר את סוג ההתקנה, לבחור אזור זמן, ולקבוע פרטים נוספים.
    14. לאחר השלמת ההתקנה, Ubuntu יאתחל ויכנס למערכת החדשה שהתקנת.

    שלב 3: הפעלת Ubuntu על VMware Workstation¶

    1. הפעלת המכונה הווירטואלית:
    2. לאחר סיום ההתקנה, תוכל להפעיל את המכונה הווירטואלית מהתפריט הראשי של VMware Workstation.

    3. לחץ על "Power on this virtual machine" כדי להפעיל את Ubuntu.

    4. כניסה למערכת:

    5. לאחר boot המערכת, תתבקש להזין את שם המשתמש והסיסמה שהגדרת במהלך ההתקנה.

    6. הזן את הפרטים הללו ותיכנס למערכת.

    7. התקנת כלים נוספים (VMware Tools):

    8. מומלץ להתקין את VMware Tools במכונה הווירטואלית כדי לשפר את הביצועים והאינטגרציה בין המערכת הווירטואלית למערכת המארחת.
    9. לאחר כניסה ל-Ubuntu, בחר "Install VMware Tools" מהתפריט של VMware Workstation, עקוב אחר ההנחיות שעל המסך כדי להתקין את הכלים.
