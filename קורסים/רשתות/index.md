# קורס רשתות

קורס מקיף שמלמד רשתות מחשבים מהיסודות ועד לרמה מתקדמת. מתחילים מהביטים שרצים בכבל, עוברים דרך כל שכבות מודל ה-OSI, ומגיעים עד ניהול רשתות ארגוניות, אבטחת רשת, ועבודה מעשית עם כלי רשת בלינוקס ובווינדוס.

## מה נלמד בקורס

- **בסיס** - מבוא לרשתות, מושגי יסוד, ואיך מחשבים מתקשרים
- **מודל ה-OSI** - השכבה הפיזית, שכבת הקו ופרוטוקול Ethernet, שכבת הרשת ופרוטוקולי IP ו-ARP, שכבת התעבורה עם TCP ו-UDP
- **שכבת האפליקציה** - סוקטים בפייתון, פרוטוקולי DNS, SSH, HTTP, והצפנות
- **האינטרנט** - בעיית הכתובות, NAT, איך האינטרנט עובד בפועל, ניתוח תעבורה עם Wireshark, פרוקסי, טינול ו-VPN
- **רשתות מנוהלות** - Active Directory, ניהול משתמשים וקבוצות, DNS בווינדוס, אותנטיקציה עם Kerberos, הרצת קוד מרוחק
- **אבטחה ברשת** - ארכיטקטורת DMZ, פיירוולים, VLANs, ניטור רשת, מוצרי אבטחה ושירותים פנימיים
- **רשתות עם לינוקס** - פקודות בסיסיות, ניטור, nc, socat, iptables, tcpdump, ufw
- **רשתות עם ווינדוס** - ncat, netsh, net
- **סיום** - ספריית Scapy לבניית חבילות רשת בפייתון

## פרויקטים

- **פרויקט שכבות** - כתיבת הרצאה מלאה שמסבירה את האינטראקציה בין השכבות
- **שרת HTTP** - בניית שרת HTTP עובד מאפס עם סוקטים
- **פרויקט אינטרנט** - ניתוח ומחקר של תעבורת רשת אמיתית
- **פרויקט לינוקס** - עבודה מעשית עם כלי רשת בסביבת לינוקס
- **פרויקט סיום** - פרויקט מקיף עם Scapy

## למי מתאים הקורס

הקורס מתאים למי שרוצה להבין איך רשתות עובדות - בין אם לצורך IT, פיתוח, DevOps או אבטחת מידע. נדרש ידע בסיסי בטרמינל ובפייתון.

## למה הקורס הזה

- **90 שיעורים** שמכסים את הנושא מקצה לקצה, מביטים ועד Active Directory
- **מעשי** - שני שלישים מהזמן מוקדשים לתרגול, עם כלי רשת אמיתיים
- **לינוקס וווינדוס** - כלי רשת מעשיים בשתי מערכות ההפעלה
- **רשתות ארגוניות** - פרק שלם על Active Directory, דומיינים, GPO ו-Kerberos
- **אבטחת רשת** - DMZ, פיירוולים, VLANs, ניטור, ומוצרי אבטחה
- **תכנות רשתות** - סוקטים בפייתון ו-Scapy למניפולציה של חבילות רשת
- **בסיס לסייבר** - הידע הזה הוא תנאי הכרחי למי שרוצה להתמקצע באבטחת מידע ובדיקות חדירות
