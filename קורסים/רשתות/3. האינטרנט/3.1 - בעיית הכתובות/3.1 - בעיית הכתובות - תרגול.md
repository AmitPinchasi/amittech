## האתר הראשון שלי
- כתוב עמוד html פשוט ותריץ אותו עם שרת http לבחירתך על פורט לבחירתך
	- למדו בעצמכם על איך לכתוב עמוד html פשוט, או העתיקו אחד מהאינטרנט.
	- השרת http יכול להיות שרת הhttp שכתבתם בפייתון, או שימוש בפקודה הפייתונית:
```python
python3 -m http.server 8000
```
כאשר המספר 8000 הוא הפורט שיפתח ושירים את השרת http שלכם.
- בדקו שהשרת http שלכם עובד בפנייה בדפדפן שלכם ל http://127.0.0.1:8000/index.html כמובן אם הקובץ הhtml שלכם הוא index.html והוא נמצא בתקייה שבה הרצתם את שרת הhttp שלכם.

- פתחו את האתר שלכם לאינטרנט באמצעות התחברות לממשק הadmin של הראוטר שלכם, והוסיפו חוק port forwarding לip של המחשב שלכם מהפורט שבחרתם לפורט 80 (הסנטדרטי לhttp).
	- מצאו את הIP של הראוטר שלכם, התחברו לממשק ADMIN שלו, מצאו את הפרטי התחברות ומצאו את העמוד של הport forwarding.
	- לא עובד לכם? יכול להיות שהראוטר שלכם הוא dobule nat- התקשרו לספקית התקשורת שלכם ותשאלו את התמיכה הטכנית שלה.
	- מצאו את הIP של הבית שלכם באמצעות האתר "what is my ip" ונסו להתחבר לאתר שלכם מרשת אחרת, למשל מהטלפון הנייד שלכם, או ממחשב של חבר.
