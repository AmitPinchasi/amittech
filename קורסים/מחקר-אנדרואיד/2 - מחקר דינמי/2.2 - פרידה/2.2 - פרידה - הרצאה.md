## הכלי frida
פרידה היא כלי מטורף שמאפשר לנו לבצע ״instrumentation״ על תוכנות.
מכשור או ״instrumentation״ היא קבוצה של כלים, שמאפשרת לנו לנתח תוכנות בזמן ריצה- לערוך זכרון, לקרוא זכרון, לדבג, ועוד.
פרידה זה כלי instrumentation שמאפשר לנו לגשת לקוד native, ויש לו עטיפה יפה גם לART (קוד אנדרואיד), לios/macos (ObjectiveC, Swift), וגם js + webassmbley.
כמובן שבמקרה שלנו, בעיקר נשתמש בעטיפה של Java/dalvik.

כדי להתקין frida עלינו גם להתקין frida-server שירוץ על המכשיר, וגם כלי cli לוקלי על המחשב שלנו שיתקשר עם הfrida-server.
לפני שאנחנו בוחרים frida-server, אנחנו צריכים לראות מה הארכטיקטורה של המכשיר, כדי לדעת מה היא- נריץ `getprop ro.vendor.product.cpu.abilist`

## פרידה אנדרואיד
בפרידה, אנחנו אנחנו כותבים סקריפטי javascript שמאפשרים לנו לבצע המון פעולות.
ולאנדרואיד ספציפית- יש את האובייקט Java שמאפשר לנו לשלוט על כל היכולות של פרידה באנדוראיד בקלות.
כדי לבצע פעולה מול אנדוראיד בפרידה, הקוד חייב להיות בתוך הבלוק:
```javascript
Java.perform(function () {

});
```
הקוד בתוך `Java.perform` ירוץ כאשר ה-Java VM של האפליקציה מוכן.

### גישה למחלקה
ניתן לקבל reference למחלקה בעזרת `Java.use`:
```javascript
Java.perform(function () {
    var MainActivity = Java.use("com.example.app.MainActivity");
});
```
כעת ניתן לגשת למתודות ולשדות של המחלקה.
### הוק - hook
פעולת ״הוק״ היא הפעולה שבה אנחנו יכולים להחליף מימוש של פונקציה בקוד בפונקציה שלנו.
זה ישמש אותנו הרבה, אם נרצה למשל שפונקציה מסוימת תחזיר ערך שאנחנו רוצים- או שפונקציה שנקראת כל הזמן תבצע פעולה מסוימת שאנחנו רוצים.
אם נרצה למשל לעשות hook לפונקציה checkPassword במחלקה MainActivity, נוכל לעשות זאת כך:
```javascript
Java.perform(function () {
    var MainActivity = Java.use("com.example.app.MainActivity");
    MainActivity.checkPassword.implementation = function (password) {
        console.log("Password: " + password);
        var result = this.checkPassword(password);
        console.log("Result: " + result);
        return result;
    };
});
```

מה שקורה כאן:
1. כל פעם שהאפליקציה תקרא ל-`checkPassword` הקריאה תריץ את הקוד שלנו.
2. ניתן לראות את הפרמטרים.
3. ניתן לקרוא למימוש המקורי.
4. ניתן גם לשנות את הערך המוחזר.

הרבה פעמים נעשה hook לפונקציות שאנחנו חוקרים אפליקציה למשל כדי לראות איזה פרמטרים מעבירים להם, וכו.
**טיפ ממש חשוב:** בjadx אם תלחצי על המקש ״f״ בזמן שהעכבר שלך על פונקציה, יועתק לך לclipboard סקריפט פרידה שעושה hook על הפונקציה שבחרת.
שימושי מאוד מאוד מחקר עם jadx.
### פעולות נוספות שניתן לבצע
פרידה זה לא רק הוקים.
פרידה מאפשרת לנו לבצע עוד המון פעולות באפליקציה:
### Java.choose()
מחפש מופעים (instances) חיים של מחלקה בזיכרון.
```javascript
Java.choose("com.example.app.User", {
    onMatch: function(instance) {
        console.log(instance);
    },
    onComplete: function() {}
});
```
### Java.enumerateLoadedClasses()
מחזיר רשימה של כל מחלקות ה-Java שטעונות כרגע בזיכרון.
```javascript
Java.enumerateLoadedClasses({
    onMatch: function(name) {
        console.log(name);
    },
    onComplete: function() {}
});
```
### Java.cast()
מבצע casting של אובייקט Java למחלקה מסוימת.
```javascript
var obj = Java.cast(instance, MyClass);
```
### Java.array()
יוצר מערך Java מתוך JavaScript.
```javascript
var arr = Java.array("java.lang.String", ["a", "b"]);
```
### Java.registerClass()
יוצר מחלקת Java חדשה בזמן ריצה.
```javascript
var MyClass = Java.registerClass({
    name: "com.example.MyClass",
    methods: {
        test: function () {
            console.log("hello");
        }
    }
});
```
### Java.openClassFile()
טוען קובץ ‎DEX‎ חדש לתוך האפליקציה.
```javascript
Java.openClassFile("/data/local/tmp/test.dex").load();
```
### enumerateMethods()
ניתן להדפיס את כל המתודות של מחלקה.
```javascript
var cls = Java.use("com.example.app.MainActivity");
console.log(cls.class.getDeclaredMethods());
```

לאחר שכתבנו סקריפט פרידה, כדי להריץ אותו- עלינו לכתוב את הפקודה הבאה:
```bash
frida -U -l ./script.js -f com.packagename
```
הדגל U מחבר אותנו למכשיר בusb
הדגל l מסמן את הסקריפט פרידה שאנחנו רוצים להריץ
הדגל f מסמן את זה שאנחנו רוצים שfrida תפתח את האפליקציה ותרים אותה מחדש
ובסוף השם של החבילה של האפליקציה. אפשר למצוא את השם לפי frida-ps -U או ps בתוך adb, או מחיפוש בjadx.
