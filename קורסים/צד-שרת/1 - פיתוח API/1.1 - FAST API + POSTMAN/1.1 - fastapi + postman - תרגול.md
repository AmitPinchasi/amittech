צרו פרויקט fastapi שבו תתחילו לבנות את הפרויקט שלכם.
התחילו לכתוב route-ים, controller-ים, ו- model-ים.

בפרויקט שלי, אגדיר modelים שישמור מידע על כל טיול, כאשר כל טיול יכיל מידע על הטיסות, המלונות והאטרקציות.
יהיה controllerים שידעו לשלוף על טיולים, טיסות, מלונות וארטיקציות.
וכמובן route-ים שיאפשרו לקבל את כל הטיולים, טיסות, מלנות, אטרקציות ובנוסף גם לקבל מידע על מלון ספציפי, או טיול ספציפי לפי סוג של id (מזהה כלשהו)

הוסיפו גם route-ים שמגדירים בקשות post מסוימות, למשל במקרה שלי זה יכול להיות route שמאפשר להוסיף מלון למסד הנתונים (באמצעות controller שאכתוב)ֿֿ

בקובץ בפתרון אציג את הפרויקט שלי ואיך ביצעתי אותו. (מוזמנים לצפות בו כדי ללמוד ולקבל השראה לפרויקט שלכם)
