### פתרון – ייצוגים בינאריים, הקסדצימליים, ותווי ASCII

---

#### שאלה 1: יחידות אחסון

1. **כמה ביטים יש ב־2 מגהבייטים?**

- 1 MB = 1024 KB = 1024 × 1024 בייטים = 1,048,576 בייטים

- 2 MB = 2 × 1,048,576 = 2,097,152 בייטים

- כל בייט = 8 ביטים

- 2,097,152 × 8 = **16,777,216 ביטים**

1. **כמה קילובייטים יש ב־0.5 גיגהבייט?**

- 1 GB = 1024 MB = 1024 × 1024 KB = 1,048,576 KB

- 0.5 GB = 0.5 × 1,048,576 = **524,288 KB**

1. **כמה בייטים יש ב־1.5 טרהבייט?**

- 1 TB = 1024 GB = 1024 × 1024 × 1024 בייטים = 1,099,511,627,776 בייטים

- 1.5 TB = 1.5 × 1,099,511,627,776 = **1,649,267,441,664 בייטים**

#### שאלה 2: הקסדצימלי

1. **המר את המספר הבינארי הבא להקסדצימלי: `110101100111`**

- נחלק ל־4 ביטים: `0001 1010 1100 0111`

- או מהסוף: `1101 0110 0111`

- `1101` = D, `0110` = 6, `0111` = 7

- תשובה: **0xD67**

1. **המר את `0x9F` לבינארי:**

- `9` = `1001`, `F` = `1111`

- תשובה: **10011111**

1. **איזה מספר עשרוני מייצג `0x1A`?**

- `1A` = 1×16 + 10 = **26**

#### שאלה 3: Endianness

`0xCAFEBABE`

בייטים: `CA FE BA BE`

1. **Big Endian** (לפי הסדר):

```
Address:  0x00   0x01   0x02   0x03  
Value:    0xCA   0xFE   0xBA   0xBE
```

1. **Little Endian** (בסדר הפוך):

```
Address:  0x00   0x01   0x02   0x03  
Value:    0xBE   0xBA   0xFE   0xCA
```

#### שאלה 4: Signed ו־Unsigned

1. **מה הערך של `11111111`?**

- Unsigned: כל הביטים דלוקים → 255

- Signed: זהו -1 (שיטת two's complement)

1. **טווחי ייצוג:**

- Unsigned (8 ביטים): **0 עד 255**

- Signed (8 ביטים): **-128 עד 127**

1. **מה הערך של `10000001` (signed)?**

- ערך זה מייצג את **-127**

#### שאלה 5: Floating Point

1. **מהו `0 10000010 10100000000000000000000`?**

- Sign = 0 → חיובי

- Exponent = `10000010` = 130

- Bias = 127 → exponent real = 3

- Mantissa = `1.101` = 1 + 0.5 + 0.125 = 1.625

- ערך: **1.625 × 2³ = 13.0**

1. **-3.75 ל־IEEE 754:**

- `-3.75` → `11.11` = `1.111 × 2¹`

- Sign = 1

- Exponent = 1 + 127 = 128 = `10000000`

- Mantissa = `11100000000000000000000`

- תשובה:  
`1 10000000 11100000000000000000000`

1. **מה ההבדל בין Bias למעריך אמיתי?**

- Bias = קבוע שנוסף כדי להפוך את המעריך לחיובי תמיד (127 ב־32 ביט)

- המעריך האמיתי = הערך של exponent - bias

#### שאלה 6: ASCII

1. **"Ok!"**

- `'O'` = `01001111` = `0x4F`

- `'k'` = `01101011` = `0x6B`

- `'!'` = `00100001` = `0x21`

- בינארי: `01001111 01101011 00100001`

- הקס: **0x4F 0x6B 0x21**

1. **פענוח הרצף:**  
`01001000` → H  
`01100101` → e  
`01101100` → l  
`01101100` → l  
`01101111` → o  
→ **Hello**

2. **0x0A** = `00001010` = תו **newline** (`\n`)

#### שאלה 7: מיזוג מושגים

1. **"Amit" – ייצוג בינארי והקס:**

- `'A'` = `01000001` = `0x41`

- `'m'` = `01101101` = `0x6D`

- `'i'` = `01101001` = `0x69`

- `'t'` = `01110100` = `0x74`

- בינארי:  
`01000001 01101101 01101001 01110100`

- הקס: **0x41 0x6D 0x69 0x74**

1. **`-128` Signed (8 ביטים):**

- בינארי: `10000000`

- הקס: `0x80`

1. **`0x3F2C` ב־Little Endian:**

- בייטים: `3F` `2C`

- הפוך:

```
Address:  0x00   0x01  
Value:    0x2C   0x3F
```
