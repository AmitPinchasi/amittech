## הפרמידה
```python

level_number = int(input("Enter a level: "))  
  
for level in range(level_number + 1):  
    print(char * level)
```

## מספרים ראשונים (קשה)
```python 
for number in range(100):
	is_prime = True
	for i in range(2, number):
		if number % i == 0:
			is_prime = False
    if is_prime:  
        print(number)
```
