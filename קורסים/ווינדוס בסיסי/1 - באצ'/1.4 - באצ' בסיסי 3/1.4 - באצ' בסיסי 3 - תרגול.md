### תרגיל 1: שימוש ב-Pipe

1. הריצו את הפקודה **`tasklist`** ומצאו אם קיימת תוכנה בשם **`Notepad`** בעזרת פקודת **`find`**:

```cmd
tasklist | find "Notepad"
```

2. השתמשו ב-Pipe כדי למצוא תהליכים אחרים במערכת.

---

### תרגיל 2: הפניית פלט לקובץ

1. השתמשו בפקודה **`echo`** כדי להדפיס את המילים **"Hello Batch"** לקובץ בשם **`batchfile.txt`**:

```cmd
echo Hello Batch > batchfile.txt
```

2. פתחו את הקובץ **`batchfile.txt`** וודאו שהתוכן נכנס כראוי.

---

### תרגיל 3: הוספת תוכן לקובץ

1. השתמשו בפקודה **`echo`** כדי להוסיף את המילה **"More Content"** לקובץ **`batchfile.txt`**:

```cmd
echo More Content >> batchfile.txt
```

2. קראו את התוכן של הקובץ וודאו שהתוכן נוסף.

---

### תרגיל 4: הרצת פקודות ברצף

1. השתמשו באופרטור **`;`** כדי להריץ פקודות ברצף, לדוגמה:

```cmd
dir C:\Windows; dir C:\Users
```


---

### תרגיל 5: הרצת פקודות בתנאי הצלחה

1. השתמשו באופרטור **`&&`** כדי להריץ פקודה רק אם הפקודה הקודמת הצליחה. למשל:

```cmd
dir C:\Windows && echo Directory Exists
```


---

### תרגיל 6: הרצת פקודות בתנאי כישלון

1. השתמשו באופרטור **`||`** כדי להריץ פקודה רק אם הפקודה הקודמת נכשלה. לדוגמה:

```cmd
dir C:\InvalidFolder || echo Directory Not Found
```


---

### תרגיל 7: שימוש במשתני סביבה

1. השתמשו בפקודה **`echo %USERNAME%`** כדי לראות את שם המשתמש.
2. השתמשו בפקודה **`echo %COMPUTERNAME%`** כדי לראות את שם המחשב.
3. השתמשו בפקודה **`echo %OS%`** כדי לראות את גרסת מערכת ההפעלה.
4. השתמשו בפקודה **`echo %USERPROFILE%`** כדי לראות את נתיב תיקיית הבית של המשתמש.

---

### תרגיל 8: יצירת תיקיות בעזרת Batch

1. השתמשו בפקודה **`mkdir`** כדי ליצור תיקייה בשם **`TestFolder`**.
2. השתמשו בפקודה **`rmdir`** כדי למחוק את התיקייה **`TestFolder`**.

---

### תרגיל 9: יצירת קובץ וכתיבה לתוכו

1. השתמשו בפקודה **`echo`** כדי ליצור קובץ בשם **`testfile.txt`** ולכתוב לו את המילים **"Batch is Awesome!"**:

```cmd
echo Batch is Awesome! > testfile.txt
```
