הנה פתרון לתרגיל:

**קובץ `ComputerSystem.mof`**:

```mof
#pragma namespace ("\\\\.\\root\\cimv2")

class ComputerSystem
{
   [key] string ComputerName;
   string OperatingSystem;
};

instance of ComputerSystem
{
   ComputerName = "MyComputer";
   OperatingSystem = "Windows 10";
};

```
**הסבר**:

- אחרי הגדרת המחלקה `ComputerSystem`, יצרנו מופע שלה עם ערכים לדוגמה:
    - `ComputerName = "MyComputer"`.
    - `OperatingSystem = "Windows 10"`.
1. **קימפול הקובץ**:
    
    - יש להשתמש בפקודה `mofcomp` כדי לקמפל את קובץ ה-MOF.
    - הפקודה: `mofcomp ComputerSystem.mof`.

2. **שליפת האובייקט**:
```powershell
Get-WmiObject -Namespace "root\cimv2" -Class "ComputerSystem"
```
הפקודה הזו תציג את כל המופעים של המחלקה `ComputerSystem` שנמצאים ב-WMI, כולל את הערכים של `ComputerName` ו-`OperatingSystem`.
