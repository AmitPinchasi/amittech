    10.3 - Technical SEO - ОфОеОњОЋОю

    ОфОеОњОЋОю - SEO ОўОЏОаОЎ - Technical SEO¶

    ОфОеОњОЎОю 1 - ОъОЉОаОћ URL ОЋОљОЊОеОЎОЏОюОЋОф ОљОфОе¶

    ОЎОЕ ОюОЏОЮ ОљОфОе ОЌОЊОЕОЋОф ОўОЏОаОЋОюОЋОњОЎ ОбОЮ ОћОДОўОњОЋОеОЎОЋОф ОћОЉОљОЋОф:
    - ОЌОЊОЕОЋОф (ОъОфОбОЊОЏОЪ ОЎОЋОъОЎОф)
    - ОъОЊОеОЎОЏОЎОЮ (ОюОцОЎ ОаОЋОЕОљ: React, Node.js, Python, DevOps)
    - ОАОДОЎОеОЋОф ОъОЋОдОеОЎОЮ
    - ОцОЋОЊОДОљОАОў (ОцОеОДОЎОЮ)
    - ОљОЋОЊОЋОф / ОдОЋОе ОДОЕОе

    ОфОЏОаОаОЋ:
    1. ОъОЉОаОћ URL ОъОюОљ ОюОЏОю ОАОЋОњ ОЊОБ (ОюОцОЌОЋОф 10 URL-ОЎОЮ ОюОЊОЋОњОъОћ)
    2. ОъОЉОаОћ ОфОЎОДОЎОЋОф ОЉ-Next.js App Router ОЕОфОЋОъОџ ОЉ-URL-ОЎОЮ ОЕОћОњОЊОеОфОЮ
    3. ОћОАОЉОЎОеОЋ ОљОф ОћОћОЌОюОўОЋОф ОЕОюОЏОЮ (ОюОъОћ ОЉОЌОеОфОЮ ОљОф ОћОъОЉОаОћ ОћОќОћ)

    ОфОеОњОЎОю 2 - sitemap.xml ОЊОЎОаОъОЎ¶

    ОЉОаОЋ ОъОбОеОЏОф sitemap ОъОюОљОћ ОЉ-Next.js ОбОЉОЋОе ОљОфОе ОЌОаОЋОф ОбОЮ:
    - 500 ОъОЋОдОеОЎОЮ ОЉОДОўОњОЋОеОЎОЋОф ОЕОЋОаОЋОф
    - 30 ОДОўОњОЋОеОЎОЋОф
    - 50 ОъОљОъОеОЎ ОЉОюОЋОњ
    - ОЊОцОЎОЮ ОАОўОўОЎОЎОЮ (ОЉОЎОф, ОљОЋОЊОЋОф, ОдОЋОе ОДОЕОе, ОфОаОљОЎ ОЕОЎОъОЋОЕ)

    ОћОЊОеОЎОЕОЋОф:
    - ОцОЎОдОЋОю Ою-sitemap index ОбОЮ sitemaps ОаОцОеОЊОЎОЮ ОюОъОЋОдОеОЎОЮ, ОДОўОњОЋОеОЎОЋОф, ОЉОюОЋОњ, ОЋОЊОцОЎОЮ ОАОўОўОЎОЎОЮ
    - priority ОЕОЋОаОћ ОюОЏОю ОАОЋОњ ОЊОБ
    - changeFrequency ОъОфОљОЎОЮ ОюОЏОю ОАОЋОњ
    - lastModified ОъОбОЋОЊОЏОЪ

    ОфОеОњОЎОю 3 - robots.txt ОъОфОДОЊОЮ¶

    ОЏОфОЉОЋ robots.txt ОюОљОфОе SaaS ОЕОЏОЋОюОю:
    - ОљОќОЋОе ОЕОЎОЋОЋОДОЎ ОдОЎОЉОЋОеОЎ (/, /features, /pricing, /blog)
    - ОљОќОЋОе ОљОцОюОЎОДОдОЎОћ (/app/*)
    - ОљОќОЋОе API (/api/*)
    - ОљОќОЋОе ОаОЎОћОЋОю (/admin/*)
    - ОЊОцОЎ ОЌОЎОцОЋОЕ (/search?q=...)
    - ОЊОцОЎ ОъОЕОфОъОЕ (/users/*)
    - ОДОЉОдОЎ assets (/assets/*)

    ОЉОаОЋОАОБ:
    - ОЌОАОъОЋ ОљОф GPTBot ОЋ-CCBot (ОЉОЋОўОЎОЮ ОЕОю AI)
    - ОљОцОЕОеОЋ Ою-Googlebot ОњОЎОЕОћ Ою-assets
    - ОћОЋОАОЎОцОЋ crawl-delay ОюОЉОЋОўОЎОЮ ОъОАОЋОЎОъОЎОЮ

    ОЏОфОЉОЋ ОњОЮ ОљОф ОћОњОеОАОћ ОЉ-Next.js (app/robots.ts).

    ОфОеОњОЎОю 4 - Breadcrumbs ОбОЮ Structured Data¶

    ОЉОаОЋ ОДОЋОъОцОЋОаОаОўОф Breadcrumbs ОъОюОљОћ ОЕОЏОЋОюОюОф:
    - ОфОдОЋОњОћ ОЋОЎОќОЋОљОюОЎОф ОбОЮ CSS
    - ОфОъОЎОЏОћ ОЉ-RTL
    - ОАОЏОъОф JSON-LD ОъОАОЋОњ BreadcrumbList
    - ОаОњОЎОЕОЋОф (aria attributes)

    ОћОДОЋОъОцОЋОаОаОўОћ ОдОеОЎОЏОћ ОюОДОЉОю ОъОбОеОџ ОЕОю ОцОеОЎОўОЎОЮ ОЋОюОћОЎОЋОф ОњОаОеОЎОф - ОаОЎОфОаОф ОюОЕОЎОъОЋОЕ ОЌОЋОќОе ОЉОЏОю ОЊОБ.

    ОЉОЊОДОЋ ОљОЋОфОћ ОбОЮ ОаОфОЎОЉОЎ ОаОЎОЋОЋОў ОЕОЋОаОЎОЮ:
    1. ОЊОБ ОћОЉОЎОф > ОъОЋОдОеОЎОЮ > ОъОЌОЕОЉОЎОЮ > Dell XPS 15
    2. ОЊОБ ОћОЉОЎОф > ОЉОюОЋОњ > React > ОъОЊОеОЎОџ Hooks
    3. ОЊОБ ОћОЉОЎОф > ОДОЋОеОАОЎОЮ > ОцОЎОфОЋОЌ ОцОеОЋОаОўОљОаОЊ > ОЕОЎОбОЋОе 5

    ОфОеОњОЎОю 5 - ОЉОЊОЎОДОф ОДОЎОЕОЋОеОЎОЮ ОцОаОЎОъОЎОЎОЮ¶

    ОЏОфОЉОЋ ОАОДОеОЎОцОў (ОЉ-TypeScript) ОЕОАОЋОеОД ОљОфОе ОЋОЉОЋОЊОД:
    1. ОДОЎОЕОЋОеОЎОЮ ОЕОЉОЋОеОЎОЮ (404)
    2. ОЊОцОЎОЮ "ОЎОфОЋОъОЎОЮ" (ОЊОцОЎОЮ ОЕОљОЎОЪ ОљОюОЎОћОЮ ОЕОЋОЮ ОДОЎОЕОЋОе ОцОаОЎОъОЎ)
    3. ОЕОеОЕОеОф ОћОцОаОЎОЋОф (redirect chains) - ОЎОЋОфОе Оъ-2 ОћОцОаОЎОЋОф ОЉОеОдОБ
    4. ОДОЎОЕОЋОеОЎОЮ ОбОЮ anchor text ОеОЎОД ОљОЋ ОњОаОеОЎ ("ОюОЌОдОЋ ОЏОљОЪ")

    ОћОАОДОеОЎОцОў ОЎОДОЉОю URL ОЉОАОЎОАОЎ ОЋОЎОЌОќОЎОе ОЊОЋОЌ ОбОЮ ОЏОю ОћОЉОбОЎОЋОф.

    ОеОъОќ: ОћОЕОфОъОЕОЋ ОЉ-fetch ОЋ-DOMParser (ОљОЋ cheerio) ОюОаОЎОфОЋОЌ HTML.
