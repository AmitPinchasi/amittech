    7.4 - ОеОЎОљОДОў ОеОљОЋОўОе - ОфОеОњОЋОю

    ОфОеОњОЋОю - ОеОЎОљОДОў ОеОљОЋОўОе - React Router¶

    ОфОеОњОЎОю 1 - ОљОфОе ОфОЊОъОЎОф ОЉОАОЎОАОЎ¶

    ОдОеОЋ ОљОфОе ОфОЊОъОЎОф ОбОЮ ОаОЎОЋОЋОў ОЉОАОЎОАОЎ.

    ОЊОеОЎОЕОЋОф:
    - ОЊОцОЎОЮ: ОЉОЎОф, ОљОЋОЊОЋОф, ОЕОЎОеОЋОфОЎОЮ, ОдОеОЋ ОДОЕОе
    - ОфОцОеОЎОў ОаОЎОЋОЋОў ОДОЉОЋОб (navbar) ОбОЮ NavLink ОЕОъОЊОњОЎОЕ ОљОф ОћОЊОБ ОћОцОбОЎОю
    - ОЏОю ОЊОБ ОъОдОЎОњ ОфОЋОЏОЪ ОЕОЋОаОћ
    - ОЊОБ 404 ОъОЋОфОљОЮ ОюОаОфОЎОЉОЎОЮ ОюОљ ОДОЎОЎОъОЎОЮ
    - ОЏОцОфОЋОе "ОЌОќОеОћ ОюОЊОБ ОћОЉОЎОф" ОЉОЊОБ 404

    ОфОеОњОЎОю 2 - ОЉОюОЋОњ ОбОЮ ОаОфОЎОЉОЎОЮ ОЊОЎОаОъОЎОЎОЮ¶

    ОдОеОЋ ОЉОюОЋОњ ОцОЕОЋОў ОбОЮ ОеОЕОЎОъОф ОцОЋОАОўОЎОЮ ОЋОЊОБ ОцОЋОАОў ОЉОЋОЊОЊ.

    ОЊОеОЎОЕОЋОф:
    - ОЊОБ ОеОљОЕОЎ ОЕОъОдОЎОњ ОеОЕОЎОъОф ОцОЋОАОўОЎОЮ (ОЏОЋОфОеОф, ОфОљОеОЎОџ, ОфОДОдОЎОе)
    - ОюОЌОЎОдОћ ОбОю ОцОЋОАОў ОъОбОЉОЎОеОћ ОюОЊОБ /posts/:postId
    - ОЊОБ ОцОЋОАОў ОЉОЋОЊОЊ ОЕОъОдОЎОњ ОљОф ОћОфОЋОЏОЪ ОћОъОюОљ
    - ОЏОцОфОЋОе ОЌОќОеОћ ОюОеОЕОЎОъОћ
    - ОљОЮ ОъОаОАОЎОЮ ОюОњОЕОф ОюОцОЋОАОў ОюОљ ОДОЎОЎОЮ, ОъОдОЎОњОЎОЮ ОћОЋОЊОбОф ОЕОњОЎОљОћ

    ОаОфОЋОаОЎ ОЊОЋОњОъОћ:

    ОфОеОњОЎОю 3 - ОюОЋОЌ ОЉОДОеОћ ОбОЮ ОаОфОЎОЉОЎОЮ ОъОДОЋОаОаОЎОЮ¶

    ОдОеОЋ ОюОЋОЌ ОЉОДОеОћ (Dashboard) ОбОЮ ОаОфОЎОЉОЎОЮ ОъОДОЋОаОаОЎОЮ.

    ОЊОеОЎОЕОЋОф:
    - layout ОеОљОЕОЎ ОбОЮ sidebar ОЋОЌОюОД ОфОЋОЏОЪ
    - ОаОфОЎОЉОЎОЮ ОъОДОЋОаОаОЎОЮ: /dashboard (ОАОДОЎОеОћ), /dashboard/users (ОъОЕОфОъОЕОЎОЮ), /dashboard/products (ОъОЋОдОеОЎОЮ), /dashboard/settings (ОћОњОЊОеОЋОф)
    - Оћ-sidebar ОъОдОЎОњ ОљОф ОћОаОфОЎОЉОЎОЮ ОбОЮ NavLink ОцОбОЎОю
    - ОЉОЊОБ ОъОЕОфОъОЕОЎОЮ, ОюОЌОЎОдОћ ОбОю ОъОЕОфОъОЕ ОъОбОЉОЎОеОћ Ою-/dashboard/users/:userId
    - Breadcrumbs (ОаОЎОЋОЋОў ОцОЎОеОЋОеОЎ ОюОЌОЮ) ОЕОъОдОЎОњ ОљОф ОћОаОфОЎОЉ ОћОаОЋОЏОЌОЎ

    ОфОеОњОЎОю 4 - ОЌОаОЋОф ОъОДОЋОЋОаОф ОбОЮ ОАОЎОаОЋОЪ ОЉ-URL¶

    ОдОеОЋ ОЌОаОЋОф ОъОДОЋОЋОаОф ОЕОћОАОЎОаОЋОЪ ОЕОюОћ ОаОЕОъОе ОЉ-URL.

    ОЊОеОЎОЕОЋОф:
    - ОЊОБ ОъОЋОдОеОЎОЮ ОбОЮ ОАОЎОаОЋОЪ ОюОцОЎ: ОДОўОњОЋОеОЎОћ, ОўОЋОЋОЌ ОъОЌОЎОеОЎОЮ, ОъОЎОЋОЪ
    - ОЏОю ОћОцОЎОюОўОеОЎОЮ ОаОЕОъОеОЎОЮ ОЉ-query string: ?category=electronics&sort=price&minPrice=100
    - ОЕОЎОфОЋОБ ОюОЎОаОД ОбОЮ ОћОцОЎОюОўОеОЎОЮ ОћОцОбОЎОюОЎОЮ
    - ОеОбОаОЋОЪ ОћОЊОБ ОЕОЋОъОе ОљОф ОћОцОЎОюОўОеОЎОЮ
    - ОЏОцОфОЋОе "ОаОДОћ ОцОЎОюОўОеОЎОЮ" ОЕОъОљОцОА ОљОф Оћ-URL
    - ОЊОБ ОъОЋОдОе ОЉОЋОЊОЊ ОбОЮ ОЏОцОфОЋОе "ОЌОќОеОћ ОюОъОЋОдОеОЎОЮ" ОЕОЕОЋОъОе ОљОф ОћОцОЎОюОўОеОЎОЮ

    ОфОеОњОЎОю 5 - ОљОЋОфОаОўОЎОДОдОЎОћ ОбОЮ ОаОфОЎОЉОЎОЮ ОъОЋОњОаОЎОЮ¶

    ОдОеОЋ ОъОбОеОЏОф ОљОЋОфОаОўОЎОДОдОЎОћ ОъОюОљОћ ОбОЮ routing.

    ОЊОеОЎОЕОЋОф:
    - ОЊОцОЎОЮ ОдОЎОЉОЋОеОЎОЎОЮ: ОЉОЎОф, ОћОфОЌОЉОеОЋОф, ОћОеОЕОъОћ
    - ОЊОцОЎОЮ ОъОЋОњОаОЎОЮ: ОцОеОЋОцОЎОю, ОћОњОЊОеОЋОф, ОюОЋОЌ ОЉОДОеОћ
    - ОљОЮ ОъОЕОфОъОЕ ОюОљ ОъОЌОЋОЉОе ОаОЎОњОЕ ОюОЊОБ ОъОЋОњОЪ, ОћОЋОљ ОъОЋОцОаОћ ОюОЊОБ ОћОфОЌОЉОеОЋОф
    - ОљОЌОеОЎ ОћОфОЌОЉОеОЋОф, ОћОъОЕОфОъОЕ ОЌОЋОќОе ОюОЊОБ ОЕОаОЎОАОћ ОюОњОЕОф ОљОюОЎОЋ
    - ОљОЮ ОъОЕОфОъОЕ ОъОЌОЋОЉОе ОаОЎОњОЕ ОюОЊОБ ОћОфОЌОЉОеОЋОф, ОћОЋОљ ОъОЋОцОаОћ ОюОЊОБ ОћОЉОЎОф
    - ОЏОцОфОЋОе ОћОфОаОфОДОЋОф ОЕОъОцОаОћ ОюОЊОБ ОћОЉОЎОф

    ОфОеОњОЎОю 6 - ОљОцОюОЎОДОдОЎОЎОф ОъОфОЏОЋОаОЎОЮ ОъОюОљОћ¶

    ОдОеОЋ ОљОцОюОЎОДОдОЎОЎОф ОъОфОЏОЋОаОЎОЮ ОъОюОљОћ ОбОЮ routing ОъОфОДОЊОЮ.

    ОЊОеОЎОЕОЋОф:
    - ОъОЉОаОћ ОаОфОЎОЉОЎОЮ: / (ОЉОЎОф), /recipes (ОЏОю ОћОъОфОЏОЋОаОЎОЮ), /recipes/:id (ОъОфОЏОЋОЪ ОЉОЋОЊОЊ), /categories/:category (ОъОфОЏОЋОаОЎОЮ ОюОцОЎ ОДОўОњОЋОеОЎОћ), /favorites (ОъОЋОбОЊОцОЎОЮ - ОъОЋОњОЪ), /add-recipe (ОћОЋОАОцОћ - ОъОЋОњОЪ)
    - ОаОЎОЋОЋОў ОбОЮ NavLink ОЋ-breadcrumbs
    - ОАОЎОаОЋОЪ ОЋОЌОЎОцОЋОЕ ОЉ-URL (useSearchParams)
    - ОъОЋОбОЊОцОЎОЮ ОаОЕОъОеОЎОЮ ОЉ-localStorage
    - ОљОаОЎОъОдОЎОЎОф ОъОбОЉОе ОЉОЎОЪ ОЊОцОЎОЮ (CSS transitions)
    - ОўОбОЎОаОф ОаОфОЋОаОЎОЮ ОъОЊОЋОъОћ ОбОЮ state loading
    - ОЊОБ 404 ОъОЋОфОљОЮ

    ОЕОљОюОЋОф¶

    2. ОъОћ ОћОћОЉОЊОю ОЉОЎОЪ navigate(path) Ою-Navigate component? ОъОфОЎ ОаОЕОфОъОЕ ОЉОЏОю ОљОЌОЊ?
    3. ОъОћ ОћОЎОфОеОЋОЪ ОЕОю ОЕОъОЎОеОф ОцОЎОюОўОеОЎОЮ ОЉ-URL ОбОЮ useSearchParams ОюОбОЋОъОф useState?
