    11.8 - ОњОЎОЕОћ ОюОцОеОЋОЎОДОў ОцОеОЋОаОўОљОаОЊ - ОфОеОњОЋОю

    ОфОеОњОЋОю - ОњОЎОЕОћ ОюОцОеОЋОЎОДОў ОцОеОЋОаОўОљОаОЊ¶

    ОфОеОњОЎОю 1 - ОфОЏОаОЋОЪ ОцОеОЋОЎОДОў¶

    ОЉОЌОеОЋ ОљОЌОЊ ОъОћОцОеОЋОЎОДОўОЎОЮ ОћОЉОљОЎОЮ ОЋОфОЏОаОаОЋ ОљОЋОфОЋ:

    Ољ. ОљОцОюОЎОДОдОЎОЎОф ОаОЎОћОЋОю ОъОЕОЎОъОЋОф ОюОдОЋОЋОфОЎОЮ
    ОЉ. ОцОюОўОцОЋОеОъОф ОЉОюОЋОњ ОбОЮ ОъОбОеОЏОф ОфОњОЋОЉОЋОф
    Оњ. ОЊОЕОЉОЋОеОЊ ОаОЎОфОЋОЌ ОаОфОЋОаОЎОЮ

    ОбОЉОЋОе ОћОцОеОЋОЎОДОў ОЕОЉОЌОеОфОЮ, ОћОЏОЎОаОЋ:
    - ОфОЎОљОЋОе ОћОцОеОЋОЎОДОў (3-5 ОъОЕОцОўОЎОЮ)
    - ОеОЕОЎОъОф ОЊОеОЎОЕОЋОф ОцОЋОаОДОдОЎОЋОаОюОЎОЋОф (ОюОцОЌОЋОф 8)
    - ОеОЕОЎОъОф ОЊОеОЎОЕОЋОф ОюОљ-ОцОЋОаОДОдОЎОЋОаОюОЎОЋОф (ОюОцОЌОЋОф 5)
    - ОћОњОЊОеОф MVP - ОъОћ ОћОњОеОАОћ ОћОъОЎОаОЎОъОюОЎОф?
    - ОеОЕОЎОъОф ОЊОцОЎОЮ/ОъОАОЏОЎОЮ ОеОљОЕОЎОЎОЮ

    ОфОеОњОЎОю 2 - ОЉОЌОЎОеОф Tech Stack¶

    ОбОЉОЋОе ОћОцОеОЋОЎОДОў ОЕОЉОЌОеОфОЮ, ОћОдОЎОбОЋ Tech Stack ОъОаОЋОъОД:

    • ОцОеОЎОЎОъОЋОЋОеОД (ОЋОюОъОћ)
    • ОАОцОеОЎОЎОф UI (ОЋОюОъОћ)
    • ОаОЎОћОЋОю ОАОўОЎОЎОў (ОЋОюОъОћ)
    • ОАОцОеОЎОЎОф ОўОцОАОЎОЮ
    • ОцОфОеОЋОЪ ОљОЎОъОЋОф
    • ОъОАОЊ ОаОфОЋОаОЎОЮ ОЋ-ORM
    • ОцОюОўОцОЋОеОъОф ОцОеОЎОАОћ
    • ОЏОюОЎ ОЉОЊОЎОДОЋОф

    ОюОЏОю ОЉОЌОЎОеОћ, ОћОАОЉОЎОеОЋ ОюОъОћ ОЉОЌОеОфОЮ ОЉОЏОюОЎ ОћОќОћ ОЋОюОљ ОЉОљОюОўОеОаОўОЎОЉОЋОф.

    ОфОеОњОЎОю 3 - ОъОЉОаОћ ОцОеОЋОЎОДОў¶

    ОдОеОЋ ОљОф ОъОЉОаОћ ОћОфОЎОДОЎОЋОф ОћОъОюОљ ОЕОю ОћОцОеОЋОЎОДОў ОЕОюОЏОЮ:

    • ОЏОю ОћОфОЎОДОЎОЋОф ОЋОћОДОЉОдОЎОЮ ОћОеОљОЕОЎОЎОЮ
    • ОћОАОЉОе ОДОдОе ОюОЏОю ОфОЎОДОЎОЎОћ
    • ОЊОЋОњОъОћ ОюОДОЋОЉОЦ index.ts ОЕОъОЎОЎОдОљ ОъОфОЎОДОЎОЎОф components

    ОЉОаОЋОАОБ, ОдОеОЋ ОЊОЎОљОњОеОъОф ОљОеОЏОЎОўОДОўОЋОеОћ ОЕОъОеОљОћ:
    - ОЏОЎОдОЊ ОћОаОфОЋОаОЎОЮ ОќОЋОеОъОЎОЮ ОъОћОЕОеОф ОюОюОДОЋОЌ
    - ОћОЎОЏОЪ ОЎОЋОЕОЉ ОаОЎОћОЋОю ОћОАОўОЎОЎОў
    - ОЏОЎОдОЊ ОћОљОЎОъОЋОф ОбОЋОЉОЊ

    ОфОеОњОЎОю 4 - ОћОњОЊОеОф ОЏОюОЎ ОцОЎОфОЋОЌ¶

    ОћОњОЊОЎОеОЋ ОАОЉОЎОЉОф ОбОЉОЋОЊОћ ОъОДОдОЋОбОЎОф ОЕОЏОЋОюОюОф:

    • ESLint ОбОЮ ОЏОюОюОЎОЮ ОъОЋОфОљОъОЎОЮ (ОЏОЋОюОю React Hooks ОЋОаОњОЎОЕОЋОф)
    • Prettier ОбОЮ ОћОњОЊОеОЋОф
    • Husky ОбОЮ pre-commit hook ОЕОъОеОЎОЦ lint-staged
    • commitlint ОбОЮ Conventional Commits
    • ОДОЋОЉОЦ .editorconfig
    • ОДОЋОЉОЦ .nvmrc ОюОаОбОЎОюОф ОњОеОАОф Node.js
    • ОДОЋОЉОЦ .env.example

    ОЏОфОЉОЋ ОљОф ОЏОю ОДОЉОдОЎ ОћОћОњОЊОеОЋОф.

    ОфОеОњОЎОю 5 - ОцОЎОеОЋОД ОъОЕОЎОъОЋОф¶

    ОцОеОДОЋ ОљОф ОћОцОеОЋОЎОДОў ОЕОюОЏОЮ ОюОъОЕОЎОъОЋОф:

    • ОЌОюОДОЋ ОюОљОЉОаОЎ ОЊОеОџ (milestones) ОЕОю ОЕОЉОЋОб-ОЕОЉОЋОбОЎОЎОЮ
    • ОЉОЏОю ОљОЉОЪ ОЊОеОџ, ОцОеОўОЋ ОљОф ОћОъОЕОЎОъОЋОф ОћОАОцОдОЎОцОЎОЋОф
    • ОћОбОеОЏОф ОќОъОЪ ОюОЏОю ОъОЕОЎОъОћ (ОЉОЕОбОЋОф)
    • ОАОЊОе ОбОЊОЎОцОЋОЎОЋОф (P0 - ОДОеОЎОўОЎ, P1 - ОЌОЕОЋОЉ, P2 - ОаОЌОъОЊ)
    • ОќОЎОћОЋОЎ ОфОюОЋОЎОЋОф ОЉОЎОЪ ОъОЕОЎОъОЋОф

    ОЌОЕОЉОЋ ОбОю ОЏОю ОЕОюОЉ Оъ-setup ОеОљОЕОЋОаОЎ ОЋОбОЊ ОцОеОЎОАОћ ОЋОЉОЊОЎОДОЋОф.
