    5.3 - ОЏОфОЎОЉОћ ОЕОеОЎОеОЋОфОЎОф ОбОЮ n - ОфОеОњОЋОю

    ОфОеОњОЋОю - ОЏОфОЎОЉОћ ОЕОеОЎОеОЋОфОЎОф ОбОЮ n¶

    ОЉОфОеОњОЋОю ОћОќОћ ОфОћОцОЏОЋ ОљОф ОЌОЋОюОЕОф Оћ-format string ОъОъОЏОЋОаОф ОДОеОЎОљОћ ОюОъОЏОЋОаОф ОЏОфОЎОЉОћ. ОћОъОўОеОћ: ОюОЊОеОЋОА ОъОЕОфОаОћ ОњОюОЋОЉОюОЎ ОЉОбОеОџ ОъОЊОЋОЎОД ОЕОљОфОЮ ОЉОЋОЌОеОЎОЮ, ОДОЋОЊОЮ ОЉОЎОЊОЎОЎОЮ (ОАОЋОцОеОЎОЮ ОЉОфОЎОЮ, ОЉОЋОаОЎОЮ ОљОф ОъОЌОеОЋОќОф Оћformat, ОъОўОцОюОЎОЮ ОЉОАОЊОе ОбОЋОюОћ ОЋОЉoverflow), ОЋОеОД ОљОЌОе ОЏОџ ОбОЮ fmtstr_payload ОЕОЎОбОЕОћ ОљОф ОќОћ ОЉОЕОЋОеОћ ОљОЌОф. ОљОю ОфОЊОюОњОЋ ОбОю ОћОЌОюОД ОћОЎОЊОаОЎ - ОћОЋОљ ОъОћ ОЕОЎОЉОЊОЎОю ОљОфОЏОЮ ОъОъОЎОЕОћОЋ ОЕОеОД ОъОбОфОЎОД ОАОДОеОЎОцОўОЎОЮ. ОфОЉОЎОаОЋ ОъОћ ОДОЋОеОћ ОъОфОЌОф ОюОъОЏОАОћ, ОЋОљОќ ОфОеОЕОЋ ОюОЏОюОЎ ОюОбОЕОЋОф ОљОф ОќОћ ОЉОЕОЉОЎОюОЏОЮ.

    ОАОЉОЎОЉОћ ОЋОћОаОЌОЋОф¶

    ОћОЏОю ОеОЦ ОъОДОЋОъОЎОф ОбОю ОюОЎОаОЋОДОА ОбОЮ python3 ОЋ-pwntools, gcc ОЋ-gdb/pwndbg. ОћОЉОЎОаОљОеОЎ ОЕОюОаОЋ ОъОДОЋОъОцОю ОЉОюОЎ PIE (ОЏОфОЋОЉОф ОњОюОЋОЉОюОЎОф ОДОЉОЋОбОћ), ОЉОюОЎ canary, ОбОЮ NX, ОЋОЉОюОЎ _FORTIFY_SOURCE (ОљОЌОеОф glibc ОфОЌОАОЋОЮ ОљОф %n ОбОю ОъОЌОеОЋОќОф format ОЏОфОЎОЉОћ). ОЕОъОеОЋ ОЏ-writeme.c:

    #include <stdio.h>

long secret = 0;    // your target

int main(void) {
    char buf[256];
    setvbuf(stdout, NULL, _IONBF, 0);
    printf("secret @ %p\n", (void*)&secret);
    fgets(buf, sizeof(buf), stdin);
    printf(buf);
    printf("\nsecret = %#lx\n", secret);
    if (secret == 0x1337c0de)
        puts("WIN: secret overwritten exactly!");
    return 0;
}

    ОДОЋОъОцОЎОюОдОЎОћ:

    gcc -fno-stack-protector -no-pie -O0 -o writeme writeme.c

    ОЉОЊОДОЋ ОљОф ОћОћОњОаОЋОф ОбОЮ checksec --file=./writeme ОЋОЋОЊОљОЋ ОЕОљОфОЮ ОеОЋОљОЎОЮ No PIE, No canary, NX enabled, Partial RELRO.

    ОфОеОњОЎОю 1 - ОАОЎОЋОе ОЋОъОдОЎОљОф offset¶

    ОюОцОаОЎ ОЕОЏОЋОфОЉОЎОЮ, ОдОеОЎОџ ОюОЊОбОф ОљОЎОцОћ ОћОДОюОў ОЕОюОаОЋ ОЎОЋОЕОЉ ОбОю ОћОъОЌОАОаОЎОф.

    ОћОъОўОеОћ:

    1. ОЕОюОцОЋ ОљОф ОЏОфОЋОЉОф secret ОЉОЕОфОЎ ОЊОеОЏОЎОЮ: ОцОбОЮ ОбОЮ nm writeme | grep ' secret', ОЋОцОбОЮ ОъОћОЕОЋОеОћ ОЕОћОфОЋОЏОаОЎОф ОъОЊОцОЎОАОћ ОЉОеОЎОдОћ. ОЋОЊОљОЋ ОЕОћОЪ ОќОћОЋОф.
    2. ОъОдОљОЋ ОљОф Оћ-offset - ОћОљОЎОаОЊОДОА ОЕОЉОЋ ОћОДОюОў ОЕОюОЏОЮ ОъОфОЌОЎОю ОЉОљОеОњОЋОъОаОўОЎОЮ ОЕОю printf. ОЕОюОЌОЋ ОДОюОў ОЏОъОЋ AAAAAAAA.%p.%p.%p.%p.%p.%p.%p.%p ОЋОќОћОЋ ОЉОљОЎОќОћ %p ОъОЋОцОЎОб 0x4141414141414141.
    3. ОеОЕОъОЋ ОюОбОдОъОЏОЮ ОљОф ОћОъОАОцОе ОћОќОћ. ОЏОю ОЕОљОе ОћОфОеОњОЎОю ОфОюОЋОЎ ОЉОЋ.

    ОеОъОќ: ОЉ-x86-64 ОЌОъОЕОф ОћОљОеОњОЋОъОаОўОЎОЮ ОћОеОљОЕОЋОаОЎОЮ ОъОњОЎОбОЎОЮ ОъОеОњОЎОАОўОеОЎОЮ, ОљОќ ОћОДОюОў ОЕОбОю ОћОъОЌОАОаОЎОф ОъОфОЌОЎОю ОЉОЊОеОџ ОЏОюОю ОЉОљОЎОаОЊОДОА 6 ОЋОъОбОюОћ. ОљОю ОфОаОЎОЌОЋ - ОфОъОЊОЊОЋ.

    ОеОъОќ: ОљОЮ ОДОЕОћ ОюОАОцОЋОе, ОфОаОЋ ОюОЏОю %p ОъОцОеОЎОЊ ОЉОеОЋОе ОбОЮ ОаОДОЋОЊОЋОф, ОЋОфОЕОЋОЋ ОљОф ОъОАОцОе ОћОаОДОЋОЊОЋОф ОбОЊ ОЕОљОфОЮ ОеОЋОљОЎОЮ ОљОф ОеОдОБ Оћ-41-ОЎОЮ.

    ОфОеОњОЎОю 2 - ОЏОфОЎОЉОф ОЉОЎОф ОЉОЋОЊОЊ ОбОЮ %hhn¶

    ОаОфОЌОЎОю ОЉОДОўОЪ: ОюОЏОфОЋОЉ ОбОеОџ ОЕОю ОЉОЎОф ОљОЌОЊ ОЏОЊОЎ ОюОћОеОњОЎОЕ ОљОф ОћОъОаОњОаОЋОЪ.

    ОћОъОўОеОћ:

    1. ОЉОаОЋ ОДОюОў ОЕОЏОЋОфОЉ ОљОф ОћОбОеОџ 0x41 (65 ОфОЋОЋОЎОЮ) ОюОЉОЎОф ОћОаОъОЋОџ ОЉОЎОЋОфОе ОЕОю secret, ОЏОюОЋОъОе ОюОЏОфОЋОЉОф &secret + 0.
    2. ОћОЕОфОъОЕОЋ ОЉОдОЋОеОћ %<ОеОЋОЌОЉ>c%<ОљОЎОаОЊОДОА>$hhn. ОЌОЕОЉОЋ: ОЏОъОћ ОфОЋОЋОЎОЮ ОдОеОЎОџ ОюОћОЊОцОЎОА ОюОцОаОЎ Оћ-%hhn?
    3. ОЋОЊОљОЋ ОЉОцОюОў ОЕОћОЕОЋОеОћ secret = ... ОъОАОфОЎОЎОъОф ОЉ-41.

    ОеОъОќ: ОћОЏОфОЋОЉОф &secret ОдОеОЎОЏОћ ОюОЕОЉОф ОЉОљОеОњОЋОъОаОў ОЕОљОюОЎОЋ ОћОљОЎОаОЊОДОА ОЉОфОЋОџ %hhn ОъОдОЉОЎОб. ОЕОЎОъОЋ ОљОЋОфОћ ОЉОАОЋОБ Оћ-payload (ОЉОњОюОю Оћ-null ОЉОфОЎОЮ), ОъОеОЋОцОЊОф Оюalignment ОЕОю 8, ОЋОЌОЕОЉОЋ ОљОф ОћОљОЎОаОЊОДОА: offset + ОъОАОцОе Оћ-qwords ОЕОю ОЌОюОД Оћformat.

    ОеОъОќ: %65c ОъОЊОцОЎОА 65 ОфОЋОЋОЎОЮ. ОљОЮ ОдОЎОеОцОфОЮ ОбОЋОЊ ОфОЋОЋОЎ padding ОюОцОаОЎ ОќОћ, ОќОЏОеОЋ ОюОАОцОЋОе ОњОЮ ОљОЋОфОЮ ОЉОъОЋОаОћ.

    ОфОеОњОЎОю 3 - ОЏОфОЎОЉОћ ОъОюОљОћ ОЉОЎОЊОЎОЎОЮ ОбОЮ %hn¶

    ОбОЏОЕОЎОЋ ОћОЎОбОЊ ОћОъОюОљ: ОюОЏОфОЋОЉ ОљОф ОћОбОеОџ 0x1337c0de Ою-secret ОЏОџ ОЕОћОфОЋОЏОаОЎОф ОфОЊОцОЎОА WIN. ОбОЕОЋ ОљОф ОќОћ ОЉОюОЎ fmtstr_payload.

    ОћОъОўОеОћ:

    1. ОцОеОДОЋ ОљОф 0x1337c0de ОюОЕОаОЎ ОаОфОЌОЎОЮ ОЕОю 16 ОЉОЎОў: ОћОаОфОЌ ОћОаОъОЋОџ (0xc0de) ОюОЏОфОЋОЉОф &secret+0, ОЋОћОњОЉОЋОћ (0x1337) ОюОЏОфОЋОЉОф &secret+2.
    2. ОъОЎОЎОаОЋ ОљОф ОћОаОфОЌОЎОЮ ОЉОАОЊОе ОбОЋОюОћ ОюОцОЎ ОћОбОеОџ ОЕОеОЋОдОЎОЮ ОюОЏОфОЋОЉ. ОъОЎ ОдОеОЎОџ ОюОћОЎОЏОфОЉ ОДОЋОЊОЮ?
    3. ОЉОаОЋ ОљОф ОъОЌОеОЋОќОф Оћformat: ОћОЊОцОЎОАОЋ ОљОф ОћОћОцОеОЕ ОЉОЎОЪ ОаОфОЌ ОюОаОфОЌ, ОЋОљОЌОеОЎ ОЏОю ОћОцОеОЕ ОЉОдОбОЋ %<idx>$hn ОюОЏОфОЋОЉОф ОћОъОфОљОЎОъОћ.
    4. ОћОдОъОЎОЊОЋ ОљОф ОЕОфОЎ ОћОЏОфОЋОЉОЋОф ОЉОАОЋОБ Оћ-payload, ОъОеОЋОцОЊОЋОф Оюalignment, ОЋОЏОЋОЋОаОЋ ОљОф ОћОљОЎОаОЊОДОАОЎОЮ.
    5. ОћОеОЎОдОЋ ОЋОЋОЊОљОЋ ОЕОљОфОЮ ОеОЋОљОЎОЮ WIN: secret overwritten exactly!.

    ОеОъОќ: 0x1337 (4919) ОДОўОЪ Оъ-0xc0de (49374), ОљОќ ОћОаОфОЌ ОћОњОЉОЋОћ ОдОеОЎОџ ОюОћОЎОЏОфОЉ ОДОЋОЊОЮ. ОљОЌОеОЎОЋ ОъОЊОцОЎОАОЎОЮ ОбОЋОЊ 49374 - 4919 ОфОЋОЋОЎОЮ ОЏОЊОЎ ОюОћОњОЎОб Ою-0xc0de.

    ОеОъОќ: ОљОЮ ОљОфОЮ ОъОфОДОЕОЎОЮ ОюОЎОЎОдОЉ ОљОф ОћОљОЎОаОЊОДОА, ОеОцОЊОЋ ОљОф ОЌОюОД Оћformat ОюОљОЋОеОџ ОДОЉОЋОб (ОЏОцОЋОюОћ ОЕОю 8) ОбОЮ ОфОЋОЋОЎ a ОъОЎОЋОфОеОЎОЮ, ОЋОљОќ ОћОљОЎОаОЊОДОА ОЕОю ОћОЏОфОЋОЉОф ОћОеОљОЕОЋОаОћ ОћОЋОљ ОЉОЊОЎОЋОД offset + FMT_QWORDS.

    ОеОъОќ: ОљОЌОеОЎ ОЕОќОћ ОбОЋОЉОЊ, ОаОАОЋ ОЉОъОЏОЋОЋОЪ ОюОћОЌОюОЎОБ ОљОф ОАОЊОе ОћОЏОфОЎОЉОЋОф (ОДОЋОЊОЮ ОћОаОъОЋОџ ОЋОљОќ ОћОњОЉОЋОћ) ОЋОфОеОљОЋ ОљОЎОџ ОќОћ ОаОЕОЉОе. ОќОћ ОЎОЌОЊОЊ ОюОЏОЮ ОюОъОћ ОћОАОЊОе ОћОбОЋОюОћ ОДОеОЎОўОЎ.

    ОфОеОњОЎОю 4 - ОљОЋОфОЋ ОћОЊОЉОе ОбОЮ fmtstr_payload¶

    ОбОЏОЕОЎОЋ ОЏОЕОћОЉОаОфОЮ ОљОф ОћОЏОљОЉ, ОфОаОЋ ОюОЏОюОЎ ОюОбОЕОЋОф ОљОф ОћОбОЉОЋОЊОћ.

    ОћОъОўОеОћ:

    1. ОћОЕОЎОњОЋ ОљОф ОљОЋОфОћ ОЏОфОЎОЉОћ ОЕОю 0x1337c0de Ою-secret ОбОЮ ОЕОЋОеОћ ОљОЌОф ОЕОю fmtstr_payload(offset, {addr: value}, write_size='short').
    2. ОћОеОЎОдОЋ ОњОЮ ОбОЮ write_size='byte' ОЋОњОЮ ОбОЮ 'short'. ОћОЕОЋОЋ ОљОф ОљОЋОеОџ Оћ-payload ОЕОаОЋОдОе (len(payload)) ОЋОљОф ОъОАОцОе ОћОфОЋОЋОЎОЮ ОЕОъОЋОЊОцОАОЎОЮ. ОљОЎОќОћ ОъОћОЮ "ОеОЋОбОЕ" ОЎОЋОфОе?
    3. ОћОЊОцОЎОАОЋ ОљОф Оћ-payload ОбОдОъОЋ (print(payload)) ОЋОћОЕОЋОЋ ОљОЋОфОЋ ОюОъОћ ОЕОЉОаОЎОфОЮ ОЎОЊОаОЎОф ОЉОфОеОњОЎОю 3. ОъОћ ОЊОЋОъОћ ОЋОъОћ ОЕОЋОаОћ?

    ОеОъОќ: ОљОЮ ОћОфОЋОЏОаОЎОф ОъОЊОцОЎОАОћ ОъОЕОћОЋ ОюОцОаОЎ ОъОЌОеОЋОќОф Оћformat ОЕОюОЏОЮ (ОЏОљОЪ ОћОЎОљ ОюОљ, ОљОЉОю ОЉОљОфОњОеОЎОЮ ОљОЌОеОЎОЮ ОЏОЪ), ОћОбОЉОЎОеОЋ ОљОф ОъОАОцОе ОћОфОЋОЋОЎОЮ ОЊОеОџ numbwritten=.

    ОеОъОќ: fmtstr_payload ОъОаОЎОЌ ОЕОћ-payload ОЕОюОЏОЮ ОъОфОЌОЎОю ОЉОЊОЎОЋОД ОЉ-offset. ОљОЮ ОЕОЎОаОЎОфОЮ ОљОф ОъОЉОаОћ ОћОДОюОў (ОћОЋОАОцОфОЮ prefix), ОбОЊОЏОаОЋ ОљОф Оћ-offset ОЉОћОфОљОЮ.

    ОфОеОњОЎОю 5 (ОљОфОњОе) - ОЏОфОЎОЉОф ОЏОфОЋОЉОф ОЕОюОъОћ ОЕОю 8 ОЉОфОЎОЮ¶

    ОбОЊ ОбОЏОЕОЎОЋ ОЏОфОЉОаОЋ ОбОеОџ "ОДОўОЪ". ОбОЏОЕОЎОЋ ОаОЏОфОЋОЉ ОЏОфОЋОЉОф ОъОюОљОћ, ОЏОъОЋ ОЕОаОдОўОеОџ ОЏОЕОаОЊОеОЋОА ОъОдОЉОЎОб ОцОЋОаОДОдОЎОћ ОЉОЕОЎОбОЋОеОЎОЮ ОћОЉОљОЎОЮ.

    ОћОъОўОеОћ:

    1. ОЕОаОЋ ОљОф writeme.c ОЏОџ ОЕ-secret ОЎОљОЋОфОЌОю ОюОъОдОЉОЎОб ОЏОюОЕОћОЋ (ОюОъОЕОю ОћОЏОфОЋОЉОф ОЕОю puts ОЕОљОфОЮ ОъОЊОцОЎОАОЎОЮ), ОЋОћОъОўОеОћ ОфОћОЎОћ ОюОЊОеОЋОА ОљОЋОфОЋ ОЉОбОеОџ ОЕОеОЎОеОЋОфОЎ ОЉОЪ 8 ОЉОфОЎОЮ ОЏОъОЋ 0xdeadbeefcafe1234.
    2. ОбОЮ fmtstr_payload ОЋ-write_size='short', ОЏОфОЉОЋ ОљОф ОћОбОеОџ ОћОъОюОљ. ОљОъОфОЋ ОбОЮ gdb ОЕОЏОю ОљОеОЉОбОф ОћОаОфОЌОЎОЮ ОаОЏОфОЉОЋ ОаОЏОЋОЪ: x/gx &secret.
    3. ОбОЏОЕОЎОЋ ОаОАОЋ ОюОЉОаОЋОф ОљОф ОќОћ ОЉОЎОЊОЎОЎОЮ: ОљОеОЉОб ОЏОфОЎОЉОЋОф %hn ОюОЏОфОЋОЉОЋОф &secret+0/+2/+4/+6, ОъОъОЋОЎОаОЋОф ОЉОАОЊОе ОбОЋОюОћ, ОбОЮ ОўОЎОцОЋОю ОЉoverflow ОЏОЕОћОбОеОџ ОћОЉОљ ОДОўОЪ ОъОћОъОЋОаОћ.

    ОеОъОќ: ОћОбОеОџ 0xdeadbeefcafe1234 ОъОфОцОеОД ОюОаОфОЌОЎОЮ 0x1234, 0xcafe, 0xbeef, 0xdead (ОюОЏОфОЋОЉОЋОф +0, +2, +4, +6). ОъОЎОЎОаОЋ ОљОЋОфОЮ ОюОцОЎ ОћОбОеОџ ОЉОАОЊОе ОбОЋОюОћ, ОЋОфОДОЉОюОЋ ОћОцОеОЕОЎОЮ ОЌОЎОЋОЉОЎОЎОЮ ОЉОюОЎ ОдОЋОеОџ ОЉoverflow. overflow ОаОЊОеОЕОф ОеОД ОљОЮ ОЕОаОЎ ОаОфОЌОЎОЮ ОЊОЋОеОЕОЎОЮ ОљОЋОфОЋ ОбОеОџ (ОћОћОцОеОЕ ОЎОЋОдОљ 0), ОЋОљОќ ОъОЊОцОЎОАОЎОЮ ОъОЌОќОЋОе ОЕОюОЮ ОЕОю 0x10000.

    ОеОъОќ: ОЏОЕОћОъОЋОаОћ ОЏОЉОе ОњОЊОЋОю ОЋОћОаОфОЌ ОћОЉОљ ОДОўОЪ, ОћОћОцОеОЕ (chunk - printed) % 0x10000 ОаОЋОфОЪ ОюОЏОЮ ОЉОЊОЎОЋОД ОЏОъОћ ОбОЋОЊ ОюОћОЊОцОЎОА. ОљОЮ ОЎОдОљ 0, ОћОЊОцОЎОАОЋ 0x10000 ОЕОюОЮ (ОъОЌОќОЋОе).

    ОеОъОќ: ОљОъОфОЋ ОЏОю ОаОфОЌ ОЉОаОцОеОЊ ОЉ-gdb. ОљОЮ ОЉОЎОф ОљОЌОЊ ОЕОњОЋОЎ, ОАОЎОъОЪ ОЕОћОъОЋОаОћ ОљОЋ ОћОљОЎОаОЊОДОА ОЕОю ОљОЋОфОЋ ОаОфОЌ ОюОљ ОъОЊОЋОЎОД - ОЉОЊОДОЋ ОљОф Оћalignment ОЕОю ОЉОюОЋОД ОћОЏОфОЋОЉОЋОф.
