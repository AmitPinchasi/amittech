    7.2 - ОЕОЎОъОЋОЕ ОюОљОЌОе ОЕОЌОеОЋОе - ОћОеОдОљОћ

    ОбОЊ ОбОЏОЕОЎОЋ ОЏОю ОћОЌОЋОюОЕОЋОф ОЕОфОДОцОаОЋ ОЌОЎОЋ ОбОю ОћОъОЌОАОаОЎОф - overflows, ОЕОюОЎОўОћ ОЉ-RIP, ROP. ОбОЏОЕОЎОЋ ОљОаОЌОаОЋ ОЎОЋОеОЊОЎОЮ ОюОќОЎОеОћ ОћОЕОаОЎОЎОћ ОЋОћОъОбОаОЎОЎОаОф ОюОљ ОцОЌОЋОф: Оћheap. ОЉОЕОЎОбОЋОе ОћОДОЋОЊОЮ (7.1) ОћОЏОеОаОЋ ОљОф ОћОцОаОЎОъОЎОЋОф ОЕОю ОћОъОДОдОћ ОЕОю glibc - ОъОћ ОќОћ chunk, ОљОЎОџ ОаОеОљОЎОЮ Оћ-bins, ОЋОъОћ ОДОЋОеОћ Ою-chunk ОЏОЕОъОЕОЌОеОеОЎОЮ ОљОЋОфОЋ. ОбОЏОЕОЎОЋ ОаОЎОДОЌ ОљОф ОћОЎОЊОб ОћОќОћ ОЋОаОЉОаОћ ОъОъОаОЋ ОљОф ОљОЌОф ОћОЌОЋОюОЕОЋОф ОћОЏОЎ ОаОцОЋОдОЋОф ОЋОъОАОЋОЏОаОЋОф ОЉОбОЋОюОЮ ОћОљОъОЎОфОЎ: ОЕОЎОъОЋОЕ ОюОљОЌОе ОЕОЌОеОЋОе. ОќОЋ ОћОЌОЋОюОЕОћ ОЕОъОцОЎОюОћ ОЊОцОЊОцОаОЎОЮ, kernels ОЋОъОЏОЋОаОЋОф ОЋОЎОеОўОЋОљОюОЎОЋОф, ОЋОЏОљОЪ ОаОюОъОЊ ОљОЋОфОћ ОбОю ОЊОЋОњОъОћ ОаОДОЎОЎОћ ОЋОъОЊОЋОЎОДОф - ОћОљОфОњОе ОћОљОњОЊОЎ uaf Оъ-pwnable.kr, ОЕОЉОЋ ОаОЌОўОЋОБ ОљОф ОќОеОЎОъОф ОћОЉОДОеОћ ОЕОю ОфОЋОЏОаОЎОф C++ ОЊОеОџ ОЊОеОЎОАОф ОъОдОЉОЎОб Ою-vtable.

    ОћОеОбОЎОЋОЪ ОЉОЉОАОЎОА ОцОЕОЋОў ОюОћОЌОеОЎОЊ: ОљОЌОеОЎ ОЕ-free ОъОЕОЌОеОе ОЉОюОЋОД ОќОЎОЏОеОЋОЪ, ОћОъОдОЉОЎОб ОћОЎОЕОЪ ОбОЊОЎОЎОЪ ОДОЎОЎОЮ ОЋОбОЊОЎОЎОЪ ОъОдОЉОЎОб ОюОЕОЮ. ОљОЮ ОаОдОюОЎОЌ ОюОњОеОЋОЮ ОюОъОДОдОћ ОюОћОЌОќОЎОе ОљОф ОљОЋОфОЋ ОЉОюОЋОД ОюОћОДОдОљОћ ОЌОЊОЕОћ ОЕОљОаОЌОаОЋ ОЕОЋОюОўОЎОЮ ОЉОфОЋОЏОЪ ОЕОюОћ, ОљОќ ОћОъОдОЉОЎОб ОћОЎОЕОЪ ОцОфОљОЋОЮ "ОДОЋОеОљ" ОљОф ОћОаОфОЋОаОЎОЮ ОЕОюОаОЋ. ОЋОЏОЕОћОъОдОЉОЎОб ОћОЎОЕОЪ ОћОЋОљ ОъОдОЉОЎОб ОюОљОЋОЉОЎОЎОДОў ОбОЮ ОцОЋОаОДОдОЎОЋОф ОЋОЎОеОўОЋОљОюОЎОЋОф - ОћОЕОюОЎОўОћ ОЉОфОЋОЏОЪ ОћОЋОцОЏОф ОюОЕОюОЎОўОћ ОбОю ОћОДОеОЎОљОћ ОћОЉОљОћ ОЕОю ОцОЋОаОДОдОЎОћ.

    ОъОдОЉОЎОб ОфОюОЋОЎ - Dangling Pointer¶

    ОЉОЋОљОЋ ОаОфОЌОЎОю ОъОћОЉОљОњ ОбОдОъОЋ. ОЉ-C ОЋОЉ-C++ ОћОцОЋОаОДОдОЎОћ free (ОљОЋ ОћОљОЋОцОеОўОЋОе delete) ОъОЌОќОЎОеОћ ОЉОюОЋОД ОќОЎОЏОеОЋОЪ ОюОъОДОдОћ, ОљОЉОю ОћОЎОљ ОюОљ ОаОЋОњОбОф ОЉОъОЕОфОаОћ ОЕОъОЌОќОЎОД ОљОф ОћОЏОфОЋОЉОф. ОћОъОдОЉОЎОб ОъОъОЕОЎОџ ОюОћОдОЉОЎОб ОЉОЊОЎОЋОД ОюОљОЋОфОЋ ОъОДОЋОЮ - ОеОД ОЕОћОъОДОЋОЮ ОћОќОћ ОЏОЉОе ОюОљ ОЕОЎОЎОџ ОюОаОЋ.

    char *p = malloc(0x20);
strcpy(p, "hello");
free(p);          // the block went back to the allocator
// p still holds the same address! This is a dangling pointer
puts(p);          // use-after-free read - may print garbage or new data
p[0] = 'H';       // use-after-free write - overwrites data that's no longer ours

    ОћОъОЋОаОЌ "ОЕОЎОъОЋОЕ ОюОљОЌОе ОЕОЌОеОЋОе" ОъОЏОАОћ ОЏОю ОњОЎОЕОћ ОюОъОдОЉОЎОб ОЏОќОћ ОљОЌОеОЎ Оћ-free: ОДОеОЎОљОћ, ОЏОфОЎОЉОћ, ОљОЋ - ОћОЏОЎ ОъОАОЋОЏОЪ - ОДОеОЎОљОћ ОЕОю ОъОдОЉОЎОб ОцОЋОаОДОдОЎОћ ОъОфОЋОџ ОћОЉОюОЋОД ОЋОљОќ ОДОцОЎОдОћ ОљОюОЎОЋ.

    ОюОъОћ ОќОћ ОъОАОЋОЏОЪ ОЋОюОљ ОАОфОЮ ОЉОљОњ? ОЏОЎ ОћОъОДОдОћ ОъОъОЌОќОе ОќОЎОЏОеОЋОЪ. ОћОЉОюОЋОД ОЕОЕОЌОеОеОаОЋ ОюОљ ОаОъОЌОД ОЋОюОљ ОъОфОљОцОА - ОћОЋОљ ОаОЏОаОА ОюОеОЕОЎОъОф ОцОаОЋОЎОЎОЮ ОЋОъОъОфОЎОЪ ОюОћОДОдОљОћ ОћОЉОљОћ ОЉОљОЋОфОЋ ОњОЋОЊОю. ОљОЮ ОћОфОЋОДОБ ОЎОЏОЋОю ОюОњОеОЋОЮ ОюОћОДОдОљОћ ОЌОЊОЕОћ ОЉОњОЋОЊОю ОћОќОћ, ОћОЋОљ ОЎОЕОфОюОў ОбОю ОљОЋОфОЮ ОЉОфОЎОЮ ОЉОЊОЎОЋОД, ОЋОћОъОдОЉОЎОб ОћОфОюОЋОЎ ОЎОДОеОљ ОъОбОЏОЕОЎОЋ ОљОф ОъОћ ОЕОћОфОЋОДОБ ОЏОфОЉ.

    Step 1: there is a live object, two pointers point to it
   obj ------> +------------------+
   (ours)      | vptr | age | name |   block of size 0x20
              +------------------+

Step 2: free(obj). The pointer hasn't changed, but the block is now in the free list
   obj ------> +------------------+
   (dangling!) |  fd  | age | name |   <-- the allocator wrote fd at the start
              +------------------+

Step 3: a new malloc(0x20) that we fill -> returns the same block
   obj -----> +------------------+
   (dangling) \  | AAAA | BBBB |...  |   our data!
   new ----/  +------------------+

    ОЕОЎОъОЋ ОюОЉ ОюОцОеОў ОДОўОЪ ОЋОДОеОЎОўОЎ ОЉОЕОюОЉ 2: ОЏОЕОћОъОДОдОћ ОъОЕОЌОеОе ОЉОюОЋОД ОЋОъОЏОаОЎОА ОљОЋОфОЋ Ою-bin, ОћОЋОљ ОЏОЋОфОЉ ОљОф ОъОдОЉОЎОб Оћ-fd (ОћОъОдОЉОЎОб ОюОЉОюОЋОД ОћОЉОљ ОЉ-bin) ОюОфОЋОџ ОЕОъОЋОаОф ОћОЉОфОЎОЮ ОћОеОљОЕОЋОаОЎОЮ ОЕОю ОљОќОЋОе ОћОъОЕОфОъОЕ. ОљОюОћ ОЉОЊОЎОЋОД ОћОЉОфОЎОЮ ОЕОЉОћОЮ ОЎОЋОЕОЉ Оћ-vptr ОЕОю ОљОЋОЉОЎОЎОДОў C++. ОЏОюОЋОъОе, ОбОдОЮ ОћОЕОЌОеОЋОе ОЏОЉОе ОъОЕОЌОЎОф ОюОаОЋ ОљОф Оћ-vptr ОћОЎОЕОЪ. ОќОЋ ОћОАОЎОЉОћ ОЕОљОАОЋОе ОцОЕОЋОў ОюОДОеОЋОљ ОюОљОЋОЉОЎОЎОДОў ОљОЌОеОЎ free ОЉОюОЎ ОюОъОЌОќОе ОљОЋОфОЋ ОДОЋОЊОЮ - ОаОдОюОЋОю ОюОќОћ ОЉОћОъОЕОџ.

    ОъОаОњОаОЋОЪ ОћОъОЎОЌОќОЋОе - tcache ОЋ-bins¶

    ОЏОЊОЎ Оюexploit UAF ОдОеОЎОџ ОюОЕОюОЋОў ОЉОЕОљОюОћ: ОЏОЕОљОаОЎ ОДОЋОеОљ Ою-malloc, ОљОЎОќОћ ОЉОюОЋОД ОљОаОЎ ОъОДОЉОю? ОћОфОЕОЋОЉОћ ОаОДОЉОбОф ОбОю ОЎОЊОЎ Оћ-bins ОЕОю glibc, ОЋОЉОъОЎОЋОЌОЊ Оћ-tcache.

    ОъОљОќ glibc 2.26 ОЎОЕ ОюОЏОю thread ОъОўОъОЋОЪ ОцОеОўОЎ ОЉОЕОЮ tcache. ОюОЏОю ОъОЌОюОДОф ОњОЋОЊОю (0x20, 0x30, 0x40, ...) ОЎОЕ ОеОЕОЎОъОћ ОъОДОЋОЕОеОф ОъОЕОюОћ, ОЋОћОЎОљ ОъОфОаОћОњОф ОЏОъОЋ ОъОЌОАОаОЎОф - LIFO. ОЏОюОЋОъОе, ОћОЉОюОЋОД ОЕОЕОЋОЌОеОе ОљОЌОеОЋОЪ ОћОЋОљ ОћОеОљОЕОЋОЪ ОЕОЎОЋОДОдОћ ОъОЌОЊОЕ.

    free(A);  free(B);  free(C);      // all the same size, say 0x20
tcache[0x20] head -> C -> B -> A -> NULL

malloc(0x20)  ->  returns C   (the head)
malloc(0x20)  ->  returns B
malloc(0x20)  ->  returns A

    ОќОћ ОЉОЊОЎОЋОД ОъОћ ОЕОаОЋОфОЪ ОюОаОЋ ОЕОюОЎОўОћ. ОљОЮ ОћОфОЋОЏОаОЎОф ОъОЕОЌОеОеОф ОЕОаОЎ ОљОЋОЉОЎОЎОДОўОЎОЮ m ОЋОљОќ w, ОћОЮ ОаОЏОаОАОЎОЮ Ою-tcache ОЉОАОЊОе: ОћОеОљОЕ ОћОЋОљ w (ОЕОЋОЌОеОе ОљОЌОеОЋОЪ), ОљОЌОеОЎОЋ m. ОЕОфОЎ ОћОДОдОљОЋОф ОЌОЊОЕОЋОф ОЉОњОЋОЊОю ОћОќОћ ОЎОЌОќОЎОеОЋ ОДОЋОЊОЮ ОљОф w ОЋОљОќ ОљОф m - ОЋОљОаОЌОаОЋ ОаОъОюОљ ОљОф ОЕОаОЎОћОЮ ОЉОаОфОЋОаОЎОЮ ОЕОюОаОЋ.

    ОћОбОеОћ ОЌОЕОЋОЉОћ ОбОю ОњОеОАОљОЋОф: Оћ-tcache ОДОЎОЎОЮ Оъ-glibc 2.26 ОЋОљОЎОюОџ. ОћОЉОЎОаОљОеОЎ ОЕОю uaf ОЉ-pwnable.kr ОћОЋОЊОе ОбОю ОАОЉОЎОЉОћ ОЎОЕОаОћ ОЎОЋОфОе (Ubuntu ОбОЮ glibc 2.23), ОЕОЮ ОљОЎОЪ tcache ОЋОћОЉОюОЋОДОЎОЮ ОћОДОўОаОЎОЮ ОаОЏОаОАОЎОЮ Ою-fastbins. ОљОЉОю ОюОдОЋОеОџ UAF ОќОћ ОюОљ ОъОЕОаОћ ОЏОъОбОў ОЏОюОЋОЮ: ОњОЮ fastbin ОЋОњОЮ tcache ОћОЮ ОеОЕОЎОъОЋОф LIFO ОюОцОЎ ОњОЋОЊОю, ОљОќ ОљОЋОфОћ ОћОДОдОљОћ ОЌОЋОќОеОф ОЉОЊОЎОЋОД ОЉОљОЋОфОЋ ОљОЋОцОЪ. ОћОўОЏОаОЎОДОћ ОќОћОћ. ОЉОЊОДОЋ ОбОю ОљОЎОќОћ glibc ОљОфОЮ ОеОдОЎОЮ ОбОЮ:

    ldd --version | head -1

    ОЕОюОЋОЕОф ОћОцОаОЎОЮ ОЕОю UAF - ОДОеОЎОљОћ, ОЏОфОЎОЉОћ, ОЋОДОеОЎОљОћ-ОюОЉОЎОдОЋОб¶

    ОюОљ ОЏОю UAF ОЕОЋОЋОћ. ОЌОЕОЋОЉ ОюОћОЉОЎОЪ ОъОћ ОЉОЊОЎОЋОД ОљОаОЌОаОЋ ОЎОЏОЋОюОЎОЮ ОюОбОЕОЋОф ОбОЮ ОћОъОдОЉОЎОб ОћОфОюОЋОЎ:

    • ОДОеОЎОљОћ ОюОљОЌОе ОЕОЌОеОЋОе - ОљОЮ ОћОфОЋОЏОаОЎОф ОДОЋОеОљОф ОъОћОЉОюОЋОД ОћОъОЕОЋОЌОеОе ОљОЌОеОЎ ОЕОъОЎОюОљОаОЋ ОљОЋОфОЋ, ОќОћ ОбОеОЋОЦ leak ОъОЎОЊОб ОъОдОЋОЎОЪ. ОљОцОЕОе ОюОћОЕОфОъОЕ ОЉОќОћ ОЏОЊОЎ Оюleak ОЏОфОЋОЉОЋОф heap ОљОЋ libc ОЋОюОбОДОЋОБ ASLR.
    • ОЏОфОЎОЉОћ ОюОљОЌОе ОЕОЌОеОЋОе - ОљОЮ ОћОфОЋОЏОаОЎОф ОЏОЋОфОЉОф ОЊОеОџ ОћОъОдОЉОЎОб ОћОфОюОЋОЎ, ОљОаОЌОаОЋ ОЎОЏОЋОюОЎОЮ ОюОћОЕОЌОЎОф ОЉОюОЋОД ОЌОЎ ОљОЌОе ОЕОъОЋОДОдОћ ОбОю ОљОЋОфОЋ ОъОДОЋОЮ, ОљОЋ ОюОћОЕОЌОЎОф metadata ОЕОю ОћОъОДОдОћ.
    • ОДОеОЎОљОћ-ОюОЉОЎОдОЋОб - ОќОћ ОћОќОћОЉ. ОљОЮ ОћОЉОюОЋОД ОћОъОЕОЋОЌОеОе ОъОЏОЎОю ОъОдОЉОЎОб ОцОЋОаОДОдОЎОћ, ОЋОћОфОЋОЏОаОЎОф ОДОЋОеОљОф ОЊОеОЏОЋ ОюОцОЋОаОДОдОЎОћ ОљОЌОеОЎ ОЕОъОЎОюОљОаОЋ ОљОЋОфОЋ, ОљОаОЌОаОЋ ОЕОЋОюОўОЎОЮ ОбОю Оћ-RIP ОЎОЕОЎОеОЋОф. ОЉ-C++ ОќОћ ОЉОЊОЎОЋОД ОъОћ ОЕОДОЋОеОћ ОбОЮ Оћ-vtable ОЕОю ОљОЋОЉОЎОЎОДОў.

    ОЉОЕОЎОбОЋОе ОћОќОћ ОаОфОъОДОЊ ОЉОцОЪ ОћОЕОюОЎОЕОЎ, ОЏОЎ ОћОЋОљ ОћОЏОЎ ОЌОЊ ОЋОЏОЎ ОћОЋОљ ОЉОЊОЎОЋОД ОъОћ ОЕОљОфОњОе uaf ОЉОЋОаОћ. ОЏОЊОЎ ОюОћОЉОЎОЪ ОљОЋОфОЋ ОдОеОЎОџ ОюОЊОбОф ОљОЎОџ ОљОЋОЉОЎОЎОДОў C++ ОбОЮ ОцОЋОаОДОдОЎОЋОф ОЋОЎОеОўОЋОљОюОЎОЋОф ОЉОаОЋОЎ ОЉОќОЎОЏОеОЋОЪ.

    ОљОЋОЉОЎОЎОДОў C++ ОЋОўОЉОюОф ОћОцОЋОаОДОдОЎОЋОф ОћОЋОЋОЎОеОўОЋОљОюОЎОф - vtable¶

    ОЏОЕОъОЌОюОДОћ ОЉ-C++ ОъОЏОеОЎОќОћ ОбОю ОцОЋОаОДОдОЎОћ virtual, ОћОъОћОЊОе ОюОљ ОЎОЏОЋОю ОюОЊОбОф ОЉОќОъОЪ ОДОЋОъОцОЎОюОдОЎОћ ОљОЎОќОЋ ОњОеОАОћ ОЕОю ОћОцОЋОаОДОдОЎОћ ОфОЎОДОеОљ (ОЕОю ОъОЌОюОДОф ОћОЉОАОЎОА ОљОЋ ОЕОю ОћОЎОЋОеОЕОф). ОюОЏОЪ ОћОЋОљ ОЉОЋОаОћ ОўОЉОюОф ОцОЋОаОДОдОЎОЋОф ОЋОЎОеОўОЋОљОюОЎОЋОф - vtable: ОъОбОеОџ ОЕОю ОъОдОЉОЎОбОЎ ОцОЋОаОДОдОЎОћ, ОљОЌОЊ ОюОЏОю ОъОфОЋОЊОћ ОЋОЎОеОўОЋОљОюОЎОф, ОюОцОЎ ОАОЊОе ОћОћОЏОеОќОћ. ОЏОю ОљОЋОЉОЎОЎОДОў ОЕОю ОъОЌОюОДОћ ОЏОќОЋ ОъОфОЌОЎОю ОЉОъОдОЉОЎОб ОюОўОЉОюОћ ОћОќОЋ - Оћ-vptr.

    ОаОАОфОЏОю ОбОю ОъОЉОаОћ ОЕОъОЊОъОћ ОљОф uaf:

    class Human {
    virtual void give_shell() { system("/bin/sh"); }  // first virtual method -> slot 0
protected:
    int age;
    string name;
public:
    virtual void introduce() { /* ... */ }             // second virtual method -> slot 1
};
class Man : public Human {
    void introduce() { /* Man ... */ }                 // overrides introduce (slot 1)
};

    ОЉОљОЋОЉОЎОЎОДОў Man ОЉОќОЎОЏОеОЋОЪ (64 ОЉОЎОў, ОбОЮ string ОъОАОЋОњ ОЎОЕОЪ ОЕОћОЋОљ ОъОдОЉОЎОб ОЎОЌОЎОЊ) ОћОъОЉОаОћ ОћОЋОљ:

    Man object (0x18 = 24 bytes of user data)
+0x00  vptr  ------------------>  Man's vtable
+0x08  int age (with padding to 8)
+0x10  string name (pointer)

Man's vtable:
+0x00  give_shell   <-- slot 0  (inherited from Human, not overridden)
+0x08  Man::introduce <-- slot 1

    Оћ-vptr ОЉОљОЋОЉОЎОЎОДОў ОъОдОЉОЎОб ОюОфОЌОЎОюОф ОљОќОЋОе Оћ-slots (slot 0). ОЏОЕОДОЋОеОљОЎОЮ Ою-m->introduce(), ОћОъОћОЊОе ОЎОЋОЊОб ОЕ-introduce ОћОЎОљ ОћОъОфОЋОЊОћ ОћОЋОЋОЎОеОўОЋОљОюОЎОф ОћОЕОаОЎОЎОћ, ОЏОюОЋОъОе slot 1, ОЕОаОъОдОљ ОЉ-offset 8 ОЉОфОЋОџ ОћОўОЉОюОћ. ОћОљОАОъОЉОюОЎ ОЕОаОЋОдОе ОаОеОљОћ ОЏОџ:

    mov    rax, QWORD PTR [rdi]      ; rax = vptr (the object's first byte)
mov    rax, QWORD PTR [rax+0x8]  ; rax = vtable[1] = introduce
; rdi already holds this
call   rax                       ; jump to the function

    ОбОдОеОЋ ОеОњОб ОЋОћОАОфОЏОюОЋ ОбОю ОЕОфОЎ ОћОЕОЋОеОЋОф ОћОеОљОЕОЋОаОЋОф. ОћОДОеОЎОљОћ ОћОЋОЋОЎОеОўОЋОљОюОЎОф ОбОЋОЕОћ ОЕОфОЎ ОћОцОаОЎОЋОф ОбОДОЎОцОЋОф: ОДОЋОЊОЮ ОДОЋОеОљОф ОљОф Оћ-vptr ОъОћОљОЋОЉОЎОЎОДОў, ОЋОљОќ ОДОЋОеОљОф ОљОф ОъОдОЉОЎОб ОћОцОЋОаОДОдОЎОћ Оъ-vptr+8. ОљОЮ ОљОаОЌОаОЋ ОЕОЋОюОўОЎОЮ ОбОю Оћ-vptr - ОЕОћОЋОљ ОћОЉОЎОф ОћОеОљОЕОЋОЪ ОЕОю ОћОљОЋОЉОЎОЎОДОў - ОљОаОЌОаОЋ ОЕОЋОюОўОЎОЮ ОюОЌОюОЋОўОЎОЪ ОбОю ОъОћ ОЕ-call rax ОЎОДОцОЋОЦ ОљОюОЎОЋ. ОЋОќОћ ОЉОЊОЎОЋОД ОъОћ ОЕ-UAF ОаОЋОфОЪ ОюОаОЋ: ОљОф ОћОЎОЏОЋОюОф ОюОЏОфОЋОЉ ОюОЉОЎОф ОћОеОљОЕОЋОЪ ОЕОю ОљОЋОЉОЎОЎОДОў ОъОЕОЋОЌОеОе.

    ОЌОўОЎОцОф Оћ-vptr ОЊОеОџ UAF - ОћОеОбОЎОЋОЪ ОћОъОеОЏОќОЎ¶

    ОбОЏОЕОЎОЋ ОаОЌОЉОе ОљОф ОћОЏОю. ОћОќОеОЎОъОћ ОЕОю ОћОћОфОДОцОћ:

    1. ОћОфОЋОЏОаОЎОф ОъОДОдОћ ОљОЋОЉОЎОЎОДОў Man - ОЉОюОЋОД ОЉОњОЋОЊОю 0x20 (24 ОЉОфОЎОЮ ОаОфОЋОаОЎОЮ), ОбОЮ vptr ОфОДОЎОЪ ОЉ-offset 0.
    2. ОћОфОЋОЏОаОЎОф ОъОЕОЌОеОеОф ОљОЋОфОЋ (delete m). ОбОЏОЕОЎОЋ ОЎОЕ ОъОдОЉОЎОб ОфОюОЋОЎ, ОЋОћОЉОюОЋОД ОЉ-tcache/fastbin.
    3. ОљОаОЌОаОЋ ОъОЉОДОЕОЎОЮ ОъОћОфОЋОЏОаОЎОф ОюОћОДОдОЋОф ОЉОюОЋОД ОъОљОЋОфОЋ ОњОЋОЊОю ОЋОюОъОюОљ ОљОЋОфОЋ ОЉОаОфОЋОаОЎОЮ ОЕОюОаОЋ. ОћОъОДОдОћ ОъОЌОќОЎОе ОљОф ОћОЉОюОЋОД ОћОъОЕОЋОЌОеОе, ОЋОљОаОЌОаОЋ ОЏОЋОфОЉОЎОЮ ОљОф 8 ОћОЉОфОЎОЮ ОћОеОљОЕОЋОаОЎОЮ - Оћ-vptr - ОюОбОеОџ ОЕОљОаОЌОаОЋ ОЉОЋОЌОеОЎОЮ.
    4. ОћОфОЋОЏОаОЎОф ОДОЋОеОљОф m->introduce() ОЊОеОџ ОћОъОдОЉОЎОб ОћОфОюОЋОЎ. ОћОДОеОЎОљОћ ОћОЋОЋОЎОеОўОЋОљОюОЎОф ОДОЋОеОљОф [vptr+8] ОЋОъОДОцОЎОдОћ ОюОЕОЮ.

    ОћОЕОљОюОћ ОћОЎОЌОЎОЊОћ ОЕОаОЕОљОеОћ: ОюОљОЎОќОћ ОбОеОџ ОюОЏОфОЋОЉ ОљОф Оћ-vptr ОћОъОќОЋОЎОБ ОЏОЊОЎ ОЕ-[vptr+8] ОЎОдОЉОЎОб Ою-give_shell?

    ОћОаОћ ОћОўОеОЎОД ОћОЎОцОћ. ОљОаОЌОаОЋ ОюОљ ОдОеОЎОЏОЎОЮ ОюОЉОаОЋОф vtable ОъОќОЋОЎОБ ОъОЕОюОаОЋ - ОљОаОЌОаОЋ ОЎОЏОЋОюОЎОЮ ОюОъОЌОќОе ОљОф Оћ-vtable ОћОљОъОЎОфОЎ ОЕОЏОЉОе ОДОЎОЎОЮ ОЉОЉОЎОаОљОеОЎ. ОљОаОЌОаОЋ ОЎОЋОЊОбОЎОЮ ОЕ-give_shell ОЎОЋОЕОЉ ОЉ-slot 0 ОЕОю ОћОўОЉОюОћ, ОЏОюОЋОъОе ОЉ-offset 0. ОћОДОеОЎОљОћ ОЕОю introduce ОДОЋОеОљОф Оъ-vptr+8. ОљОќ ОљОЮ ОаОЏОфОЋОЉ Ою-vptr ОљОф ОћОбОеОџ (ОЏОфОЋОЉОф slot 0) ОцОЌОЋОф 8, ОљОќ:

    fake vptr = slot0_address - 8
introduce reads [vptr+8] = [slot0_address - 8 + 8] = [slot0_address] = give_shell

    ОЏОюОЋОъОе, ОћОќОќОаОЋ ОљОф Оћ"ОўОЉОюОћ" ОЕОъОЋОаОћ ОЉОфОЎОЮ ОљОЌОЋОеОћ, ОЏОџ ОЕОъОћ ОЕ-introduce ОЌОЋОЕОЉ ОЕОћОЋОљ slot 1 ОћОЋОљ ОЉОбОдОЮ slot 0 = give_shell. ОљОюОњОаОўОЎ, ОЋОюОљ ОЊОЋОеОЕ ОЕОЋОЮ ОЏОфОЋОЉОф heap - ОеОД ОљОф ОћОЏОфОЋОЉОф ОћОДОЉОЋОбОћ ОЕОю Оћ-vtable ОЉОЉОЎОаОљОеОЎ.

    ОќОћ ОбОЋОЉОЊ ОеОД ОЏОЎ ОћОЉОЎОаОљОеОЎ ОћОЋОљ No PIE. ОЉОюОЎ PIE, Оћ-vtable ОЋ-give_shell ОЎОЋОЕОЉОЎОЮ ОЉОЏОфОЋОЉОЋОф ОДОЉОЋОбОЋОф ОЉОЏОю ОћОеОдОћ, ОљОќ ОљОцОЕОе ОюОЏОфОЋОЉ ОљОЋОфОЪ ОДОЕОЎОЌ. ОљОЮ ОћОЎОћ PIE, ОћОЎОЎОаОЋ ОдОеОЎОЏОЎОЮ ОДОЋОЊОЮ ОюОЊОюОЋОБ ОЏОфОЋОЉОф ОДОЋОЊ (ОюОъОЕОю ОЊОеОџ ОцОЪ Оћ"ОДОеОЎОљОћ ОюОљОЌОе ОЕОЌОеОЋОе" ОЕОю ОљОЋОфОћ ОЌОЋОюОЕОћ) ОЋОљОќ ОюОЌОЕОЉ ОљОф ОћОЉОАОЎОА.

    virtual void introduce()  { puts("I am a human"); } // slot 1

    char *data;
    int op;

        printf("1.use 2.alloc 3.free > ");
        if (scanf("%d", &op) != 1) break;

        } else if (op == 2) {

        }
    }
}

    ОћОъОДОеОћ ОћОДОюОљОАОЎ - pwnable.kr uaf¶

    ОбОЏОЕОЎОЋ ОюОљОфОњОе ОбОдОъОЋ. uaf ОЉОДОўОњОЋОеОЎОЎОф Rookiss ОћОЋОљ ОфОЋОЏОаОЎОф C++ ОбОЮ ОъОЌОюОДОЋОф Human, Man ОЋ-Woman. ОћОцОЋОаОДОдОЎОћ main ОъОДОдОћ ОЕОаОЎ ОљОЋОЉОЎОЎОДОўОЎОЮ ОЋОљОќ ОаОЏОаОАОф ОюОюОЋОюОљОф ОфОцОеОЎОў:

    Human* m = new Man("Jack", 25);
Human* w = new Woman("Jill", 21);
while (1) {
    // 1. use   2. after   3. free
    switch (op) {
      case 1:  m->introduce();  w->introduce();  break;
      case 2:  len  = atoi(argv[1]);
               data = new char[len];
               read(open(argv[2], O_RDONLY), data, len);   break;
      case 3:  delete m;  delete w;  break;
    }
}

    ОЕОЎОъОЋ ОюОЉ ОюОЏОъОћ ОцОеОўОЎОЮ ОЕОъОбОдОЉОЎОЮ ОљОф ОЏОю ОћОћОфОДОцОћ:

    • ОћОњОЋОЊОю ОЉОљ Оъ-argv[1], ОЋОћОфОЋОЏОЪ Оъ-argv[2]. ОљОЋОцОдОЎОћ 2 ("after") ОъОДОдОћ new char[len] ОЋОДОЋОеОљОф len ОЉОфОЎОЮ ОъОДОЋОЉОЦ ОЕОљОаОЌОаОЋ ОЉОЋОЌОеОЎОЮ. ОЏОюОЋОъОе, ОљОаОЌОаОЋ ОЕОЋОюОўОЎОЮ ОњОЮ ОЉОњОЋОЊОю ОћОћОДОдОљОћ ОЋОњОЮ ОЉОЏОю ОЉОЎОф ОЕОюОћ. ОќОћ ОћОЏОюОЎ ОћОъОЋОЕОюОЮ ОюОъОЌОќОе ОЉОюОЋОД ОъОЕОЋОЌОеОе ОЋОюОъОюОљ ОљОЋОфОЋ.
    • ОњОЋОЊОю ОћОљОЋОЉОЎОЎОДОў ОћОЋОљ 0x18 = 24 ОЉОфОЎОЮ. Оћ-string ОЉОъОЌОюОДОћ ОћОЋОЊОе ОбОЮ Оћ-ABI ОћОЎОЕОЪ (COW string) ОЕОћОЋОљ ОъОдОЉОЎОб ОЎОЌОЎОЊ, ОљОќ: vptr (8) + int age ОбОЮ padding (8) + string (8) = 24. ОюОЏОЪ new char[24] ОаОЋОцОю ОюОљОЋОфОћ ОъОЌОюОДОф ОњОЋОЊОю ОЉОЊОЎОЋОД ОЏОъОЋ ОћОљОЋОЉОЎОЎОДОў (chunk ОЕОю 0x20), ОЋОъОъОЌОќОе ОљОЋОфОЋ.
    • ОљОЋОцОдОЎОћ 3 ОъОЕОЌОеОеОф ОљОф ОЕОаОЎОћОЮ: delete m ОЋОљОќ delete w. ОЕОаОЎ ОЉОюОЋОДОЎОЮ ОЕОю 0x20 ОаОЏОаОАОЎОЮ Ою-bin, ОЏОЕОћОеОљОЕ ОћОЋОљ w. ОЕОфОЎ ОћОДОдОљОЋОф ОЕОю 24 ОЉОфОЎОЮ ОЎОЌОќОЎОеОЋ ОДОЋОЊОЮ ОљОф w ОЋОљОќ ОљОф m.
    • ОљОЋОцОдОЎОћ 1 ОДОЋОеОљОф m->introduce() ОфОЌОЎОюОћ. ОљОЮ ОаОЊОеОЋОА ОљОф Оћ-vptr ОЕОю m, ОаОДОЉОю shell ОЏОЉОе ОЉОДОеОЎОљОћ ОћОеОљОЕОЋОаОћ.

    ОюОъОћ ОдОеОЎОџ ОюОћОДОдОЋОф ОцОбОъОЎОЎОЮ ОЋОюОљ ОцОбОЮ ОљОЌОф? ОЏОЎ ОЏОъОЋ ОЕОеОљОЎОаОЋ, ОбОдОЮ Оћ-free ОЊОЋОеОА ОљОф Оћ-vptr ОЕОю ОЏОю ОљОЋОЉОЎОЎОДОў ОбОЮ ОъОдОЉОЎОб Оћ-fd ОЕОю Оћ-bin. ОљОЌОеОЎ delete m; delete w;, ОњОЮ Оћ-vptr ОЕОю m ОЋОњОЮ ОЕОю w ОъОЋОЕОЌОфОЎОЮ. ОљОЮ ОаОДОеОљ Ою-use ОъОЎОЊ, m->introduce() ОЎОДОеОЋОА ОбОю vptr ОќОЉОю. ОюОЏОЪ ОаОДОдОћ ОцОбОъОЎОЎОЮ: ОћОДОдОљОћ ОеОљОЕОЋОаОћ ОъОъОЌОќОеОф ОљОф w ОЋОЏОЋОфОЉОф ОюОЋ vptr ОўОЋОЉ, ОћОДОдОљОћ ОЕОаОЎОЎОћ ОъОъОЌОќОеОф ОљОф m ОЋОЏОЋОфОЉОф ОюОЋ vptr ОўОЋОЉ. ОеОД ОљОќ use ОЉОўОЋОЌ.

    ОЋОљОЎОќОћ ОбОеОџ ОЏОЋОфОЉОЎОЮ? ОЉОЊОЎОЋОД ОћОўОеОЎОД ОъОДОЋОЊОЮ. ОъОЋОдОљОЎОЮ ОЉ-gdb ОљОф slot0 ОЕОю ОћОўОЉОюОћ (ОЕОЮ ОЎОЋОЕОЉ give_shell), ОЋОЏОЋОфОЉОЎОЮ ОљОЋОфОЋ ОцОЌОЋОф 8. ОЉОЉОЎОаОљОеОЎ ОћОќОћ give_shell ОЎОЋОЕОЉОф ОЉ-0x400ec4, Оћ-vptr ОћОљОъОЎОфОЎ ОЕОю ОћОљОЋОЉОЎОЎОДОў ОћОЋОљ 0x401570, ОЋОюОЏОЪ ОћОбОеОџ ОЕОаОЏОфОЋОЉ ОћОЋОљ 0x401568. ОљОф ОЏОю ОћОдОбОЊОЎОЮ ОћОъОЊОЋОЎОДОЎОЮ - ОљОЎОџ ОъОЋОдОљОЎОЮ ОљОф ОќОћ ОЉ-gdb, ОљОЎОџ ОЉОЋОаОЎОЮ ОљОф ОћОДОЋОЉОЦ, ОЋОЉОљОЎОќОћ ОАОЊОе ОюОЋОЌОдОЎОЮ ОбОю ОћОфОцОеОЎОў - ОаОбОЕОћ ОЎОЌОЊ ОЉОцОфОеОЋОЪ.

    ОћОњОаОЋОф ОЋОћОЕОюОЏОЋОф - Mitigations¶

    ОЌОЕОЋОЉ ОюОћОЉОЎОЪ ОъОћ ОбОЋОдОе UAF ОЋОъОћ ОюОљ:

    • ASLR ОЋ-PIE - ОљОюОћ ОћОћОњОаОЋОф ОћОеОюОЋОЋОаОўОЎОЋОф ОЉОЎОЋОфОе ОЏОљОЪ. UAF ОЕОЌОЋОўОБ vptr ОЊОЋОеОЕ ОЏОфОЋОЉОф ОДОЉОЋОбОћ ОЕОю give_shell ОљОЋ ОЕОю vtable. ОЉОЉОЎОаОљОеОЎ No PIE ОЏОъОЋ uaf ОћОЏОфОЋОЉОЋОф ОДОЉОЋОбОЋОф ОЋОћОфОДОЎОцОћ ОЎОЕОЎОеОћ. ОбОЮ PIE, ОдОеОЎОџ ОДОЋОЊОЮ ОюОЊОюОЋОБ ОЏОфОЋОЉОф ОДОЋОЊ - ОюОъОЕОю ОЊОеОџ ОцОЪ Оћ"ОДОеОЎОљОћ" ОЕОю ОљОЋОфОћ ОЌОЋОюОЕОћ, ОљОЋ ОЊОеОџ ОЌОЋОюОЕОћ ОаОюОЋОЋОЎОф.
    • ОъОцОфОЌ Оћ-tcache ОаОњОЊ double free (Оъ-glibc 2.29) - ОюОљ ОбОЋОќОе ОаОњОЊ UAF-ОъОЎОЌОќОЋОе. ОћОЉОЊОЎОДОћ ОћОќОЋ ОЕОДОЋОцОћ ОюОњОъОеОЎ ОюОћОфОДОцОћ ОЕОюОаОЋ.

    • NX - ОцОбОЎОю, ОЋОюОљ ОъОцОеОЎОб: ОљОаОЌОаОЋ ОюОљ ОъОеОЎОдОЎОЮ shellcode, ОљОюОљ ОДОЋОцОдОЎОЮ ОюОДОЋОЊ ОДОЎОЎОЮ (give_shell).
