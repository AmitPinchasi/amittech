    8.3 - ОЉОЎОаОљОеОЎ ОќОбОЎОе - ОфОеОњОЋОю

    ОфОеОњОЋОю - ОЉОЎОаОљОеОЎ ОќОбОЎОе (tiny_easy)¶

    ОЉОфОеОњОЋОю ОћОќОћ ОфОцОдОЌОЋ ОљОф tiny_easy Оъ-pwnable.kr - ELF ОќОбОеОЋОеОЎ ОЋОъОЋОцОЕОў ОъОАОъОюОЎОЮ ОЕОЏОю ОъОћ ОЕОћОЋОљ ОбОЋОЕОћ ОќОћ ОюОДОеОЋОА. ОћОЊОеОџ ОюОцОдОЌ ОљОЋОфОЋ ОбОЋОЉОеОф ОЊОеОџ ОћОЉОаОћ ОъОЊОЋОЎОДОф ОЕОю ОъОдОЉ ОћОфОћОюОЎОџ ОЉОеОњОб ОћОЏОаОЎОАОћ: ОъОћ ОЎОЕ ОбОю ОћОъОЌОАОаОЎОф, ОъОћ ОЉ-eax, ОЋОюОљОЪ ОЉОЊОЎОЋОД ОћОДОЋОЊ ОДОЋОцОЦ. ОЏОЊОЎ ОюОфОеОњОю ОЉОаОЋОЌОЋОф ОаОЉОаОћ ОДОЋОЊОЮ ОЉОЎОаОљОеОЎ ОъОДОЋОъОЎ ОЕОъОЌОДОћ ОљОф tiny_easy ОЉОЋОю, ОаОцОдОЌ ОљОЋОфОЋ, ОЋОеОД ОљОќ ОаОЏОЋОЋОЪ ОюОЕОеОф ОћОљОъОЎОфОЎ. ОбОЉОЊОЋ ОюОцОЎ ОћОАОЊОе - ОЏОю ОфОеОњОЎОю ОЉОЋОаОћ ОбОю ОћОДОЋОЊОЮ.

    ОћОљОфОњОе ОаОъОдОљ ОЉОДОўОњОЋОеОЎОћ Toddler's Bottle. ОъОфОЌОЉОеОЎОЮ ОљОюОЎОЋ ОЏОџ:

    ssh tiny_easy@pwnable.kr -p2222
# the password appears on the challenge page on the site

    ОЉОфОЎОДОЎОЎОф ОћОЉОЎОф ОфОъОдОљОЋ ОЉОЎОаОљОеОЎ ОЉОЕОЮ tiny_easy (ОбОЮ ОЉОЎОў setuid) ОЋОДОЋОЉОЦ flag ОЕОеОД ОћОъОЕОфОъОЕ ОћОъОЎОЋОЌОА ОЎОЏОЋОю ОюОДОеОЋОљ. ОЕОЎОъОЋ ОюОЉ: ОљОЎОЪ ОЏОљОЪ ОДОЋОЉОЦ ОъОДОЋОе .c. ОћОЉОЎОаОљОеОЎ ОЏОю ОЏОџ ОДОўОЪ ОЕОљОфОЮ ОљОъОЋОеОЎОЮ ОюОцОеОД ОљОЋОфОЋ ОЋОюОћОЉОЎОЪ ОљОЋОфОЋ ОЉОбОдОъОЏОЮ. ОбОЉОЊОЋ ОЉОфОЎОДОЎОЎОћ ОќОъОаОЎОф ОфОЌОф /tmp ОЏОъОЋ ОЕОюОъОЊОаОЋ ОЉ-0.5.

    ОћОЏОаОћ - ОЉОЎОаОљОеОЎ ОъОДОЋОъОЎ ОЕОъОЌОДОћ ОљОф tiny_easy¶

    ОюОцОаОЎ ОЕОаОЋОњОбОЎОЮ ОЉОЕОеОф, ОаОЉОаОћ ОбОЋОфОД ОъОДОЋОъОЎ ОюОћОфОљОъОЪ ОбОюОЎОЋ. ОдОеОЋ ОДОЋОЉОЦ ОљОАОъОЉОюОЎ tiny.asm ОЕОъОЌОДОћ ОљОф ОаОДОЋОЊОф ОћОЏОаОЎОАОћ ОЕОю tiny_easy:

    ; tiny.asm - mimics the entry point of tiny_easy
BITS 32
global _start
_start:
    pop eax        ; eax = argc
    pop eax        ; eax = argv[0]  (pointer to the program name string)
    mov eax, [eax] ; eax = *(argv[0]) = the first four bytes of the program name
    jmp eax        ; jump to the address held in those four bytes

    ОДОъОцОюОЋ ОЋОДОЕОеОЋ ОљОЋОфОЋ ОЏОЉОЎОаОљОеОЎ ОюОљ-PIE ОбОЮ ОъОЌОАОаОЎОф ОаОЎОфОаОф ОюОћОеОдОћ (ОљОЮ ОЌОАОеОЋОф ОАОцОеОЎОЋОф 32 ОЉОЎОў: sudo apt install gcc-multilib nasm):

    nasm -f elf32 tiny.asm -o tiny.o
ld -m elf_i386 -z execstack -o tiny tiny.o
strip tiny

    ОЏОЉОЋ ASLR ОюОдОЋОеОџ ОцОЎОфОЋОЌ ОеОљОЕОЋОаОЎ:

    echo 0 | sudo tee /proc/sys/kernel/randomize_va_space

    ОфОеОњОЎОю 1 - ОАОДОЎОеОћ ОЕОю ОћОЉОЎОаОљОеОЎ ОћОъОЎОаОЎОъОюОЎ¶

    ОюОцОаОЎ ОЕОфОЋОДОцОЎОЮ, ОъОЉОЎОаОЎОЮ ОъОЋОю ОъОћ ОбОЋОъОЊОЎОЮ.

    1. ОћОеОЎОдОЋ file tiny, wc -c tiny, ОЋ-checksec --file=tiny. ОЕОЎОъОЋ ОюОЉ ОюОњОЋОЊОю ОћОќОбОеОЋОеОЎ, Ою-NX disabled, ОЋОю-No PIE.
    2. ОћОеОЎОдОЋ readelf -h tiny | grep Entry ОЋОеОЕОъОЋ ОљОф ОаОДОЋОЊОф ОћОЏОаОЎОАОћ. ОћОеОЎОдОЋ readelf -l tiny ОЋОеОљОЋ ОЏОъОћ ОъОДОўОбОЎ LOAD ОЎОЕ ОЋОЉОљОЎОюОЋ ОћОеОЕОљОЋОф.
    3. ОЌОЕОЉОЋ: ОъОЊОЋОб Оћ-NX ОъОЏОЋОЉОћ ОюОъОеОЋОф ОЕОюОљ ОЉОЎОДОЕОфОЮ ОќОљОф ОЉОъОцОЋОеОЕ? ОъОћ ОЉОДОЋОЉОЦ (ОљОЋ ОъОћ ОЌОАОе ОЉОЋ) ОњОЋОеОЮ ОюОЏОџ?

    ОеОъОќ: ОЌОцОЕОЋ ОљОЮ ОДОЎОЎОЮ ОъОДОўОб GNU_STACK ОЉОцОюОў ОЕОю readelf -l. ОљОЮ ОћОЋОљ ОЌОАОе, Оћkernel ОъОцОбОЎОю READ_IMPLIES_EXEC ОЋОЏОю ОЊОБ ОДОеОЎОљ ОћОЋОцОџ ОњОЮ ОюОаОЎОфОЪ ОюОћОеОдОћ.

    ОеОъОќ: ОаОДОЋОЊОф ОћОЏОаОЎОАОћ ОаОъОдОљОф ОЉОЊОЎОЋОД ОљОЌОеОЎ ОЏОЋОфОеОф Оћ-ELF (52 ОЉОфОЎОЮ) ОЋОЏОЋОфОеОф ОћОфОЋОЏОаОЎОф (32 ОЉОфОЎОЮ). ОаОАОЋ ОюОЌОЕОЉ ОљОф ОћОЏОфОЋОЉОф ОЋОюОеОљОЋОф ОЕОћОЎОљ ОъОфОљОЎОъОћ ОюОцОюОў ОЕОю readelf.

    ОфОеОњОЎОю 2 - ОцОЎОеОЋОД ОћОДОЋОЊ ОЋОъОбОДОЉ ОљОЌОеОЎ eax¶

    ОбОЏОЕОЎОЋ ОаОЉОЎОЪ ОъОћ ОћОЉОЎОаОљОеОЎ ОбОЋОЕОћ ОЉОцОЋОбОю.

    1. ОћОеОЎОдОЋ objdump -d tiny ОЋОЏОфОЉОЋ ОюОбОдОъОЏОЮ ОљОф ОћОћОЋОеОљОЋОф ОЉОаОДОЋОЊОф ОћОЏОаОЎОАОћ.
    2. ОбОДОЉОЋ ОљОЌОеОЎ eax ОћОЋОеОљОћ-ОћОЋОеОљОћ, ОЏОЕОъОцОф ОћОъОЌОАОаОЎОф ОЉОЏОаОЎОАОћ (argc, argv[], envp[]) ОюОцОаОЎОЏОЮ. ОъОћ eax ОъОЌОќОЎОД ОљОЌОеОЎ ОЏОю ОљОЌОЊ ОъОЕОаОЎ Оћ-pop? ОъОћ ОљОЌОеОЎ Оћ-mov?
    3. ОбОаОЋ ОЉОъОЊОЋОЎОД: ОюОљОЎОќОЋ ОЏОфОЋОЉОф ОДОЋОцОдОф ОћОћОЋОеОљОћ ОћОљОЌОеОЋОаОћ, ОЋОъОљОЎОцОћ ОъОњОЎОб ОћОбОеОџ ОЕОю ОћОЏОфОЋОЉОф ОћОќОЋ?
    4. ОћОеОЎОдОЋ ./tiny ОеОњОЎОю ОЋОеОљОЋ ОљОф ОћОДОеОЎОАОћ. ОћОЕОфОъОЕОЋ ОЉ-dmesg | tail ОљОЋ ОЉ-gdb ОЏОЊОЎ ОюОеОљОЋОф ОЉОљОЎОќОЋ ОЏОфОЋОЉОф Оћ-eip ОДОеОА. ОћОљОЮ ОћОЏОфОЋОЉОф ОъОфОљОЎОъОћ ОюОЉОфОЎОЮ ОћОеОљОЕОЋОаОЎОЮ ОЕОю ОћОъОЌОеОЋОќОф ./tiny?

    ОеОъОќ: ОљОЌОеОЎ ОЕОаОЎ Оћ-pop, eax ОъОЌОќОЎОД ОљОф ОћОъОдОЉОЎОб argv[0]. ОћОћОЋОеОљОћ mov eax,[eax] ОДОЋОеОљОф ОљОф ОљОеОЉОбОф ОћОЉОфОЎОЮ ОћОеОљОЕОЋОаОЎОЮ ОЕОю ОћОъОЌОеОЋОќОф ОЕ-argv[0] ОъОдОЉОЎОб ОљОюОЎОћ - ОЏОюОЋОъОе ОЕОю ОЕОЮ ОћОфОЋОЏОаОЎОф.

    ОеОъОќ: ./ti ОЉОДОЎОЊОЋОЊ little-endian ОћОЋОљ 0x69742e2e. ОљОЮ ОћОДОеОЎОАОћ ОЉ-eip ОћОЎОљ ОЉОЏОфОЋОЉОф ОћОќОЋ, ОћОЉОаОфОЮ ОаОЏОЋОЪ: ОЎОбОЊ ОћОДОцОЎОдОћ ОаОЕОюОў ОюОњОъОеОЎ ОъОљОеОЉОбОф ОћОЉОфОЎОЮ ОћОеОљОЕОЋОаОЎОЮ ОЕОю ОЕОЮ ОћОфОЋОЏОаОЎОф.

    ОфОеОњОЎОю 3 - ОъОдОЉ ОћОфОћОюОЎОџ ОЉОЏОаОЎОАОћ ОЉ-gdb¶

    ОаОљОЕОе ОљОф ОъОцОф ОћОъОЌОАОаОЎОф ОъОћОћОеОдОљОћ ОЉОбОЎОаОЎОЎОЮ ОЕОюОаОЋ.

    1. ОцОфОЌОЋ ОљОф ОћОЉОЎОаОљОеОЎ ОЉ-gdb ОЋОбОдОеОЋ ОЉОЏОаОЎОАОћ ОбОЮ starti (ОбОЋОдОе ОбОю ОћОћОЋОеОљОћ ОћОеОљОЕОЋОаОћ ОЕОю _start).
    2. ОћОАОфОЏОюОЋ ОбОю ОеОљОЕ ОћОъОЌОАОаОЎОф: x/8x $esp. ОћОбОеОџ ОћОеОљОЕОЋОЪ ОћОЋОљ argc. ОљОЌОеОЎОЋ ОъОдОЉОЎОбОЎ argv, ОљОЌОеОЎОЋ NULL, ОљОЌОеОЎОЋ ОъОдОЉОЎОбОЎ envp.
    3. ОбОДОЉОЋ ОљОЌОеОЎ ОъОдОЉОЎОб argv[0] (x/s ОбОю ОћОбОеОџ ОЕОЉ-$esp+4) ОЋОеОљОЋ ОЕОћОЋОљ ОъОдОЉОЎОб ОбОю ОъОЌОеОЋОќОф ОЕОЮ ОћОфОЋОЏОаОЎОф.
    4. ОћОЏОаОЎОАОЋ ОъОЕОфОаОћ ОАОЉОЎОЉОћ ОъОќОћОћ (ОюОъОЕОю AAAA=BBBBCCCC), ОћОеОЎОдОЋ ОЕОЋОЉ, ОЋОЌОцОЕОЋ ОљОЋОфОЋ ОбОю ОћОъОЌОАОаОЎОф: search -s "BBBBCCCC". ОЕОЎОъОЋ ОюОЉ ОЕОћОЋОљ ОЎОЋОЕОЉ ОњОЉОЋОћ, ОЉОеОљОЕ ОћОъОЌОАОаОЎОф.

    ОеОъОќ: ОћОцОДОЋОЊОћ starti ОЉ-pwndbg ОбОЋОдОеОф ОбОЋОЊ ОюОцОаОЎ ОћОДОцОЎОдОћ, ОЏОЕОћОъОЌОАОаОЎОф ОбОЊОЎОЎОЪ ОЉОъОдОЉ ОћОЏОаОЎОАОћ ОЕОћkernel ОЉОаОћ. ОќОћ ОћОеОњОб ОћОъОЋОЕОюОЮ ОюОеОљОЋОф ОљОф argc/argv/envp.

    ОфОеОњОЎОю 4 - ОћОЕОфОюОф shellcode ОЉОАОЉОЎОЉОћ ОЋОЏОЎОЋОЋОЪ ОћОДОцОЎОдОћ (ОъОДОЋОъОЎ)¶

    ОбОЏОЕОЎОЋ ОаОцОеОЋОЦ ОљОф ОћОЉОЎОаОљОеОЎ ОћОъОДОЋОъОЎ.

    1. ОЉОаОЋ shellcode ОЕОю 32 ОЉОЎОў ОЕОъОеОЎОЦ execve("/bin/sh", 0, 0) (ОћОЎОЊ-ОЏОфОЋОЉ Оъ-3.1, ОљОЋ asm(shellcraft.i386.linux.sh())).
    2. ОћОЏОаОЎОАОЋ ОюОъОЕОфОаОћ ОАОЉОЎОЉОћ ОъОќОЌОюОф ОбОаОДОЎОф ОЕОю \x90 (ОюОъОЕОю 0x10000 ОЉОфОЎОЮ) ОЋОљОЌОеОЎОћ Оћ-shellcode.
    3. ОъОдОљОЋ ОЉ-gdb ОЏОфОЋОЉОф ОбОъОЋОД ОЉОфОЋОџ ОћОъОќОЌОюОф (search -s "\x90\x90\x90\x90\x90\x90\x90\x90" ОбОю Оћ-[stack]).
    4. ОћОцОбОЎОюОЋ ОљОф ОћОЉОЎОаОљОеОЎ ОЊОеОџ execve ОбОЮ argv[0] = p32(addr), ОЏОЕОћ-addr ОћОЋОљ ОЏОфОЋОЉОф ОЉОфОЋОџ ОћОъОќОЌОюОф. ОЋОЊОљОЋ ОЕОаОцОфОЌ shell. ОаОАОЋ id.

    ОеОъОќ: ОћОЕОюОЎОўОћ ОЕОюОЏОЮ ОћОЎОљ ОЉ-argv[0]. ОЏОЊОЎ ОюОЕОЎОЮ ОЕОЮ ОЉОфОЎОЮ ОњОЋОюОъОЎОЎОЮ ОЌОЎОЎОЉОЎОЮ execve - ОЉ-pwntools ОќОћ process(['./tiny'], executable='./tiny', argv=[p32(addr)], env={...}).

    ОеОъОќ: ОъОЏОЎОЋОЋОЪ ОЕОљОфОЮ ОДОЋОеОљОЎОЮ ОеОД ОљОф ОљОеОЉОбОф ОћОЉОфОЎОЮ ОћОеОљОЕОЋОаОЎОЮ ОЕОю argv[0], ОъОАОцОЎОД ОЕ-p32(addr) ОЎОћОЎОћ ОЉОюОЎ ОЉОЎОф ОљОцОА. ОЏОфОЋОЉОф ОъОЌОАОаОЎОф ОўОЎОцОЋОАОЎОф ОЏОъОЋ 0xffffd800 ОаОДОЎОЎОћ.

    ОеОъОќ: ОљОЮ ОќОћ ОъОцОАОцОА, ОЋОЊОљОЋ ОЕОюОљ ОЕОЎОаОЎОфОЮ ОљОф ОњОЋОЊОю ОћОАОЉОЎОЉОћ ОЉОЎОЪ ОћОЌОЎОцОЋОЕ ОЉ-gdb ОюОЉОЎОЪ ОћОћОеОдОћ - ОЏОю ОЕОЎОаОЋОЎ ОЏОќОћ ОъОќОЎОќ ОљОф ОЏОю ОћОъОЌОАОаОЎОф.

    ОфОеОњОЎОю 5 - ОћОфОъОЋОЊОЊОЋОф ОбОЮ ASLR ОЋОюОЋОюОљОф brute force¶

    ОбОЊ ОбОЏОЕОЎОЋ ОбОЉОЊОаОЋ ОбОЮ ASLR ОЏОЉОЋОЎ. ОбОю ОћОЕОеОф ОћОљОъОЎОфОЎ ОћОЋОљ ОЏОаОеОљОћ ОцОбОЎОю.

    1. ОћОЊОюОЎОДОЋ ASLR ОъОЌОЊОЕ: echo 2 | sudo tee /proc/sys/kernel/randomize_va_space, ОЋОћОеОЎОдОЋ ОљОф Оћ-exploit ОъОфОеОњОЎОю 4. ОћОЋОљ ОЏОаОеОљОћ ОЎОцОАОцОА, ОЏОЎ ОЏОфОЋОЉОф ОћОъОЌОАОаОЎОф ОќОќОћ ОЉОЏОю ОћОеОдОћ.
    2. ОбОўОцОЋ ОљОф ОћОћОцОбОюОћ ОЉОюОЋОюОљОћ: ОДОЉОбОЋ ОЏОфОЋОЉОф ОаОЎОЌОЋОЕ ОљОЌОф ОЉОфОЋОџ ОфОЌОЋОЮ ОћОъОќОЌОюОф, ОЋОћОеОЎОдОЋ ОЕОЋОЉ ОЋОЕОЋОЉ ОбОЊ ОЕОаОцОфОЌ shell. ОљОЌОеОЎ ОЏОю ОЏОЎОЕОюОЋОЪ ОАОњОеОЋ ОљОф ОћОфОћОюОЎОџ ОЋОаОАОЋ ОЕОЋОЉ.
    3. ОъОЊОЊОЋ ОЏОъОћ ОаОЎОАОЎОЋОаОЋОф ОюОДОЌ. ОаОАОЋ ОюОћОњОЊОЎОю ОљОф ОћОъОќОЌОюОф ОЋОюОеОљОЋОф ОЕОъОАОцОе ОћОаОЎОАОЎОЋОаОЋОф ОЎОЋОеОЊ.

    ОеОъОќ: ОЏОю ОћОеОдОћ ОъОњОеОЎОюОћ ОъОЌОАОаОЎОф ОЌОЊОЕОћ. ОЉОАОЋОцОЋ ОЕОю ОЊОЉОе ОћОъОќОЌОюОф ОћОњОЊОЋОюОћ ОЕОюОЏОЮ "ОаОЋОЌОфОф" ОъОбОю ОћОЏОфОЋОЉОф ОЕОаОЎОЌОЕОфОЮ, ОЋОћ-shellcode ОеОЦ. ОЉ-32 ОЉОЎОў ОћОљОаОўОеОЋОцОЎОћ ОаОъОЋОЏОћ, ОљОќ ОЏОъОћ ОъОљОЋОф ОбОЊ ОљОюОцОЎ ОаОЎОАОЎОЋОаОЋОф ОъОАОцОЎОДОЎОЮ.

    ОеОъОќ: ОЏОЏОю ОЕОћОъОќОЌОюОф ОњОЊОЋОюОћ ОЎОЋОфОе, ОЏОџ ОћОЎОљ ОъОЏОАОћ ОЎОЋОфОе ОЏОфОЋОЉОЋОф ОљОцОЕОеОЎОЋОф, ОЋОцОЌОЋОф ОаОЎОАОЎОЋОаОЋОф ОдОеОЎОџ. ОъОќОЌОюОф ОЕОю 0x10000 ОЉОфОЎОЮ ОЋОъОбОюОћ ОћОЎОљ ОаОДОЋОЊОф ОцОфОЎОЌОћ ОўОЋОЉОћ.

    ОфОеОњОЎОю 6 - ОцОЎОдОЋОЌ tiny_easy ОћОљОъОЎОфОЎ ОбОю ОћОЕОеОф¶

    ОбОЏОЕОЎОЋ ОюОъОўОеОћ ОћОљОъОЎОфОЎОф.

    1. ОћОфОЌОЉОеОЋ Ою-ssh tiny_easy@pwnable.kr -p2222, ОћОбОфОЎОДОЋ ОљОф ОћОЉОЎОаОљОеОЎ ОюОфОЎОДОЎОЎОћ ОќОъОаОЎОф ОЉ-/tmp, ОЋОцОеОДОЋ ОљОЋОфОЋ ОбОЮ objdump -d. ОЋОЊОљОЋ ОЕОћОъОаОњОаОЋОЪ ОќОћОћ ОюОъОћ ОЕОфОеОњОюОфОЮ (ОљОъОфОЋ ОљОф ОћОЉОфОЎОЮ - ОћОЮ ОбОЕОЋОЎОЎОЮ ОюОћОЎОЋОф ОъОбОў ОЕОЋОаОЎОЮ).
    2. ОћОфОљОЎОъОЋ ОљОф Оћ-exploit: ОћОеОЎОдОЋ ОљОф ОћОЉОЎОаОљОеОЎ ОћОљОъОЎОфОЎ ОЊОеОџ execve ОбОЮ argv[0] ОаОЕОюОў ОЋ-shellcode ОЉОАОЉОЎОЉОћ, ОЉОюОЋОюОљОф brute force ОЉОњОюОю Оћ-ASLR.
    3. ОЏОЕОаОцОфОЌ shell, ОћОеОЎОдОЋ cat flag. ОќОЏОеОЋ ОЕОћ-shell ОеОЦ ОЉОћОеОЕОљОЋОф ОћОъОЕОфОъОЕ ОћОъОЎОЋОЌОА ОЉОќОЏОЋОф ОЉОЎОў Оћ-setuid.

    ОеОъОќ: ОљОЮ ОљОЎОЪ pwntools ОбОю ОћОЕОеОф, ОЏОфОЉОЋ ОфОЋОЏОаОЎОф ОбОўОЎОцОћ ОДОдОеОћ ОЉОцОЎОЎОфОЋОЪ ОЕОДОЋОеОљОф Ою-os.execve('./tiny_easy', [struct.pack('<I', addr)], env), ОЋОћОеОЎОдОЋ ОљОЋОфОћ ОЉОюОЋОюОљОф shell: while true; do python3 wrap.py 2>/dev/null; done.

    ОеОъОќ: ОљОЮ ОДОЕОћ ОюОќОћОЋОф ОъОфОЎ ОаОцОфОЌ shell ОЉОфОЋОџ ОћОюОЋОюОљОћ, ОЕОюОЌОЋ ОцОДОЋОЊОћ ОЏОъОЋ echo HIT; cat flag ОЋОЌОцОЕОЋ ОљОф ОћОъОЌОеОЋОќОф HIT ОЉОцОюОў.

    ОфОеОњОЎОю 7 (ОъОЌОЕОЉОћ) - ОюОћОЏОюОЎОю ОљОф ОъОћ ОЕОюОъОЊОаОЋ¶

    ОЉОюОЎ ОюОЏОфОЋОЉ ОДОЋОЊ, ОбОаОЋ:

    1. ОъОЊОЋОб ОЊОЋОЋОДОљ ELF ОъОЎОаОЎОъОюОЎ (ОЉОюОЎ GNU_STACK) ОаОЋОфОЪ ОюОаОЋ ОъОЌОАОаОЎОф ОаОЎОфОаОф ОюОћОеОдОћ ОЉОЌОЎОаОЮ? ОъОћ ОћОЎОћ ОъОЕОфОаОћ ОљОЮ ОћОЉОЎОаОљОеОЎ ОћОЎОћ ОЏОЋОюОю GNU_STACK ОбОЮ ОЊОњОю RW ОЉОюОЉОЊ?
    2. Оћexploit ОаОЕОбОЪ ОбОю ОЏОџ ОЕОћОДОЋОЊ ОДОЋОцОЦ ОљОю ОбОеОџ ОЕОъОњОЎОб Оъ-argv[0]. ОљОЎОюОЋ ОбОЋОЊ ОъОДОЋОеОЋОф ОДОюОў "ОъОЋОДОЊОъОЎОЮ" (ОюОцОаОЎ main) ОбОЕОЋОЎОЎОЮ ОюОћОЎОЋОф ОЉОЕОюОЎОўОф ОћОфОЋОДОБ ОЉОЉОЎОаОљОеОЎ ОЏОќОћ?
    3. ОъОЊОЋОб ОћОЕОфОъОЕОаОЋ ОЉОъОЕОфОаОћ ОАОЉОЎОЉОћ ОЏОЊОЎ ОюОљОЌОАОЪ ОљОф Оћ-shellcode ОЋОюОљ, ОаОаОЎОЌ, ОЉОфОЋОџ argv[0] ОбОдОъОЋ? ОъОћ ОћОъОњОЉОюОћ ОбОю argv[0] ОЕОъОЋОаОбОф ОюОЕОЎОЮ ОЉОЋ ОљОф ОЏОю Оћ-shellcode?

    ОеОъОќ: ОЌОЕОЉОЋ ОбОю ОћОњОЋОЊОю ОЋОбОю ОДОўОЎОбОћ. argv[0] ОдОеОЎОџ ОЕОљОеОЉОбОф ОћОЉОфОЎОЮ ОћОеОљОЕОЋОаОЎОЮ ОЕОюОЋ ОЎОћОЎОЋ ОЏОфОЋОЉОф ОДОцОЎОдОћ - ОљОЉОю ОљОцОЕОе ОюОЕОЎОЮ ОљОЌОеОЎОћОЮ ОбОЋОЊ ОЉОфОЎОЮ. ОъОЕОфОаОћ ОАОЉОЎОЉОћ ОаОЋОфОЪ ОъОеОЌОЉ ОњОЊОЋОю ОЋОаОЋОЌ ОюОъОќОЌОюОф ОЋОю-shellcode ОЉОюОЎ ОюОцОњОЋОб ОЉОЏОфОЋОЉОф ОћОДОцОЎОдОћ.
