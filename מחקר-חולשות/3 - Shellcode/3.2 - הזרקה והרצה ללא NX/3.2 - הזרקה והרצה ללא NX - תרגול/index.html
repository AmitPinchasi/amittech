    3.2 - ОћОќОеОДОћ ОЋОћОеОдОћ ОюОюОљ NX - ОфОеОњОЋОю

    ОфОеОњОЋОю - ОћОќОеОДОћ ОЋОћОеОдОћ ОюОюОљ NX¶

    ОЉОфОеОњОЋОю ОћОќОћ ОфОЉОаОЋ ОЉОбОдОъОЏОЮ ОЉОЎОаОљОеОЎ vulnerable ОбОЮ ОъОЌОАОаОЎОф ОаОЎОфОаОф ОюОћОеОдОћ, ОЋОфОцОеОдОЋ ОљОЋОфОЋ ОЉОЕОфОЎ ОЊОеОЏОЎОЮ ОЕОЋОаОЋОф: ОцОбОЮ ОЉОЕОЎОўОф ОћОЏОфОЋОЉОф ОћОЎОЕОЎОеОћ (ОбОЮ ОъОќОЌОюОф NOP), ОЋОцОбОЮ ОЉОЕОЎОўОф ОћОўОеОъОцОЋОюОЎОаОћ jmp esp. ОюОЉОАОЋОБ, ОЏОЉОЋОаОЋОА, ОфОЕОфОъОЕОЋ ОЉОъОЕОфОаОћ ОАОЉОЎОЉОћ. ОћОъОўОеОћ ОћОЎОљ ОЕОфОеОњОЎОЕОЋ ОбОю ОћОЎОЊОЎОЎОЮ ОљОф ОћОћОЉОЊОю ОЉОЎОЪ ОЕОЎОўОћ ОЕОЕОЉОЎОеОћ ОъОЋОю ASLR ОюОЉОЎОЪ ОЕОЎОўОћ ОЕОбОъОЎОЊОћ ОЉОцОаОЎОЋ. ОбОЉОЊОЋ ОюОцОЎ ОћОАОЊОе - ОЏОю ОфОеОњОЎОю ОЉОЋОаОћ ОбОю ОћОДОЋОЊОЮ.

    ОћОЏОаОћ - ОћОЉОЎОаОљОеОЎ ОЋОћОАОЉОЎОЉОћ¶

    ОдОеОЋ ОДОЋОЉОЦ ОъОДОЋОе vuln.c:

    // vuln.c
#include <stdio.h>
#include <unistd.h>

void vuln() {
    char buf[64];
    puts("send your payload:");
    read(0, buf, 400);   // overflow: 400 bytes into a 64-byte buffer
}

int main() {
    setvbuf(stdout, NULL, _IONBF, 0);
    vuln();
    return 0;
}

    ОДОъОцОюОЋ ОљОЋОфОЋ Ою-32 ОЋОњОЮ Ою-64 ОЉОЎОў, ОбОЮ ОъОЌОАОаОЎОф ОаОЎОфОаОф ОюОћОеОдОћ, ОЉОюОЎ canary ОЋОЉОюОЎ PIE (ОљОЮ ОЌОАОеОЋОф ОАОцОеОЎОЋОф 32 ОЉОЎОў: sudo apt install gcc-multilib):

    gcc -m32 -z execstack -fno-stack-protector -no-pie -o vuln32 vuln.c
gcc      -z execstack -fno-stack-protector -no-pie -o vuln64 vuln.c

    ОЏОЉОЋ ASLR (ОюОеОЋОЉ ОћОфОеОњОЎОюОЎОЮ ОљОаОЌОаОЋ ОъОАОфОъОЏОЎОЮ ОбОю ОЏОфОЋОЉОф ОЎОдОЎОЉОћ):

    echo 0 | sudo tee /proc/sys/kernel/randomize_va_space

    ОфОеОњОЎОю 1 - ОљОЎОъОЋОф ОћОфОаОљОЎОЮ¶

    ОюОцОаОЎ ОЕОфОЋОДОцОЎОЮ, ОЋОЊОљОЋ ОЕОћОфОДОЎОцОћ ОЉОЏОюОю ОљОцОЕОеОЎОф.

    1. ОћОеОЎОдОЋ checksec --file=./vuln32 ОЋо╝-checksec --file=./vuln64. ОЋОЊОљОЋ ОЕОљОфОЮ ОеОЋОљОЎОЮ NX disabled ОЋ-Has RWX segments, ОЋОњОЮ No canary ОЋ-No PIE.
    2. ОЉОЊОДОЋ ОљОф ОЊОњОюОЎ ОъОДОўОб ОћОъОЌОАОаОЎОф: readelf -l vuln64 | grep GNU_STACK. ОЋОЊОљОЋ ОЕОљОфОЮ ОеОЋОљОЎОЮ RWE ОЋОюОљ RW.
    3. ОЋОЊОљОЋ ОЕ-ASLR ОЏОЉОЋОЎ: cat /proc/sys/kernel/randomize_va_space ОдОеОЎОџ ОюОћОЌОќОЎОе 0.

    ОеОъОќ: ОљОЮ checksec ОъОеОљОћ NX enabled, ОЕОЏОЌОфОЮ ОљОф -z execstack. ОДОъОцОюОЋ ОъОЌОЊОЕ. ОЉОюОЎ ОћОфОаОљОЎ ОћОќОћ ОЏОю ОЕОљОе ОћОфОеОњОЋОю ОюОљ ОЎОбОЉОЋОЊ.

    ОфОеОњОЎОю 2 - ОъОдОЎОљОф offset ОЋОЏОфОЋОЉОф ОЎОЕОЎОеОћ (64 ОЉОЎОў)¶

    ОбОЏОЕОЎОЋ ОаОцОеОЋОЦ ОљОф vuln64 ОЉОЕОЎОўОф ОћОЏОфОЋОЉОф ОћОЎОЕОЎОеОћ.

    1. ОъОдОљОЋ ОљОф Оћ-offset ОбОЊ Оћreturn address ОбОЮ cyclic (ОЉОЕОЎОўОћ Оъ-2.2): ОЕОюОЌОЋ cyclic 200, ОеОљОЋ ОљОЎОќОћ ОбОеОџ ОЊОеОА ОљОф rip, ОЋОћОеОЎОдОЋ cyclic_find(...).
    2. ОЉ-gdb, ОЕОЎОъОЋ breakpoint ОљОЌОеОЎ Оћ-read, ОЕОюОЌОЋ ОДОюОў ОъОќОћОћ (ОюОъОЕОю ОћОеОЉОћ A), ОЋОЌОцОЕОЋ ОљОЎОцОћ ОћОЋОљ ОаОЌОф: search -s "AAAAAAAA". ОеОЕОъОЋ ОљОф ОЏОфОЋОЉОф ОфОЌОЎОюОф Оћbuffer.
    3. ОЏОфОЉОЋ exploit ОЉ-pwntools: padding ОбОЊ Оћ-offset, ОЋОљОќ return address ОЕОъОдОЉОЎОбОћ ОюОфОЌОЎОюОф Оћbuffer, ОЋОљОќ Оћ-shellcode. ОћОЕОфОъОЕОЋ ОЉ-shellcode ОЕОю 64 ОЉОЎОў (ОћОЎОЊ-ОЏОфОЋОЉ Оъ-3.1, ОљОЋ asm(shellcraft.amd64.linux.sh())).
    4. ОћОеОЎОдОЋ ОЋо╝ОЋОЊОљОЋ ОЕОаОцОфОЌ shell. ОаОАОЋ id.

    ОеОъОќ: ОЏОЊОЎ ОЕОћОДОцОЎОдОћ ОфОаОЌОф ОбОю Оћ-shellcode, ОЕОЎОъОЋ ОљОф Оћ-shellcode ОЉОфОЌОЎОюОф Оћbuffer, ОЋОЏОЋОЋОаОЋ ОљОф Оћreturn address ОЉОЊОЎОЋОД ОюОЕОЮ. ОљОЉОю ОЏОфОЋОЉОф Оћbuffer ОЕОеОљОЎОфОЮ ОЉ-gdb ОЏОаОеОљОћ ОЕОЋОаОћ ОъОћОћОеОдОћ ОћОљОъОЎОфОЎОф - ОќОћ ОЉОЊОЎОЋОД ОъОћ ОЕОфОеОњОЎОю 3 ОцОЋОфОе.

    ОфОеОњОЎОю 3 - ОъОќОЌОюОф NOP ОЋОћОфОъОЋОЊОЊОЋОф ОбОЮ ОљОЎ-ОЊОЎОЋОД¶

    ОфОеОњОЎОю 2 ОбОЋОЉОЊ ОЉ-gdb ОљОЉОю ОљОЋОюОЎ ОДОЋОеОА ОЉОўОеОъОЎОаОю, ОљОЋ ОюОћОцОџ. ОћОАОЎОЉОћ: ОћОАОЉОЎОЉОћ ОЕОЋОаОћ, ОЋОЏОфОЋОЉОф ОћОъОЌОАОаОЎОф ОќОќОћ. ОбОЏОЕОЎОЋ ОаОбОЕОћ ОљОф ОћОћОфОДОцОћ ОљОъОЎОаОћ.

    1. ОЕОаОЋ ОљОф ОћОъОЉОаОћ: ОЕОЎОъОЋ ОљОф Оћ-shellcode ОљОЌОеОЎ Оћreturn address, ОбОЮ ОъОќОЌОюОф ОњОЊОЋОюОћ ОЕОю \x90 ОюОцОаОЎОЋ (ОюОъОЕОю 512 ОЉОфОЎОЮ).
    2. ОЏОЋОЋОаОЋ ОљОф Оћreturn address ОљОю ОљОъОдОб ОћОъОќОЌОюОф, ОюОљ ОљОю ОћОћОфОЌОюОћ ОћОъОЊОЋОЎОДОф.
    3. ОЌОЉОеОЋ gdb.attach(p) ОюОфОћОюОЎОџ, ОЌОцОЕОЋ ОљОф ОћОъОќОЌОюОф (search -s "\x90\x90\x90\x90"), ОЋОЉОЌОеОЋ ОЏОфОЋОЉОф ОбОъОЋОД ОЉОфОЋОЏОћ.
    4. ОћОеОЎОдОЋ ОљОф Оћ-exploit ОъОЌОЋОЦ Ою-gdb (ОАОфОЮ python3 exploit.py) ОЋо╝ОЋОЊОљОЋ ОЕОћОЋОљ ОбОЊОЎОЎОЪ ОбОЋОЉОЊ. ОљОЮ ОюОљ, ОћОњОЊОЎОюОЋ ОљОф ОћОъОќОЌОюОф.

    ОеОъОќ: ОћОъОќОЌОюОф ОАОЋОцОњОф ОљОф ОћОћОцОеОЕ ОЉОЎОЪ ОћОАОЉОЎОЉОћ ОЕОю gdb ОюОќОЋ ОЕОю ОћОеОдОћ ОеОњОЎОюОћ. ОљОЮ ОћОъОќОЌОюОф ОњОЊОЋОюОћ ОъОАОцОЎОД, ОљОЋОфОћ ОЏОфОЋОЉОф ОфОбОЉОЋОЊ ОЉОЕОаОЎ ОћОъОДОеОЎОЮ. ОќОЋ ОћОАОЎОЉОћ ОЕОћОъОќОЌОюОф ОћОЋОъОдОљОћ.

    ОеОъОќ: ОЏОЏОю ОЕОћbuffer ОДОўОЪ, ОбОЊОЎОБ ОюОЕОЎОЮ ОљОф ОћОъОќОЌОюОф ОЋОћ-shellcode ОљОЌОеОЎ Оћreturn address - ОЕОЮ ОЎОЕ ОюОЏОЮ ОљОф ОЏОю ОъОеОЌОЉ Оћ-read (400 ОЉОфОЎОЮ) ОЋОюОљ ОеОД 64.

    ОфОеОњОЎОю 4 - ОўОеОъОцОЋОюОЎОаОћ jmp esp (32 ОЉОЎОў)¶

    ОбОЏОЕОЎОЋ ОћОўОЏОаОЎОДОћ ОћОЌОќОДОћ: ОцОеОЎОдОћ ОЉОюОЎ ОюОЊОбОф ОЉОЏОюОю ОљОф ОЏОфОЋОЉОф ОћОъОЌОАОаОЎОф, ОбОю vuln32.

    1. ОъОдОљОЋ ОњОљОЊОњ'Оў jmp esp ОЉОЉОЎОаОљОеОЎ: ROPgadget --binary vuln32 | grep 'jmp esp', ОљОЋ ОЌОцОЕОЋ ОљОф ОћОЉОфОЎОЮ ff e4 ОбОЮ objdump -d vuln32 | grep 'ff e4'.
    2. ОЉОаОЋ payload: padding ОбОЊ Оћ-offset, ОЋОљОќ ОЏОфОЋОЉОф ОћОњОљОЊОњ'Оў jmp esp ОЉОћreturn address, ОЋОљОќ Оћ-shellcode ОЕОю 32 ОЉОЎОў ОъОЎОЊ ОљОЌОеОЎОћ.
    3. ОћОеОЎОдОЋ ОЋо╝ОЋОЊОљОЋ ОЕОаОцОфОЌ shell.
    4. ОбОЏОЕОЎОЋ ОћОЊОЉОе ОћОъОбОаОЎОЎОЪ: ОћОЊОюОЎОДОЋ ASLR ОъОЌОЊОЕ (echo 2 | sudo tee /proc/sys/kernel/randomize_va_space) ОЋОћОеОЎОдОЋ ОЕОЋОЉ. ОћОћОфОДОцОћ ОбОЊОЎОЎОЪ ОбОЋОЉОЊОф! ОћОАОЉОЎОеОЋ ОюОбОдОъОЏОЮ ОюОъОћ.

    ОеОъОќ: ОЉОеОњОб Оћ-ret, esp ОъОдОЉОЎОб ОЉОЊОЎОЋОД ОбОю ОћОАОюОЋОў ОЕОљОЌОеОЎ Оћreturn address - ОЕОЮ ОЕОъОфОЮ ОљОф Оћ-shellcode. ОћОњОљОЊОњ'Оў jmp esp ОДОЋОцОЦ ОюОЕОЮ. ОюОљ ОаОњОбОфОЮ ОЉОЏОфОЋОЉОф ОъОЌОАОаОЎОф ОЉОЏОюОю.

    ОеОъОќ: ОћОАОЎОЉОћ ОЕОќОћ ОбОЋОЉОЊ ОњОЮ ОбОЮ ASLR: ОћОЉОЎОаОљОеОЎ ОюОљ PIE, ОљОќ ОЏОфОЋОЉОф ОћОњОљОЊОњ'Оў jmp esp ОДОЉОЋОбОћ. Оћ-ASLR ОъОбОеОЉОЉ ОљОф ОћОъОЌОАОаОЎОф, ОљОЉОю ОюОљ ОљОЏОцОф ОюОаОЋ ОЏОЎ esp ОъОъОЎОюОљ ОъОдОЉОЎОб ОюОљОЪ ОЕОдОеОЎОџ.

    ОфОеОњОЎОю 5 (ОЉОЋОаОЋОА) - shellcode ОЉОъОЕОфОаОћ ОАОЉОЎОЉОћ¶

    ОъОћ ОбОЋОЕОЎОЮ ОЏОЕОћbuffer ОДОўОЪ ОъОЏОЊОЎ ОюОћОЏОЎОю shellcode + ОъОќОЌОюОф? ОъОЏОаОЎОАОЎОЮ ОљОЋОфОЮ ОюОъОЕОфОаОћ ОАОЉОЎОЉОћ.

    1. ОћОЏОаОЎОАОЋ ОюОъОЕОфОаОћ ОАОЉОЎОЉОћ SC ОъОќОЌОюОф ОњОЊОЋОюОћ ОЕОю \x90 ОЋОљОЌОеОЎОћ Оћ-shellcode.
    2. ОЌОЕОЉОЋ ОљОф ОЏОфОЋОЉОф ОћОъОЕОфОаОћ ОбОю ОћОъОЌОАОаОЎОф. ОЏОфОЉОЋ ОљОф ОфОЋОЏОаОЎОф ОћОбОќОе getenvaddr (ОъОћОћОеОдОљОћ), ОљОЋ ОљОъОфОЋ ОљОф ОћОЏОфОЋОЉОф ОЎОЕОЎОеОЋОф ОЉ-gdb ОбОЮ search -s SC.
    3. ОЏОфОЉОЋ exploit ОЕОЊОЋОеОА ОљОф Оћreturn address ОЉОЏОфОЋОЉОф ОЕОю SC (ОъОЏОЋОЋОаОф ОюОфОЋОџ ОъОќОЌОюОф Оћ-NOP ОЕОЉОфОЋОЏОЋ). ОћОеОЎОдОЋ ОЋо╝ОЋОЊОљОЋ shell.

    ОеОъОќ: ОљОЮ ОъОЕОфОъОЕОЎОЮ ОЉ-pwntools, ОљОцОЕОе ОюОћОбОЉОЎОе ОљОф ОћОАОЉОЎОЉОћ ОЎОЕОЎОеОЋОф: process('./vuln32', env={'SC': b'\x90'*400 + sc}), ОЋОљОќ ОюОЌОцОЕ ОљОф ОћОЏОфОЋОЉОф ОЕОю SC ОЉ-gdb.attach.

    ОеОъОќ: ОЏОфОЋОЉОф ОъОЕОфОаОћ ОћОАОЉОЎОЉОћ ОфОюОЋОЎОћ ОЉОљОЋОеОџ ОЕОЮ ОћОфОЋОЏОаОЎОф (argv[0]) ОЋОЉОњОЋОЊОю ОЕОљОе ОћОАОЉОЎОЉОћ. ОљОЮ ОћОЏОфОЋОЉОф ОДОЋОцОдОф, ОъОќОЌОюОф ОњОЊОЋОюОћ ОЎОЋОфОе ОЉОфОЋОџ SC ОфОЉОюОб ОљОф ОћОАОўОЎОЎОћ.
