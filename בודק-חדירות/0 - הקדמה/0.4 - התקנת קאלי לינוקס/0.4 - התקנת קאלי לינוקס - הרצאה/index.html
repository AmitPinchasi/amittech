    0.4 - התקנת קאלי לינוקס - הרצאה

    סרטון יוטיוב שלי מתקין קאלי.

    התקנת Kali Linux לסביבת בדיקות חדירות¶

    מבוא¶

    מערכת Kali Linux היא אחת ההפצות הנפוצות ביותר לבדיקות חדירות (Pentesting). היא כוללת כלים מתקדמים לאיסוף מידע, תקיפה, ניתוח חולשות והגנות. בהתקנה נכונה, Kali מספקת סביבת עבודה מאובטחת ונוחה למבחני אבטחה ולמידת תחום הסייבר.

    למה Kali Linux חשובה לבודקי חדירות?¶

    1. כלים מובנים – Kali מגיעה עם מאות כלים ייעודיים לבדיקות חדירות, כמו Metasploit, Nmap, Burp Suite ועוד.
    2. שימוש בסביבה מבודדת – התקנת Kali בסביבה וירטואלית מאפשרת בדיקות ללא סיכון למערכת ההפעלה הראשית.
    3. עדכוני אבטחה תכופים – הפצה זו מתעדכנת באופן קבוע כדי להתאים לכלים החדשים ביותר בתחום הסייבר.
    4. תמיכה בקהילה רחבה – קיימים מדריכים רבים, קורסים ותמיכה מקוונת לכלי ההפצה ולמגוון השימושים האפשריים שלה.

    התקנת VMware Workstation Player (Community Edition)¶

    לפני התקנת Kali, נצטרך תוכנה להרצת מכונות וירטואליות. אחת מהאפשרויות הנוחות ביותר היא VMware Workstation Player, שהיא חינמית לשימוש אישי.

    התקנה ב-Windows:¶

    1. הורדת התוכנה

      • היכנסו לאתר הרשמי של VMware Workstation Player.
      • הורידו את גרסת ההתקנה התואמת למערכת ההפעלה שלכם.

      • התקנה

      • הפעילו את קובץ ההתקנה ולחצו Next בכל השלבים.

      • אשרו את תנאי הרישיון ולחצו Install.
      • לאחר סיום ההתקנה, הפעילו את התוכנה ובדקו שהיא פועלת ללא שגיאות.

    התקנת Kali Linux על VMware¶

    לאחר שהתקנו VMware, נתקין את Kali Linux.

    1. הורדת Kali Linux¶

    • גשו לאתר הרשמי: https://www.kali.org/get-kali/
    • בחרו Kali Linux Virtual Machines להורדה מותאמת ל-VMware.
    • אם אתם מעדיפים אפשר גם להוריד ISO, אך אם אתם משתמשים בvmware תוכלו גם להוריד ממש קובץ vmx מהאתר שמותאם ישירות לvmware יודע להתקין את עצמו במהירות.

    2. התקנה דרך VMware (אם בחרתם באופציה של ISO)¶

    1. פתחו את VMware Workstation Player ולחצו Create a New Virtual Machine
    2. בחרו באופציה Use ISO Image ובחרו את הקובץ שהורדתם.
    3. הגדרות מומלצות:
      • מעבד: לפחות 2 ליבות
      • זכרון RAM: לפחות 4GB, מומלץ 8GB ומעלה (תלוי בכמה יש לכם במחשב הhost)
      • שטח דיסק: 20GB ומעלה (מומלץ 40GB)
    4. לחצו Finish והפעילו את ה-VM.
    5. עקבו אחרי ההוראות על המסך להשלמת ההתקנה.

    עבודה בסיסית עם Kali Linux¶

    לאחר התקנת Kali Linux, חשוב להבין כיצד לעבוד בסביבה, להכיר את סוגי interfaces המשתמש ולדעת כיצד לנווט במהירות בין הכלים.

    1. הכרת סביבת העבודה¶

    משתמש ברירת המחדל¶

    • שם משתמש: kali
    • סיסמה: kali
    • משתמש זה הוא בעל הרשאות גבוהות (sudo), אך אפשר גם להפעיל את המערכת כ-root לפי הצורך.

    הinterfaces משתמש ב-Kali¶

    מערכת ההפעלה Kali תומכת בכמה סביבות עבודה (Desktop Environments), כאשר ברירת המחדל היא XFCE, אך קיימות גם אפשרויות נוספות:

    • XFCE – קליל, מהיר ויעיל.
    • GNOME – בעל עיצוב מודרני וכבד יותר.
    • KDE Plasma – ידידותי למשתמש, דומה לחלונות עם אפשרויות התאמה רבות.

    חשבו על זה כמו אפשרות לבחור "GUI" שונה למערכת ההפעלה שלכם.
    (גם אחרי שתבחרו אחד מהם תוכלו לשנות)

    2. קיצורי מקלדת שימושיים ב-Kali¶

    קיצורים כלליים¶

    • Alt + Tab – מעבר בין חלונות פתוחים.
    • Ctrl + Alt + T – פתיחת הטרמינל.
    • Super (מקש Windows) + D – הצגת שולחן העבודה.
    • Super + L – נעילת המסך.
    • Print Screen – צילום מסך (ניתן לשמור או להעתיק).

    קיצורים ייחודיים ל-KDE¶

    • Alt + F2 – הפעלת תוכנה ישירות מהחיפוש.
    • Ctrl + F12 – הצגת ווידג'טים על שולחן העבודה.
    • Meta (Windows) + E – פתיחת מנהל הקבצים.

    3. ניווט במערכת¶

    • גישה לכלים – דרך Applications → בחירה בקטגוריה הרצויה.
    • חיפוש מהיר – בלחיצה על Super (Windows Key) אפשר להקליד ולמצוא כלים במהירות.
    • הגדרות מערכת – דרך Settings ניתן לשנות את ערכות הנושא, קיצורי הדרך ותצוגת המסך.

    סיכום¶

    לאחר התקנה מוצלחת, יש לכם סביבת Kali Linux מוכנה לבדיקות חדירות. זוהי פלטפורמה חשובה לכל חוקר אבטחה, המאפשרת בדיקות מאובטחות ויעילות. בהרצאות הבאות נעמיק בשימוש בכלים השונים שבתוכה.
