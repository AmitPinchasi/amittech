# קורס פיתוח עם AI

קורס שלוקח אתכם מהשלב שבו AI הוא צעצוע נחמד שכותב לכם פונקציה, ועד ליכולת להוביל סוכן AI שכותב, מתקן ובודק קוד אמיתי בפרויקטים שלמים - ולעשות את זה ברמה הכי גבוהה שיש היום. לא נלמד רק איזה כפתור ללחוץ, אלא איך לחשוב עם הכלים האלה: איך נותנים להם הקשר, איך מנסחים בקשה שמביאה תוצאה מעולה, מתי לסמוך עליהם ומתי לא, ואיך נשארים המהנדס שאחראי על הקוד גם כשרובו נכתב על ידי מכונה.

הקורס בנוי סביב פיתוח אג'נטי - agentic development, הדרך המודרנית והחזקה ביותר לעבוד עם AI. הכלי המרכזי שנלמד לעומק הוא Claude Code, סוכן שרץ בטרמינל וקורא, כותב ומריץ קוד בפרויקט שלכם. סביב הליבה הזו נכיר את כל שאר הנוף: עבודה עם צ'אט והנדסת פרומפט, כלים כמו Cursor ו-Copilot, ובסוף גם נבנה כלים חכמים משלנו מעל ה-API של המודל. המטרה היא שתצאו מהקורס עם ארגז כלים שלם, ועם השיפוט לבחור את הכלי הנכון לכל משימה.

הקורס עומד בפני עצמו. הוא לא דורש שום קורס אחר מעבר לידע תכנות בסיסי ונוחות בסיסית בטרמינל. את כל המושגים של AI וסוכנים אנחנו בונים מאפס.

## דרישות קדם

- ידע תכנות בסיסי - היכרות עם משתנים, פונקציות ולולאות בשפה כלשהי. אם סיימתם את [תכנות בסיסי](../תכנות-בסיסי/), אתם מוכנים בגדול.
- נוחות בסיסית בטרמינל - להריץ פקודות, לנווט בין תיקיות, ולעבוד עם git ברמה בסיסית.

אין צורך בידע קודם בבינה מלאכותית, במודלים של שפה או בסוכנים. כל אלה נלמדים בקורס.

## מה צריך כדי לתרגל

- מחשב (Windows, macOS או לינוקס) עם חיבור לאינטרנט.
- חשבון Claude - בחלק מהתרגולים נשתמש ב-Claude.ai וב-Claude Code. יש שכבה חינמית להתנסות.
- אופציונלי: חשבונות לכלים הנוספים שנסקור (Cursor, GitHub Copilot ואחרים). לרובם יש שכבת ניסיון חינמית.
- לפרק הבנייה - מפתח API של Claude וסביבת פיתוח בסיסית של Python או JavaScript.

בפרק 0 נעבור על התקנת Claude Code והכנת הסביבה צעד אחר צעד.

## תוכן הקורס

### פרק 0 - הקדמה: המהפכה בפיתוח עם AI
מה השתנה בפיתוח תוכנה, ולמה זה משנה את העבודה של כל מתכנת. נבנה מודל מנטלי נכון של מודל שפה - איך הוא חושב, במה הוא מצוין ובמה הוא נכשל, נמפה את נוף הכלים (צ'אט, תוסף IDE, סוכן טרמינל), ונקים את סביבת העבודה שתלווה אותנו לאורך הקורס.

### פרק 1 - עבודה עם צ'אט והנדסת פרומפט
היסודות שעובדים מול כל מודל. מתי צ'אט הוא הכלי הנכון, איך בונים פרומפט שמביא תשובה מעולה, טכניקות מתקדמות כמו דוגמאות ושרשור מחשבה, איך בוחרים מודל ומבינים עלות, ואוסף מתכונים מעשיים שתשתמשו בהם כל יום.

### פרק 2 - פיתוח אג'נטי עם Claude Code
כניסה לעולם הסוכנים. מה זה בעצם פיתוח אג'נטי ואיך עובדת לולאת הסוכן, הריצה הראשונה שלכם, מודל ההרשאות ומצבי העבודה, מצב התכנון, וקובץ ה-CLAUDE.md שנותן לסוכן הקשר על הפרויקט.

### פרק 3 - זרימות עבודה יומיומיות
כאן הופכים את הסוכן לחלק מהעבודה. הבנת codebase לא מוכר, כתיבת פיצ'ר מקצה לקצה, דיבוג שיטתי, עבודה עם git, כתיבת בדיקות refactoring ותיעוד, והמיומנות החשובה מכולן - אימות, איך לא לתת לסוכן לשכנע אתכם שמשהו עובד כשהוא לא.

### פרק 4 - שליטה בהקשר ובפרודוקטיביות
מה שמפריד בין משתמש מתחיל למקצוען. ניהול הקשר - context, למה הוא משאב יקר ואיך שומרים עליו, פקודות סלאש מובנות ומותאמות אישית, עבודה עם קבצים גדולים תמונות ופלט ארוך, והרגלי פרודוקטיביות שמכפילים את המהירות.

### פרק 5 - סקירת הכלים האחרים
לא מתחתנים עם כלי אחד. נכיר את Cursor, את GitHub Copilot, סוכני CLI נוספים, כלים ליצירת ממשקים, וכלים לסקירת קוד אוטומטית - ונלמד איך בוחרים ומשלבים ביניהם לפי המשימה.

### פרק 6 - Claude Code מתקדם
העמקה אמיתית בסוכן. פרוטוקול MCP שמחבר את הסוכן לכלים ולמידע חיצוני, שרתי MCP שימושיים, סוכני משנה שעובדים במקביל, הוקים שמפעילים אוטומציה, מיומנויות ופקודות מותאמות, וזרימות עבודה של כמה סוכנים למשימות גדולות.

### פרק 7 - בנייה עם AI
מהפכים ממשתמש בכלים לבונה כלים. הבקשה הראשונה ל-API של Claude, שימוש בכלים - tool use, בניית סוכן משלכם עם ה-Agent SDK, כתיבת שרת MCP פשוט, ודפוסים מתקדמים כמו RAG, שמירת הקשר במטמון ופלט מובנה.

### פרק 8 - מקצועיות, מגבלות ואתיקה
לסגור את התמונה כמהנדסים אחראים. איפה AI נכשל, שיקולי אבטחה בעבודה עם סוכנים, סקירה ואחריות על הקוד, עלות פרטיות ורישוי, וההרגלים שהופכים אתכם למפתח AI מעולה - עם פרויקט מסכם שמפעיל את כל מה שלמדנו.
