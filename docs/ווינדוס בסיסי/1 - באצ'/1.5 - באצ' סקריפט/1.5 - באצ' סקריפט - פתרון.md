### פתרון לתרגול 1: יצירת קובץ Batch פשוט

```batch
@echo off
echo Hello Batch!
```

---

### פתרון לתרגול 2: עבודה עם ארגומנטים

```batch
@echo off
echo %1
echo %2
echo %3
```

---

### פתרון לתרגול 3: פעולות חישוביות

```batch
@echo off
SET /A a=5
SET /A b=10
SET /A sum=%a% + %b%
echo %sum%
```

---

### פתרון לתרגול 4: שימוש בתנאים

```batch
@echo off
if %1%==10 (
    echo The value is 10
) else (
    echo Unknown value
)
```

---

### פתרון לתרגול 5: לולאת `for` עם מספרים

```batch
@echo off
FOR /L %%G IN (1,1,5) DO echo %%G
```

---

### פתרון לתרגול 6: לולאת `for` עם מחרוזות

```batch
@echo off
FOR %%G IN (Sun Mon Tue Wed Thur Fri Sat) DO echo %%G
```

---

### פתרון לתרגול 7: יצירת קבצים בעזרת לולאת `for`

```batch
@echo off
FOR /L %%G IN (1,1,5) DO (
    copy nul NewFile%%G.txt
)
```

---

### פתרון לתרגול 8: שימוש ב-Delayed Expansion

```batch
@echo off
setlocal EnableDelayedExpansion
set var=0
FOR /L %%G IN (1,1,5) DO (
    echo !var!
    set /a var+=1
)
echo Total = %var%
```
