### תרגילים מסכמים על אובייקטי COM ב-PowerShell

1. **יצירת אינטגרציה עם Excel**:
    
    - צור סקריפט ב-PowerShell שמשתמש באובייקט COM של Microsoft Excel כדי ליצור גיליון עבודה חדש, להוסיף אליו נתונים (כמו שם, גיל, כתובת), ולהחיל עליו עיצוב (לדוגמה, להדגיש את שם המשתמש או להוסיף צבע לתאים). שמור את הגיליון כקובץ Excel במחשבך.
2. **אוטומציה של שליחת מייל ב-Outlook**:
    
    - צור סקריפט ב-PowerShell שמשתמש באובייקט COM של Microsoft Outlook לשלוח דואר אלקטרוני. המייל צריך לכלול שדה נושא, גוף (Body) עם טקסט מותאם אישית, וכתובת מייל כיעד.
3. **יצירת מסמך Word עם טבלאות**:
    
    - צור סקריפט ב-PowerShell שמשתמש באובייקט COM של Microsoft Word כדי ליצור מסמך חדש. הוסף לתוכו טבלה עם 3 עמודות ו-5 שורות, מלא את הטבלה בנתונים דינמיים (כגון נתונים על עובדים או תלמידים), ועצב את הטבלה כך שהכותרות יהיו מודגשות.
4. **הפעלת יישום חיצוני בעזרת COM**:
    
    - צור סקריפט ב-PowerShell שמתחבר לתוכנה חיצונית באמצעות COM, למשל אובייקט COM שמייצג מכשיר חומרה (כגון מדפסת). השתמש באובייקט כדי לבצע פעולה כמו הדפסה או שליחת נתונים למכשיר החומרה.
5. **אינטגרציה עם Microsoft PowerPoint**:
    
    - צור סקריפט ב-PowerShell שמשתמש באובייקט COM של Microsoft PowerPoint כדי ליצור מצגת חדשה, להוסיף לה שקפים עם טקסט ותמונות, ולעצב את השקפים בצורה מסוימת (למשל צבעי רקע, סגנונות כותרת).
6. **ניהול תהליך חיפוש ב-Word**:
    
    - צור סקריפט ב-PowerShell שמשתמש באובייקט COM של Microsoft Word כדי לחפש טקסט מסוים במסמך פתוח ולהדגיש אותו. הסקריפט אמור גם לדווח אם הטקסט נמצא או לא נמצא במסמך.
