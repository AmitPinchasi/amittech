### דרישות מוקדמות

- **ידע בתכנות:** הכרות עם שפת תכנות אחת לפחות (מומלץ פייתון), באתר מוצע "קורס תכנות בסיסי" מומלץ לעשות לפני. 
- **הכרות עם Linux ו-Bash:** רצוי לדעת לעבוד עם הטרמינל ולבצע פקודות בסיסיות, באתר מוצע "קורס לינוקס בסיסי" מומלץ לעשות לפני.
- **הכרות עם עולם הרשתות:** רצוי להכיר בצורה בסיסית לפחות את עולם הרשתות, באתר מוצע "קורס רשתות מחשבים" מומלץ לעשות לפני.

### למה לעשות את הקורס?

- ללמוד את עקרונות ההאקינג בעולם הווב, איך מערכות נפרצות, וכיצד לאבטח אותן.
- הכרות עם חולשות נפוצות כמו XSS, CSRF, SQL Injection ועוד.
- התנסות מעשית בכלים כמו Burp Suite, Nikto, ו-Kali Linux.
- יתרון משמעותי לתפקידי אבטחת מידע, בודקי חדירות (Pentesters), ומפתחים שרוצים להבין כיצד לכתוב קוד מאובטח.

- **תרגול:** שני שליש מהזמן יוקדש לתרגולים מעשיים, תרגילים ופרויקטים.
