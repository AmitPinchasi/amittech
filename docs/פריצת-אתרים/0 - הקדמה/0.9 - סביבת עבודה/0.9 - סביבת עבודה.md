## הקדמה
- קודם, לפני שנתחיל ללמוד על איך לתקוף אתרים- אנחנו צריכים לדעת כיצד אתרים עובדים.
- לשם כך נתחיל ללמוד פיתוח אתרים בסיסי עם טכנולוגית "php".
- 77.4% מהאתרים בעולם כתובים בphp- (באזור ה2 מיליארד אתרים) כך שנתקל הרבה הטכנולוגיה הזו. 
- לשם כך, שווה מאוד לדעת לפתח בphp ולתקוף.

### יצירת אתר אינטרנט
כדי שאנחנו נוכל ליצור אתר משלנו אנחנו צריכים להוריד תוכנה שתנהל את כל תהליך האזנה לפורט (למשל פורט 80,) ותקשורת עם הלקוחות (מי שגולש לאתר).

כעת, אחרי שהבנו את הרעיון הכללי של שרת ואת מה שהוא עושה, אנחנו רוצים להתקין את התוכנה שתהפוך את המחשב שלנו לשרת אינטרנט מקומי. במקרה הזה, התוכנה שאנחנו הולכים להשתמש בה היא XAMPP.

#### מה זה XAMPP?

כלי XAMPP היא חבילה שמכילה את Apache (שרת HTTP), MySQL (מערכת לניהול מסדי נתונים), PHP ו-Perl. כל החבילה הזאת מאפשרת לנו להקים שרת אינטרנט אישי על המחשב בפשטות. למעשה, XAMPP הופך את המחשב המקומי שלנו לשרת אינטרנט שבו ניתן לפתח, לבחון ולבדוק אתרים בקלות.

#### שלב 1: הורדה והתקנה של XAMPP

1. **הורדה** – ניגשים לאתר הרשמי של XAMPP [כאן](https://www.apachefriends.org/download.html).
2. בוחרים את הגרסה המתאימה למערכת ההפעלה שלך.
3. מורידים את הקובץ ומתקינים את התוכנה. במהלך ההתקנה, תוכל לבחור אילו רכיבים ברצונך להתקין – אנחנו נבחר בהתקנה של Apache ו-MySQL לפחות, שהם הבסיס להקמת אתר.

#### שלב 2: הפעלת השרת

1. אחרי שההתקנה הושלמה, פותחים את XAMPP Control Panel.
2. בתפריט, נלחץ על "Start" ליד Apache ו-MySQL כדי להפעיל את השרת המקומי.
3. אם הכל עבד כמצופה, אנחנו נוכל לגשת לאתר שלנו על ידי כניסה לדפדפן ולרשום את כתובת ה-URL: `http://localhost`. אם הכל תקין, תראה את דף ברוך הבא של XAMPP.

#### שלב 3: יצירת אתר ראשון

1. עכשיו שהשרת שלנו פעיל, אנחנו רוצים להתחיל לעבוד על האתר שלנו.
2. תמצא את התיקייה בה מותקן XAMPP – בדרך כלל בנתיב `opt/lampp/htdocs/` בלינוקס ובנתיב `C:\xampp\htdocs` ב-Windows
3. כאן, תוכל ליצור תיקיה חדשה ולשים בה את קבצי האתר שלך (HTML, CSS, JavaScript וכדומה).
   כך שאם תיצור קובץ בשם hello.html תוכלו לגשת אילו באמצעות הקישור הבא
   "http://localhost/hello.html"  
   בנוסף, אם הקובץ שאתם יוצרים נקרא index.html אז כשתכנסו לאתר כך:
   "http://localhost/"
   הוא יפנה אתכם לקובץ.
 
תוכלו גם ליצור תיקיות, כך שאם תיצרו תיקיה בשם: "myfirstsite", עם קובץ index.html בפנים, הכתובת שתיתן לך לגישה לקובץ תהיה `http://localhost/myfirstsite`.

#### שלב 4: הוספת קוד HTML בסיסי

כדי לבחון שהכל עובד, פתח את עורך הטקסט שלך וכתוב את הקוד הבא:

```html
<!DOCTYPE html>
<html lang="he">
<head>
    <meta charset="UTF-8">
    <title>אתר ראשון</title>
</head>
<body>
    <h1>ברוך הבא לאתר הראשון שלי!</h1>
    <p>זהו אתר פיתוח שרץ על שרת XAMPP המקומי.</p>
</body>
</html>
```
זהו קובץ html פשוט, כנראה יזכיר לכם פורמט של xml.
שמור את הקובץ בתיקיית האתר שלך בשם `index.html`. עכשיו, כשתיגש לכתובת `http://localhost/myfirstsite`, תראה את האתר שלך.
