## מה הן פונקציות מובנות?
- פונקציות מובנות או באנגלית "built-in functions" הן פונקציות שאנחנו יכולים להשתמש בפייתון כדי לעשות כל מיני פעולות.
- איזה עוד פונקציות קיימות שהן לא מובנות? פונקציות של datatypes, כמו `upper` של מחרוזות. נלמד בהמשך על עוד סוגים של פונקציות.
## פונקציות מובנות שנתקלנו בהן
- `print`
- `input`
- `int`
- `len`
- `sorted`

## פונקציות מובנות שימושיות
```python
# Len
my_list = [1, 2, 3, 4, 5]
print(len(my_list))  # Output: 5
# Type
my_variable = 42
print(type(my_variable))  # Output: <class 'int'>
```
 - מספרים
```python
# Max
numbers = [2, 8, 1, 6, 4]
print(max(numbers))  # Output: 8
# Sum
numbers = [1, 2, 3, 4, 5]
print(sum(numbers))  # Output: 15
# Abs
print(abs(-5))  # Output: 5
# Round
print(round(5.4))  # Output: 5
```
- שינוי סוג משתנה - (type casting)
```python
num = 42
print(str(num))  # Output: '42'
print(float(4))  # Output: 4.0
print(int(4.8))  # Output: 4
```
- רשימות
```python
my_list = [3, 1, 4, 1, 5, 9, 2]
# Sorted
print(sorted(my_list))  # Output: [1, 1, 2, 3, 4, 5, 9]
# Rev
print(list(reversed([1, 2, 3, 4]))) # Output: [4, 3, 2, 1]
# Any
bool_list = [True, False, True]
print(any(bool_list))  # Output: True
```
