## הוסף ב5
```python
number = int(input("enter a number: "))
number += 5
print(number)
```
- יכלנו לכתוב את התוכנה הזו בעוד דרכים, כמו:
```python
number = int(input("enter a number: "))
print(number+5)
```
- שימו לב שיכלנו להשתמש באותה הדרך כדי להכפיל את המספר, לחלק, לחסר או כל פעולה שנרצה.
```python
number = int(input("enter a number: "))
number += 5
number *= 2
number /= 54
number -= 7
print(number)
```
