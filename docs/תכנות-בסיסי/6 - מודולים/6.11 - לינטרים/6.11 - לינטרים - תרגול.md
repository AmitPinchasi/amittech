## הרצת לינטרים
- צרו קובץ `messy_code.py` עם הקוד הבא:
```python
import os
import sys

def Calculate(x,y,operation):
    if operation=="add":
        result=x+y
    elif operation=="sub":
        result=x-y
    elif operation=="mul":
        result=x*y
    elif operation=="div":
        result=x/y
    print(result)
    return result

x=10
y=5
Calculate(x,y,"add")
Calculate(x,y,"div")
```

1. הריצו `pylint` על הקובץ ורשמו לעצמכם את כל האזהרות שהוא מצא.
2. הריצו `black` על הקובץ וראו אילו שינויים הוא עשה אוטמטית.
3. הוסיפו type hinting לפונקציה והריצו `mypy` כדי לוודא שאין בעיות.
4. תקנו את כל האזהרות שlint מצא ידנית (אחרי ש-black כבר תיקן את הפורמט).
