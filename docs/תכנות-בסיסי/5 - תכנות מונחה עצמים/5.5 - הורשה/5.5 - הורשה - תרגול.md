## צורות
- כתוב 2 interfaces: 
	- הinterface: `Shape` - מחייב מחלקה להיות צורה.
	- הinterface: `Drawable` - מחייב מחלקה להיות ניתנת לצביעה.

1. הגדר interface `Shape` עם מתודות `area` ו - `perimeter`.
2. הגדר interface `Drawable` עם מתודה `draw`.
3. ממש מחלקות `Circle` ו - `Rectangle` שמיישמות את `Drawable` ו - `Shape`, שימו לב, תצטרכו שכל מחלקה תירש משני הinterfaces בו זמנית, עושים את זה בדרך הבאה:
```python
class Circle(Shape, Drawable):
```
