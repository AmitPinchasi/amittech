## שומר סף
```python
PASSWORD = "1234"

password = input("הכנס סיסמה: ")

if password != PASSWORD:
    print("סיסמה שגויה. יציאה.")
    exit()

print("ברוכים הבאים!")
print("המשך התוכנה...")
```

## עזרה
הריצו `help(str)` ב-console וחפשו מתודות מעניינות. לדוגמה:

```python
# str.center() - ממרכזת מחרוזת ברוחב נתון
text = "hello"
print(text.center(20, "-"))  # -------hello--------

# str.zfill() - ממלאת בספרות 0 משמאל
number = "42"
print(number.zfill(6))  # 000042
```
