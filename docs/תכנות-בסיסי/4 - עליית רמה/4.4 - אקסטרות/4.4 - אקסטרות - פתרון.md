```python
PASSWORD = "1234"

password = input("Enter password: ")

if password != PASSWORD:
    print("Wrong password. Exiting.")
    exit()

print("Welcome!")
print("Continuing the program...")
```

## עזרה
הריצו `help(str)` ב-console וחפשו מתודות מעניינות. לדוגמה:

```python
# str.center() - centers a string within a given width
text = "hello"
print(text.center(20, "-"))  # -------hello--------

# str.zfill() - pads with zeros from the left
number = "42"
print(number.zfill(6))  # 000042
```
