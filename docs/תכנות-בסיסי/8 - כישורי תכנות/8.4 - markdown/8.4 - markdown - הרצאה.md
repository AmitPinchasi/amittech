## מה זה markdown (md)
- מארקדאון - זה שפה פשוטה שמאפשרת לנו לכתוב טקסט עם כותרות, נקודות, טבלאות בצורה נוחה.
- מתכנתים בדרך כלל משתמשים במארקדאון כדי לתעד את הקוד
- קבצי מארקדאון יכולים להיות מומרים לHTML, pdf מאוד בקלות
- הקורס הזה נכתב במארקדאון.

## עורכי markdown
- תוכנת obsidian - הקורס נכתב בתוכנה הזו, ממליץ מאוד לכולם.
- האתר stackedit - עורך markdown באינטרנט - https://stackedit.io/
- תוכנת notion - תוכנה נהדרת לכתיבת markdown-ים, מאוד מומלץ.

## מארקדאון
- השתמשו באחד התוכנות שציינתי למעלה והתחילו לנסות להשתמש בפיצ'רים של מארקדאון, הנה כמה מהם:
#### כותביות
```markdown
# Heading 1
## Heading 2
### Heading 3
```

#### הדגשות
```markdown
*italic* or _italic_
**bold** or __bold__
```

#### רשימות
```markdown
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2
```

#### רשימות ממוספרות
```markdown
1. Item 1
2. Item 2
   1. Subitem 2.1
   2. Subitem 2.2
```


#### לינקים
```markdown
[Link Text](http://www.example.com)
```

#### תמונות
```
![Alt Text](image_url.jpg)
```

#### קוד
```markdown
```python
def example_function():
    print("Hello, Markdown!")
``
```

#### טבלאות
```markdown
| Header 1 | Header 2 |
| ---------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
```
