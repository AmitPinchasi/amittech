## מחסנית - סוגריים תקינים
- כתבו פונקציה שמקבלת מחרוזת ומחזירה `True` אם הסוגריים בה תקינים, `False` אחרת.
- סוגריים תקינים: `()`, `[]`, `{}`
- **טיפ:** השתמשו במחסנית.
```python
is_valid("()")        # True
is_valid("()[]{}")    # True
is_valid("(]")        # False
is_valid("([)]")      # False
is_valid("{[]}")      # True
```

    - ניתן להדפיס את כל התור.
