## אתגר bandit
בפרויקט הבא תסיימו את כל השלבים של האתגר "bandit" המוכר באתר "overthewire"
- אתגר הbandit הוא אתגר שמורכב מ34 שלבים שנועד לבחון את היכולות שלכם בלינוקס. עשו את כל השלבים אחד אחרי השני!
- באתגר בכל שלב תקבלו רמז של פקודות שאתם צריכים להשתמש בהם, שימו לב לפקודות שכתובות לכם! קראו עליהם עם man ובאינטרנט.
- האתגר עומד להיות קשה! שברו עליו את הראש, זה יחזק אתכם בלינוקס. אם קשה לכם יותר מידי מוזמנים לחפש פתרונות באינטרנט לשלבים.
קראו את המדריך הבא באתר overthewire:
https://overthewire.org/wargames/bandit/

- שימו לב, בשלב מספר 0 תצטרכו להתחבר לשרת הSSH של bandit (איפה שתבצעו את כל האתגר). כתבו את הפקודה 
```bash
ssh bandit0@bandit.labs.overthewire.org -p 2220
```
כדי להתחבר לאתגר, שימו לב שפה התחברנו למשתמש "bandit0", והסיסמא שלו היא "bandit0". אם הצלחתם להתחבר סימן שהתחברתם בהצלחה לשלב 0 של bandit, לקחו את הסיסמא של השלב הראשון (מהתקיית בית שאלייה התחברתם) והמשיכו באתגר. 
- כדי להיכנס לשלב הראשון תצטרכו להתחבר ל"bandit1"- עם הסיסמא שמצאתם. משם בכל שלב תקבלו סיסמא של השלב שאחרי.

התחילו בשלב 0, ועקבו אחרי המדריך באתר:
https://overthewire.org/wargames/bandit/bandit0.html
**בהצלחה.

**אל תמשיכו עם הקורס עד שסיימתם עם כל bandit. אם אתם ממש צריכים תקראו באינטרנט איך להשלים שלבים שלא הצלחתם.
