# צד לקוח - פיתוח פרונטאנד

קורס מקיף לפיתוח צד לקוח - מ-HTML ו-CSS בסיסי, דרך JavaScript ו-TypeScript, ועד React ו-Next.js. הקורס מכסה את כל מה שצריך כדי לבנות אפליקציות ווב מודרניות.

## מה נלמד בקורס

- **הקדמה ו-HTML** - מבנה דף, תגיות, טפסים, סמנטיקה ונגישות
- **CSS** - עיצוב, Flexbox, Grid, רספונסיביות ואנימציות
- **ג'אווהסקריפט בסיסי** - משתנים, פונקציות, מערכים, אובייקטים ולולאות
- **DOM ואירועים** - מניפולציית DOM, טיפול באירועים ועבודה עם הדפדפן
- **ג'אווהסקריפט מתקדם וכלי פיתוח** - Async/Await, מודולים, Bundlers וכלי פיתוח
- **טייפסקריפט** - טיפוסים, ממשקים, Generics ושילוב בפרויקטים
- **ריאקט בסיסי** - קומפוננטות, Props, State, Hooks וניהול טפסים
- **ריאקט מתקדם וניהול סטייט** - Context, Redux, React Query וארכיטקטורה
- **עיצוב בריאקט** - Styled Components, Tailwind CSS, ספריות UI ותימינג
- **Next.js** - SSR, SSG, API Routes, ניתוב ו-Deployment
- **SEO, ביצועים ובדיקות** - אופטימיזציה, Core Web Vitals, בדיקות יחידה ו-E2E
- **נושאים מתקדמים וסיום** - PWA, אנימציות מתקדמות, ופרויקט סיום

## למי מתאים הקורס

הקורס מתאים למי שרוצה להיכנס לעולם פיתוח הווב או למי שכבר יודע backend ורוצה ללמוד frontend. נדרש ידע בסיסי בתכנות.
