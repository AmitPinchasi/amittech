# פריצת אתרים מתקדם

קורס זה הוא המשך ישיר לקורס פריצת אתרים הבסיסי. הקורס מתמקד בטכניקות מתקדמות לתקיפת אפליקציות ווב, שרשור חולשות, ועקיפת מנגנוני הגנה - ברמה של באג באונטי מקצועי ובדיקות חדירות מתקדמות.

## דרישות קדם

- סיום קורס פריצת אתרים הבסיסי
- שליטה בלינוקס ושורת הפקודה
- ידע בסיסי בתכנות (Python, JavaScript, PHP)
- הכרות עם Burp Suite

## מבנה הקורס

### [0 - הקדמה](0%20-%20הקדמה/0.1%20-%20דרישות%20קדם%20ומבנה%20הקורס/0.1%20-%20דרישות%20קדם%20ומבנה%20הקורס%20-%20הרצאה/)

### [1 - סריקות וחקירה מתקדמת](1%20-%20סריקות%20וחקירה%20מתקדמת/1.1%20-%20ניתוח%20JavaScript%20סטטי/1.1%20-%20ניתוח%20JavaScript%20סטטי%20-%20הרצאה/)

### [3 - הזרקות מתקדמות](3%20-%20הזרקות%20מתקדמות/3.1%20-%20הזרקת%20תבניות%20-%20SSTI/3.1%20-%20הזרקת%20תבניות%20-%20SSTI%20-%20הרצאה/)

### [5 - תקיפות פרוטוקול HTTP](5%20-%20תקיפות%20פרוטוקול%20HTTP/5.1%20-%20הברחת%20בקשות%20HTTP%20-%20Request%20Smuggling/5.1%20-%20הברחת%20בקשות%20HTTP%20-%20Request%20Smuggling%20-%20הרצאה/)

### [6 - תנאי מרוץ ולוגיקה עסקית](6%20-%20תנאי%20מרוץ%20ולוגיקה%20עסקית/6.1%20-%20תנאי%20מרוץ%20-%20Race%20Conditions/6.1%20-%20תנאי%20מרוץ%20-%20Race%20Conditions%20-%20הרצאה/)

### [7 - תקיפות צד שרת מתקדמות](7%20-%20תקיפות%20צד%20שרת%20מתקדמות/7.1%20-%20SSRF%20מתקדם/7.1%20-%20SSRF%20מתקדם%20-%20הרצאה/)
SSRF מתקדם, תקיפת ענן דרך SSRF, דסריאליזציה מתקדמת ב-Java/Python/.NET, העלאת קבצים מתקדמת, ו-SSTI ל-RCE.

### [8 - עקיפת WAF](8%20-%20עקיפת%20WAF/8.1%20-%20זיהוי%20והבנת%20WAF/8.1%20-%20זיהוי%20והבנת%20WAF%20-%20הרצאה/)

### [9 - שרשור חולשות ועולם אמיתי](9%20-%20שרשור%20חולשות%20ועולם%20אמיתי/9.1%20-%20שרשור%20חולשות/9.1%20-%20שרשור%20חולשות%20-%20הרצאה/)
