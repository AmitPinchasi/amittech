### דרישות מוקדמות
- תכנות ברמה בסיסית, ללמוד שפה מובנת עצמים כמו פייתון.
- הכרה בסיסית עם שפת שאילתה כמו SQL.
- מומלץ לעשות את "קורס תכנות בסיסי" לפני התחלת קורס זה.
- ממולך לעשות את ״קורס לינוקס בסיסי״ ביחד עם הקורס הזה, כי יש המון חומר חופף.

### למה לעשות את הקורס
- לצלול לתוך למערכת ההפעלה windows ולהכיר את הפיצ'רים שלה.
- להכיר שפות סקריפטים כמו batch, powershell, vbscript ומה הן נותנות לנו
- הכרה של windows היא קריטית לתפקידי IT, ותפקידים בעולם האבטחת מידע כמו בודק חדירות. בנוסף יכולה להוות יתרון גם לתפקידי מו"פ (מחקר ופיתוח) אחרים בתעשייה.

### אג'נדה
- **למידה:** שליש מהזמן אנחנו נלמד חומרים חדשים.
- **תרגול:** שני שליש מהזמן נתרגל את החומרים בתרגולים מעשיים.

## למידה
- **חיפוש באינטרנט:** בסיום כל הרצאה מצופה מכם לקרוא בעצמכם באינטרנט לגבי חומרים שאתם רוצים להרחיב בהם, או חומרים שלא הבנתם עד הסוף כמו שצריך.
- **טיפ:** מומלץ להשתמש בChatGPT לגבי שאלות על חומרים שלא הבנתם עד הסוף.

## תרגול
- **תרגילים:** בסיום רוב ההרצאות מופיע תרגול, מצופה מכם להשלים אותו. אחרי שסיימתם אתם מוזמנים להציץ בפתרון שלי לתרגיל כדי ללמוד.
- **פרויקטים:** בסיום כל נושא בקורס לרוב מופיע פרויקט מסכם, מוצפה מכם להשלים את הפרויקט, לרוב נמצא שם גם פתרון שלי לפרויקט.
- **יוזמות:** תיזמו בקורס - תתרגלו בעצמכם את החומרים הנלמדים וגם הלא נלמדים, ותפתחו קוד בעצמכם כל הזמן כדי לשפר את ההבנה והיכולות שלכם בפיתוח. ככל שתיזום יותר כך תפיק יותר מהקורס, **תיזמו**.
