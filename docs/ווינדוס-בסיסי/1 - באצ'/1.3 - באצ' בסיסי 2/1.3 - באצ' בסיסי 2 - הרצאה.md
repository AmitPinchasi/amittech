## ניוד בין תיקיות וניהול קבצים

### מעבר בין תיקיות

בטרמינל אנחנו תמיד נמצאים בתוך נתיב מסוים (תיקייה במחשב). ניתן לנוע בין תיקיות בעזרת הפקודה `cd` (קיצור של _change directory_ - שינוי תיקייה). בנוסף, אפשר להשתמש בפקודה `dir` כדי לראות את הקבצים שבתוך התיקייה הנוכחית.
- אם אתם צריכים חזרה על זה חזרו לשיעור הראשון הפרק בו אנחנו לומדים על הפקודות הללו.

### יצירת תיקייה

- ניתן ליצור תיקייה חדשה בעזרת הפקודה `mkdir` (_make directory_ - צור תיקייה):
 
 ```cmd
 mkdir NewFolder
 ```
 
 ![צילום מסך: יצירת תיקיות A ו-B מקוננות עם mkdir, מעבר ביניהן עם cd, וחזרה עם cd .. פעמיים](../../../images/Pasted image 20240628131541.png)

### מעבר לתיקייה אחרת

- כדי להיכנס לתיקייה מסוימת, נקליד `cd` ואת שם התיקייה:
 
 ```cmd
 cd NewFolder
 ```
 
 ![צילום מסך: מעבר ישיר לתיקייה מקוננת עם cd C:\Users\amitp\A\B (נתיב מלא)](../../../images/Pasted image 20240628131747.png)
 
- כדי לחזור אחורה תיקייה אחת:
 
 ```cmd
 cd ..
 ```
 
- כדי לחזור אחורה שתי תיקיות:
 
 ```cmd
 cd ../..
 ```
 
 ![צילום מסך: חזרה שתי תיקיות אחורה בבת אחת עם cd ../..](../../../images/Pasted image 20240628132140.png)
 

### מחיקת תיקיות

- ניתן למחוק תיקייה בעזרת `rmdir` (_remove directory_ - מחיקת תיקייה):
 
 ```cmd
 rmdir NewFolder
 ```
 
 ![צילום מסך: מחיקת תיקיות מקוננות עם rmdir - קודם B מבפנים, ואז A מבחוץ אחרי חזרה עם cd ..](../../../images/Pasted image 20240628132407.png)

### יצירה, קריאה ומחיקה של קבצים

- ניתן ליצור קובץ ולכתוב לתוכו נתונים בעזרת `>`:
 
 ```cmd
 whoami > file.txt
 ```
 
 ![צילום מסך: whoami > file יוצר קובץ, ו-type file מציג את תוכנו - amitp\amitp](../../../images/Pasted image 20240628132731.png)
 
 ניתן לקרוא את תוכן הקובץ בעזרת `type`:
 
 ```cmd
 type file.txt
 ```
 
 ![צילום מסך: dir file מציג פרטי הקובץ שנוצר - גודל 13 בתים](../../../images/Pasted image 20240628132840.png)
 
- מחיקת קובץ:
 
 ```cmd
 del file.txt
 ```
 
 ![צילום מסך: del file מוחקת את הקובץ, dir file לאחר מכן מחזירה File Not Found](../../../images/Pasted image 20240628133540.png)
 
- ניתן להוסיף תוכן לקובץ קיים בעזרת `>>`:
 
 ```cmd
 echo "Hello World" >> file.txt
 ```
 
 ![צילום מסך: echo my name is amit >> file.txt מוסיפה שורה לקובץ הקיים, type file.txt מציג את שתי השורות - hello world ו-my name is amit](../../../images/Pasted image 20240628133749.png)
 

### שינוי שם והעתקת קבצים

- שינוי שם קובץ (בעזרת `move`):
 
 ```cmd
 move file.txt file2.txt
 ```
 
 ![צילום מסך: move file.txt file2.txt משנה את שם הקובץ, type file2.txt מציג את התוכן שהועבר](../../../images/Pasted image 20240628134005.png)
 
- העתקת קובץ למיקום אחר (בעזרת `copy`):
 
 ```cmd
 copy file2.txt C:\Users\amitp\Documents
 ```
 
 ![צילום מסך: copy file2.txt file.txt מעתיקה את הקובץ, type file.txt מציג את התוכן שהועתק](../../../images/Pasted image 20240628134154.png)
 

### פתיחת קובץ בעורך טקסט

- ניתן לערוך קובץ בעזרת `notepad`:
 
 ```cmd
 notepad file3.txt
 ```
 
 ![צילום מסך: notepad file3.txt פותחת את הקובץ בעורך Notepad עם התוכן "hello!"](../../../images/Pasted image 20240628134740.png)

### חיפוש תוכן בתוך קובץ

- ניתן לחפש מחרוזת בתוך קובץ בעזרת `find`:
 
 ```cmd
 find "he" file3.txt
 ```
 
 ![צילום מסך: find "he" file3.txt מוצאת ומדפיסה את השורה המכילה "he" - hello!](../../../images/Pasted image 20240628135028.png)

### חיבור פקודות עם Pipe

- `|` (_pipe_ - צינור) מחבר בין פקודות: הפלט של הפקודה הראשונה מועבר לקלט של הפקודה השנייה. לדוגמה, כדי לחפש אם התוכנה `Notepad` רצה:
 
 ```cmd
 tasklist | find "Notepad"
 ```
 
 ![צילום מסך: tasklist | find "Notepad" ללא פלט - Notepad אינו רץ כרגע](../../../images/Pasted image 20240628135802.png)
 
- אם `Notepad` רצה, נקבל פלט: ![צילום מסך: לאחר הרצת notepad, tasklist | find "Notepad" מחזיר שני מופעים רצים של Notepad.exe עם PID ושימוש זיכרון](../../../images/Pasted image 20240628135836.png)
 

### סיכום

- **למדנו כיצד לנוע בין תיקיות** (`cd`, `mkdir`, `rmdir`).
- **יצרנו קבצים, כתבנו לתוכם וקראנו את תוכנם** (`>`, `>>`, `type`, `del`).
- **שינינו שם לקובץ והעתקנו קובץ למיקום אחר** (`move`, `copy`).
- **פתחנו קובץ ב-Notepad**, חיפשנו תוכן בקובץ (`find`).
- **למדנו על שימוש ב-Pipe לחיבור בין פקודות**.
