### תרגיל 1: חקר Event Viewer

פתחו את **Event Viewer** (הקישו `eventvwr` בחיפוש של Windows).

נווטו אל **Windows Logs > System**.

מצאו את 3 האירועים האחרונים מסוג **Error**. לכל אחד כתבו:
- Event ID
- תיאור קצר של השגיאה
- תאריך ושעה

---

### תרגיל 2: יצירת משימה מתוזמנת עם PowerShell

השתמשו ב-PowerShell כדי ליצור משימה מתוזמנת שמפעילה את Notepad כל יום בשעה 9:00 בבוקר:

```powershell
$action = New-ScheduledTaskAction -Execute "notepad.exe"
$trigger = New-ScheduledTaskTrigger -Daily -At 9:00AM
Register-ScheduledTask -Action $action -Trigger $trigger -TaskName "OpenNotepad" -Description "פותח נוטפד בבוקר"
```

לאחר ההרצה, פתחו את **Task Scheduler** וודאו שהמשימה "OpenNotepad" מופיעה ברשימה. לבסוף, מחקו אותה עם:

```powershell
Unregister-ScheduledTask -TaskName "OpenNotepad" -Confirm:$false
```

---

### תרגיל 3: מנהל ההתקנים

פתחו את **Device Manager** (הקישו `devmgmt.msc` בחיפוש).

פרסו את הקטגוריה **Network adapters**. כתבו את שמות כל מתאמי הרשת שמותקנים במחשב שלכם.

---

### תרגיל 4: בדיקת שימוש בזיכרון דרך Task Manager

פתחו את **Task Manager** עם המקשים `Ctrl+Shift+Esc`.

לחצו על העמודה **Memory** כדי למיין לפי שימוש בזיכרון (מהגבוה לנמוך).

כתבו את שמות 3 התהליכים שצורכים הכי הרבה זיכרון, ורשמו את כמות הזיכרון של כל אחד.
