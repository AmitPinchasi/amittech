### תרגיל 1: הצגת כל השירותים הפעילים

השתמשו ב-PowerShell כדי להציג את כל השירותים שכרגע פועלים במחשב:

```powershell
Get-Service | Where-Object { $_.Status -eq "Running" }
```

כמה שירותים פעילים יש? כתבו את שמות 3 שירותים שמצאתם מעניינים.

---

### תרגיל 2: בדיקת מצב שירות Windows Update

השתמשו ב-PowerShell כדי לבדוק את הסטטוס של שירות Windows Update:

```powershell
Get-Service -Name wuauserv
```

האם השירות פועל (Running) או עצור (Stopped)? נסו גם להשתמש ב-`Start-Service` ו-`Stop-Service` כדי לשנות את מצבו ולאחר מכן לבדוק שוב.

---

### תרגיל 3: בדיקת הרשאות ACL

פתחו את **סייר הקבצים** ונווטו לתיקייה `C:\Windows`.

לחצו עליה לחיצה ימנית ובחרו **Properties**. עברו ללשונית **Security**.

מצאו את הקבוצה **Users** ברשימה. כתבו אילו הרשאות יש לה (לדוגמה: Read, Write, Execute).

---

### תרגיל 4: מציאת ה-SID של המשתמש הנוכחי

השתמשו ב-PowerShell כדי לאתר את ה-SID של המשתמש המחובר כעת:

```powershell
Get-WmiObject -Class Win32_UserAccount | Where-Object { $_.Name -eq $env:USERNAME }
```

כתבו מה ה-SID שהתקבל. שימו לב לפורמט: הוא מתחיל ב-`S-1-5-...`.
