### תרגיל 1: חקר הרג'יסטרי עם regedit

פתחו את **regedit** (הקישו `regedit` בחיפוש של Windows).

נווטו לנתיב:

```
HKEY_CURRENT_USER\Control Panel\Desktop
```

מצאו את הערך בשם **Wallpaper**. כתבו מה הנתיב הנוכחי שמופיע שם.

---

### תרגיל 2: קריאת ערך רג'יסטרי עם PowerShell

השתמשו בפקודת **`Get-ItemProperty`** ב-PowerShell כדי לקרוא את נתיב טפט המסך מהרג'יסטרי:

```powershell
Get-ItemProperty -Path "HKCU:\Control Panel\Desktop" -Name Wallpaper
```

האם הערך שהתקבל תואם למה שראיתם ב-regedit בתרגיל הקודם?

---

### תרגיל 3: שינוי ערך רג'יסטרי עם PowerShell

השתמשו בפקודת **`Set-ItemProperty`** כדי לשנות את נתיב הטפט לנתיב אחר (לדוגמה, קובץ תמונה אחר שנמצא על המחשב שלכם):

```powershell
Set-ItemProperty -Path "HKCU:\Control Panel\Desktop" -Name Wallpaper -Value "C:\Windows\Web\Wallpaper\Windows\img0.jpg"
```

לאחר מכן, הריצו שוב את פקודת `Get-ItemProperty` מתרגיל 2 וודאו שהערך השתנה. שימו לב שהשינוי ייכנס לתוקף בהתחברות הבאה או לאחר רענון ידני.
