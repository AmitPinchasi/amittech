### תרגיל 1: הדפסה ולולאה ב-VBScript

צרו קובץ בשם `hello.vbs` עם התוכן הבא:

```vbscript
WScript.Echo "Hello"

Dim i
For i = 1 To 5
    WScript.Echo i
Next
```

הריצו אותו בעזרת הפקודה:

```cmd
cscript hello.vbs
```

מה מודפס למסך?

### תרגיל 2: פונקציה שמחשבת סכום

כתבו קובץ `sum.vbs` שמגדיר פונקציה המקבלת שני מספרים ומדפיסה את סכומם:

```vbscript
Function Add(a, b)
    Add = a + b
End Function

WScript.Echo Add(7, 3)
```

הריצו את הסקריפט עם `cscript sum.vbs`. מה התוצאה? נסו לשנות את הערכים ולהריץ שוב.

### תרגיל 3: שאילתת WMI מתוך VBScript

כתבו קובץ `cpu.vbs` שמשתמש ב-WMI כדי להדפיס את שם המעבד של המחשב:

```vbscript
Dim objWMI, colItems, objItem
Set objWMI = GetObject("winmgmts:\\.\root\cimv2")
Set colItems = objWMI.ExecQuery("SELECT Name FROM Win32_Processor")

For Each objItem In colItems
    WScript.Echo objItem.Name
Next
```

הריצו עם `cscript cpu.vbs`. כתבו מה שם המעבד שהתקבל.
