
## מערכת קבצים
השיעור נלמד כיצד להתעסק עם קבצים ולנווט בין תיקיות.
- הפקודה `pwd` - מדפיסה את התקייה הנוכחית שהShell שלי נמצא בה
![צילום מסך: הרצת pwd בטרמינל, מחזירה /home/ubuntu](../../../images/Pasted image 20250125215106.png)
כלומר, כרגע אנחנו בתקייה ubuntu שנמצאת בתקייה home.
- הפקודה  `cd` - משנה תקייה - משנה את התקייה הנוכחית
```bash
pwd
cd /home
pwd
```
![צילום מסך: cd /home ואז pwd, הפרומפט והנתיב עוברים ל-/home](../../../images/Pasted image 20250125215242.png)
ניתן לראות שעכשיו אנחנו בתקייה home.

```
pwd
cd /home/ubuntu/Downloads
pwd
```
![צילום מסך: cd /home/ubuntu/Downloads/ ואז pwd, הנתיב עובר ל-/home/ubuntu/Downloads](../../../images/Pasted image 20250125215328.png)
הנתיב ".." מחזיר נתיב של תקייה אחורה. כלומר אם נעשה `cd ..` נחזור תקייה אחורה.

![איור עץ מערכת הקבצים בלינוקס שמראה כיצד הפקודה cd נכנסת לתיקייה Downloads וחוזרת אחורה עם cd .. והנתיב המוחלט מתעדכן בהתאם](../../../images/illustrations/linux-1-2-fs-tree.svg)
```
cd ..
pwd
```
![צילום מסך: cd .. חוזר תיקייה אחורה מ-Downloads ל-/home/ubuntu](../../../images/Pasted image 20250125215759.png)
הנתיב "." מחזיר לנו את התקייה הנוכחית. כלומר אם נעשה cd ./some_folder הנתיב שיתקבל הוא הנתיב שאנחנו נמצאים בו כרגע + התקייה some_folder

```
cd ./Downloads
pwd
```
![צילום מסך: cd ./Downloads/ עם נקודה יחידה, ואז pwd מציג /home/ubuntu/Downloads](../../../images/Pasted image 20250125215923.png)
בתחתית עץ התיקיות יש את תקיית השורש, תקיית "root"- אשר נתיב שלה הוא "/"
```
cd /
pwd
```
![צילום מסך: cd / חוזר לתיקיית השורש, pwd מחזירה /](../../../images/Pasted image 20250125220031.png)

כדי לחזור לתקייה הקודמת שהיינו בה, נכתוב את הפקודה `cd -`
```
cd -
pwd
```
![צילום מסך: cd - חוזר לתיקייה הקודמת (Downloads), מדפיס את הנתיב ואז pwd מאשר](../../../images/Pasted image 20250125220146.png)
- בדרך כלל למשתמשים בלינוקס יש תקיית בית, השם של המשתמש שלי הוא "ubuntu". כל תקיות הבית נמצאות ב`home/`, כלומר תקיית הבית של המשתמש שלי תהיה ב`home/ubuntu/`. כדי להיכנס לתקיית הבית של המשתמש שאילו אני מחובר (המשתמש ubuntu במקרה הזה) אפשר לעשות "cd ~"
![צילום מסך: cd ~ מהתיקייה Downloads חוזר ישירות לתיקיית הבית /home/ubuntu](../../../images/Pasted image 20250125220506.png)
בדומה ל ".." ו- "." אפשר להשתמש גם ב "~" כדי לגשת לתקייה מסויימת:
![צילום מסך: cd ~/Downloads/ מהתיקייה השורש קופץ ישירות לתיקיית Downloads בבית המשתמש](../../../images/Pasted image 20250125220724.png)
ובדומה לכך גם:
![צילום מסך: cd ../Desktop/ עובר מ-Downloads לתיקיית Desktop (אח באותה רמה)](../../../images/Pasted image 20250125220747.png)
- הפקודה `ls` - מציגה את כל הקבצים בתקייה הנוכחית, ניתן לראות למשל את כל התיקיות והקבצים בתיקיית היוזר שלנו כך:
![צילום מסך: ls בתיקיית הבית מציג תיקיות Desktop, Documents, Downloads, Music, Pictures, Public, snap, Templates, Videos](../../../images/Pasted image 20250125220832.png)
לא חייב להיות בתקייה מסויימת כדי לראות את הקבצים שלה, אפשר גם לכתוב את הפקודה `ls` ולהביא לה נתיב מסויים:
![צילום מסך: ls / מציג את תוכן תיקיית השורש - bin, boot, dev, home, lib64, mnt, proc, root, sbin, tmp, usr, var ועוד](../../../images/Pasted image 20250125220944.png)
![צילום מסך: ls /home מציג משתמש יחיד - ubuntu](../../../images/Pasted image 20250125221018.png)
![צילום מסך: ls /home/ubuntu/Downloads/ ללא פלט - התיקייה ריקה](../../../images/Pasted image 20250125221027.png)
ניתן לראות שאין לי קבצים בתקיית הDownloads
כמובן שניתן להשתמש ב".", "..", "~" גם עם ls כמו כל פקודה.
![צילום מסך: ls ./Downloads/ ללא פלט - התיקייה ריקה](../../../images/Pasted image 20250125221203.png)

- הפקודה `echo "hello" > my_file` - תיצור קובץ חדש בשם `my_file` עם תוכן: "hello"
![צילום מסך: echo "hello" > my_file יוצר קובץ חדש, ls מציג אותו ברשימת הקבצים](../../../images/Pasted image 20250125221245.png)

הפקודה - `cat` - מחבר (concatenate) - מדפיסה את תוכן הקבצים למסך
![צילום מסך: cat my_file מדפיס את תוכן הקובץ - hello](../../../images/Pasted image 20250125221319.png)
![צילום מסך: echo "world" > my_file2 יוצר קובץ שני, cat my_file2 מציג world](../../../images/Pasted image 20250125221400.png)
ניתן להשתמש בcat גם כדי להדפיס כמה קבצים ביחד:
![צילום מסך: cat my_file my_file2 מדפיס את שני הקבצים ברצף - hello ואז world](../../../images/Pasted image 20250125221435.png)
- נפרק את הפקודה  `echo "hello" > my_file` - החלק הראשון הוא `echo "hello"` שאחראי להדפיס למסך "hello", לאחר מכן יש `> my_file` שאחראי להעביר את כל הפלט של הפקודה הקודמת לתוך הקובץ my_file.
- אפשר להשתמש באותו עקרון עם הפקודה cat, שמדפיסה תוכן של קובץ למסך.
![צילום מסך: cat my_file my_file2 > ./the_final_file משרשר את שני הקבצים לקובץ חדש, cat the_final_file מציג hello ו-world](../../../images/Pasted image 20250125221734.png)
- אם נרצה להוסיף תוכן חדש אחרי תוכן קיים בקובץ (כלומר append) נשתמש באופרטור "<<":
![צילום מסך: echo "helloooooooo" >> ./my_file מוסיף שורה לקובץ הקיים, cat my_file מציג hello ואז helloooooooo](../../../images/Pasted image 20250125221937.png)
- אם נרצה להשתמש באותיות מיוחדות, כמו `\n` כדי לרדת שורה נשתמש בדגל "-e" עם echo:
  ![צילום מסך: echo -e עם 
 יוצר קובץ my_file עם כל מילה בשורה נפרדת - hello my name is amit pinchasi and i record courses](../../../images/Pasted image 20250125222732.png)

- הפקודה `head -n 2` - מציג את שתי השורות הראשונות של הקובץ - מציג את עשר השורות הראשונות אם לא מציינים הדגל `-n`.
![צילום מסך: head -n 2 ./my_file מציג רק שתי השורות הראשונות של הקובץ - hello ו-my](../../../images/Pasted image 20250125222759.png)

- הפקודה `tail -n 2` - מציג את שתי השורות האחרונות של הקובץ - מציג את עשר השורות האחרונות אם לא מציינים דגל
  ![צילום מסך: tail -n 2 ./my_file מציג רק שתי השורות האחרונות של הקובץ - record ו-courses](../../../images/Pasted image 20250125222820.png)
נוריד את התמלול של "אליס בארץ הפלאות" בפקודה הבאה:
![צילום מסך: wget מוריד את הקובץ alice_in_the_wonderland.txt מאתר gutenberg.org עם פס התקדמות שמראה 100% והורדה מוצלחת](../../../images/Pasted image 20250125231108.png)
נעשה head ו- tail לקובץ הארוך שקיבלנו.
![צילום מסך: head alice_in_the_wonderland.txt מציג את תחילת הקובץ, tail מציג את סופו](../../../images/Pasted image 20250125231151.png)

- הפקודה `less` - מציג קבצים ארוכים בטרמינל
```bash
less ./alice_in_the_wondeland.txt
```
![צילום מסך: less מציג את תוכן קובץ alice_in_the_wonderland.txt עם אפשרות גלילה - כותרת, מחבר Lewis Carroll ותחילת הספר](../../../images/Pasted image 20250125231225.png)
עם החצים נוכל לרדת ולעלות, וכשנרצה לצאת נלחץ על q. אם נלחץ על / נוכל ממש לחפש תוכן.
![צילום מסך: חיפוש עם / בתוך less מסמן בצהוב את כל המופעים של המילה cat בטקסט הספר](../../../images/Pasted image 20250125231334.png)
הנה התוצאה שקיבלתי שלחצתי על / בשביל חיפוש וכתבתי "cat" ואנטר.

- הפקודה `rm` - מוחקת קובץ 
```bash
rm ./my_file
ls 
```
![צילום מסך: ls מציג את הקובץ alice_in_the_wonderland.txt ברשימת הקבצים, rm מוחק אותו, ls נוסף מראה שהוא נעלם](../../../images/Pasted image 20250125231435.png)

- הפקודה `mkdir` - יוצרת תקייה חדשה
- הפקודה `rmdir` - מוחקת תקייה
![צילום מסך: mkdir hello יוצר תיקייה, rmdir hello/ מוחק אותה, ואז cd hello נכשל עם השגיאה No such file or directory](../../../images/Pasted image 20250125231517.png)

- הדגל `ls -l` - מציג את כל הקבצים אבל עם יותר פרטים
![צילום מסך: ls -l מציג רשימה מפורטת של קבצים ותיקיות עם הרשאות, בעלים, גודל ותאריך שינוי אחרון](../../../images/Pasted image 20250125231600.png)


- הפקודה `touch` - משנה את התאריך של קובץ מסויים שמופיע בls -l, או יוצרת קובץ אם הקובץ לא קיים:
  ![צילום מסך: ls -l my_file מציג תאריך ישן, touch my_file מעדכן את זמן השינוי, ls -l מציג תאריך ושעה מעודכנים](../../../images/Pasted image 20250125231756.png)
  ניתן לראות שהזמן שהשתנה.
  ![צילום מסך: touch my_file3 יוצר קובץ חדש וריק, ls -l מציג אותו ברשימה עם גודל 0 ותאריך יצירה נוכחי](../../../images/Pasted image 20250125231829.png)

- הפקודה `mv` - מעבירה קובץ ממיקום אחד למיקום אחר
ניצור קובץ חדש:
![צילום מסך: ls -l מציג את התיקייה ללא my_file, ואז echo "hello" > ./my_file יוצר אותו מחדש](../../../images/Pasted image 20250125232116.png)
נעביר את הקובץ מ ./my_file ל- ./my_file2
![צילום מסך: mv ./my_file ./my_file2 משנה שם קובץ, ls -l מציג ש-my_file נעלם ובמקומו my_file2](../../../images/Pasted image 20250125232218.png)
ניתו לחשוב על mv כמו שינוי שם של קובץ.

- הפקודה `cp` - מעתיקה קובץ ממיקום אחד למיקום אחר
![צילום מסך: cp ./my_file2 ./my_file3 מעתיק קובץ, ls -l מציג ששני הקבצים my_file2 ו-my_file3 קיימים במקביל](../../../images/Pasted image 20250125232254.png)

- הפקודה `nano` - עורך טקסט
```bash
nano ./my_file
```
יפתח לכם עורך טקסט:
![צילום מסך: עורך הטקסט nano נפתח על קובץ my_file חדש וריק, עם שורת התפריט התחתונה של פקודות nano](../../../images/Pasted image 20250125232502.png)
![צילום מסך: nano עם תוכן טקסט לדוגמה שהוקלד בקובץ my_file, כוכבית בכותרת מסמנת שינויים שלא נשמרו](../../../images/Pasted image 20250125233515.png)
אחרי שנסיים להקליד את התוכן שלנו נוכל לשמור אותו עם מקש "ctrl + o" ולחיצה על אנטר
![צילום מסך: nano מציג את שורת השמירה File Name to Write עם הנתיב ./my_file לפני אישור השמירה](../../../images/Pasted image 20250125233548.png)
כדי לצאת העורך טקסט נלחץ על "ctrl + x"

![צילום מסך: find /home -name my_file מאתר ומציג את הנתיב המלא לקובץ - /home/ubuntu/my_file](../../../images/Pasted image 20250125234815.png)
יש המון דגלים לfind, מוזמנים לעשות man או --help
