### תרגיל 1: ניהול processes ושימוש ב-`jobs` ו-`bg`

1. הריצו את הפקודה הבאה:

```bash
sleep 30
```

1. עצרו את הprocess על ידי לחיצה על `Ctrl + Z`.
2. הריצו את הפקודה `jobs` כדי לראות את הג'וב המושהה.
3. החזירו את הprocess לפעולה ברקע בעזרת הפקודה `bg %1`.
4. הריצו שוב את הפקודה `jobs` וודאו שהprocess רץ ברקע.
2. עכשיו, הריצו את הפקודה הבאה:

```bash
sleep 600
```

1. עצרו את הprocess עם `Ctrl + Z` שנית.
2. השתמשו בפקודה `fg %2` כדי להחזיר את הprocess לפעולה בתור הג'וב הראשי.
3. בודדו את מצב הprocess עם הפקודה `ps aux`.

### תרגיל 2: שליחת אותות לprocesses

1. הריצו את הפקודה הבאה:

```bash
sleep 600
```

1. הריצו `ps aux` וכתבו את מזהה הprocess (PID) של process ה-`sleep`.
2. השתמשו בפקודה `kill -15 <PID>` (שלחו אות SIGTERM) כדי לסיים את הprocess באופן "רך".
3. הריצו שוב את הפקודה `ps aux` וודאו שהprocess לא קיים יותר.
2. הריצו שוב את הפקודה:

```bash
sleep 600
```

1. עצרו את הprocess עם `Ctrl + Z`.
2. שלחו לprocess אות SIGKILL על ידי הפקודה:

```bash
kill -9 <PID>
```

3. הריצו שוב את `ps aux` וודאו שהprocess לא קיים יותר.
