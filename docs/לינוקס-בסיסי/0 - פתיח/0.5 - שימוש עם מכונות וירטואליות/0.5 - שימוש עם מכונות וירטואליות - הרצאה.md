## תפריט המכונה הוירטואלית
- שתלחצו מקש ימני על הכותרת של מכונה וירטואלית תפתח בפניכם התפריט הבא:
  ![[Pasted image 20250125193234.png]]
  - כאשר הגדרות הpower, אחראיות על לכבות, הלדליק או לרסט את המכונה
  - הגדרות ה removeable device תראו את כל המכשירים שמחוברים למחשב שלכם וגם לubuntu.
  - כפתור הpause עושה pause למצב הנוכחי של המכונה.
  - תפריט הsnapshots מאפשר לנו לשמור מצב נוכחי של מכונה, אנחנו יכולים לעשות snapshot-ים שונים במצבים שונים של המכונה ואז ממש לחזור אליהם. דמיינו שאנחנו יכולים ליצור ציר זמן למכונה שלנו ובכך "לחזור אחורה בזמן". זוהי הפיצ'ר הכי חזק של מכונות וירטואליות. דמיינו את כמות הדברים שהפיצ'ר הזה מאפשר לנו!
  - תמיד מומלץ לקחת snapshot אחרי שמתקנים מכונה, כדי שאם מתישהו בטעות שברנו אותה, תמיד נוכל לחזור לsnapshot.
  - בהגדרה האחרונה נראה את דף הגדרות של המכונה. פתחו אותו:
  ![[Pasted image 20250125193610.png]]
  פה ממש נוכל לראות את החומרה הוירטואלית שאנחנו מביאים למחשב הוירטואלי שלנו. נוכל לשנות את כמות הזכרון RAM שאנחנו מביאים לו, הכמות ליבות למעבד, את הסוג חיבוא תקשורתי ועוד. אם נלך לכפתור הoptions נוכל אפילו להוסיף "shared folder" בין המחשב שלנו למכונה, זהו פיצ'ר ממש חשוב!
  - שחקו עם ההגדרות!
