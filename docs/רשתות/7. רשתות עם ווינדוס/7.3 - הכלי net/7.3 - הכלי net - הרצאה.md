### פקודת `net` ב-Windows: ניהול רשת, משתמשים ושירותים

הפקודה `net` ב-Windows היא כלי ניהול עוצמתי שמספק למנהלי רשתות ואנשי IT אפשרות לבצע מגוון פעולות עם מחשבים ברשת, שירותים, משתמשים, קבוצות ומשאבים. הפקודה נועדה לשפר את ניהול המערכת על ידי אפשרויות לשתף קבצים, ליצור משתמשים, לנהל קבוצות משתמשים ולבצע פעולות רשת נוספות, הן במחשב המקומי והן בסביבה דומיינית (Active Directory).

בהרצאה זו נסקור את השימושים השונים של הפקודה `net`, נציג דוגמאות לכל פקודה ונראה כיצד ניתן להשתמש בה גם במחשב המקומי וגם במחשב מחובר לדומיין.

---

### מבנה כללי של הפקודה `net`

הפקודה `net` מציעה מגוון של אפשרויות (או פקודות משנה), וכל אחת מהם מבצעת פעולה מסוימת. כל פקודה כתובה בצורה הבאה:

```
net [פקודה] [אפשרויות]
```

לדוגמה, אם נרצה להציג את כל המשתמשים במחשב המקומי, נשתמש בפקודה `net user`. אם נרצה להוסיף משתמש חדש, נוכל להשתמש בפקודה `net user [שם משתמש] [סיסמה] /add`.

### שימושים עיקריים של הפקודה `net`

#### 1. ניהול משתמשים עם `net user`

הפקודה `net user` מאפשרת לנהל חשבונות משתמשים במחשב המקומי או בדומיין. אפשרות זו מועילה במיוחד כאשר יש צורך להוסיף, למחוק או לשנות את פרטי המשתמשים.

- **הצגת מידע על משתמשים**  
להציג את כל המשתמשים במחשב המקומי, נשתמש בפקודה:

```
net user
```

- **יצירת משתמש חדש**  
ניתן ליצור משתמש חדש במחשב המקומי בעזרת הפקודה:

```
net user [שם משתמש] [סיסמה] /add
```

לדוגמה, יצירת משתמש בשם "johndoe" עם סיסמה "password123":

```
net user johndoe password123 /add
```

- **שינוי סיסמה למשתמש**  
אם ברצונך לשנות את הסיסמה של משתמש, השתמש בפקודה:

```
net user [שם משתמש] [סיסמה חדשה]
```

לדוגמה, שינוי הסיסמה למשתמש "johndoe":

```
net user johndoe newpassword456
```

- **הסרת משתמש**  
על מנת למחוק משתמש, נשתמש בפקודה:

```
net user [שם משתמש] /delete
```

לדוגמה, מחיקת המשתמש "johndoe":

```
net user johndoe /delete
```


#### 2. ניהול קבוצות עם `net localgroup`

הפקודה `net localgroup` מאפשרת לנהל קבוצות משתמשים, להוסיף ולהסיר חברים מקבוצות, וליצור קבוצות חדשות.

- **יצירת קבוצה חדשה**  
ליצור קבוצה חדשה במחשב המקומי:

```
net localgroup [שם קבוצה] /add
```

לדוגמה, יצירת קבוצה בשם "admins":

```
net localgroup admins /add
```

- **הוספת משתמש לקבוצה**  
כדי להוסיף משתמש לקבוצה, השתמש בפקודה:

```
net localgroup [שם קבוצה] [שם משתמש] /add
```

לדוגמה, הוספת המשתמש "johndoe" לקבוצת "admins":

```
net localgroup admins johndoe /add
```

- **הסרת משתמש מקבוצה**  
להסיר משתמש מקבוצה:

```
net localgroup [שם קבוצה] [שם משתמש] /delete
```

לדוגמה, להסיר את המשתמש "johndoe" מקבוצת "admins":

```
net localgroup admins johndoe /delete
```


#### 3. ניהול שיתופים עם `net share`

הפקודה `net share` מאפשרת לשתף תיקיות ודרכים נוספות בהן נוכל לשתף קבצים בין מחשבים. זהו כלי חיוני במיוחד במערכות רשת.

- **הצגת שיתופים פעילים**  
להציג את כל השיתופים הקיימים במחשב המקומי:

```
net share
```

- **יצירת שיתוף תיקייה**  
ליצור שיתוף תיקייה במחשב המקומי:

```
net share [שם שיתוף]=[נתיב תיקיה]
```

לדוגמה, לשתף את התיקיה "C:\Documents" תחת השם "DocsShare":

```
net share DocsShare=C:\Documents
```

- **הסרת שיתוף תיקייה**  
להסיר שיתוף תיקייה:

```
net share [שם שיתוף] /delete
```

לדוגמה, להסיר את השיתוף "DocsShare":

```
net share DocsShare /delete
```


#### 4. חיבור לכוננים רשתיים עם `net use`

הפקודה `net use` מאפשרת לחבר כוננים רשתיים ומדפסות מרוחקות. אפשרות זו מועילה כשצריך לגשת לכוננים או משאבים הנמצאים על מחשבים אחרים ברשת.

- **התחברות לכונן רשת**  
להתחבר לכונן רשת, השתמש בפקודה:

```
net use [כונן:] \\[שרת]\[שיתוף]
```

לדוגמה, התחברות לכונן R בשרת בשם "server1" לשיתוף בשם "docs":

```
net use R: \\server1\docs
```

- **התחברות עם שם משתמש וסיסמה**  
התחברות עם שם משתמש וסיסמה במקרים בהם דרושים פרטי גישה:

```
net use [כונן:] \\[שרת]\[שיתוף] [סיסמה] /user:[שם משתמש]
```

לדוגמה, התחברות לשיתוף "docs" בשרת "server1" עם שם משתמש "admin" וסיסמה "password123":

```
net use R: \\server1\docs password123 /user:admin
```

- **הסרת חיבור רשת**  
לנתק חיבור לכונן רשת:

```
net use [כונן:] /delete
```

לדוגמה, להסיר את החיבור לכונן R:

```
net use R: /delete
```


### שימוש בפקודת `net` ברשת דומיינית

במערכות דומיין (Active Directory), ניתן להשתמש בפקודת `net` על מנת לנהל חיבורים ושירותים ברשת הדומיין. במיוחד, ניתן להשתמש בפקודות `net` כדי לבצע פעולות כמו הצגת סטטוס של מחשבים, הצטרפות לדומיין, שינוי סיסמאות של משתמשים ודומיינים ועוד. הנה איך אפשר להשתמש בפקודת `net` ברשת דומיינית:

#### 1. הצטרפות למחשב לדומיין עם `net`

הפקודה `net` מאפשרת להצטרף לדומיין, דבר שמועיל במיוחד כאשר רוצים להוסיף מחשבים חדשים לדומיין או לשנות את המחשב לדומיין חדש.

- **הצטרפות לדומיין**  
אם נרצה להצטרף למחשב לדומיין, נשתמש בפקודה הבאה:

```
netdom join [שם מחשב] /domain:[שם דומיין] /userd:[שם משתמש] /passwordd:[סיסמה]
```

לדוגמה, הצטרפות למחשב בשם "workstation1" לדומיין "example.com" עם שם משתמש "admin" והסיסמה "password123":

```
netdom join workstation1 /domain:example.com /userd:admin /passwordd:password123
```


#### 2. ניהול משתמשים ברשת דומיינית

במערכת דומיין, הפקודה `net user` מאפשרת לבצע פעולות על חשבונות משתמשים ברשת.

- **הצגת כל המשתמשים בדומיין**  
אם ברצונך להציג את כל המשתמשים המנוהלים על ידי דומיין, תוכל להשתמש בפקודה:

```
net user /domain
```

- **יצירת משתמש חדש בדומיין**  
על מנת ליצור משתמש חדש בתוך הדומיין, השתמש בפקודה:

```
net user [שם משתמש] [סיסמה] /add /domain
```

לדוגמה, יצירת משתמש חדש בשם "johndoe" עם סיסמה "password123" בדומיין:

```
net user johndoe password123 /add /domain
```

- **שינוי סיסמאות בדומיין**  
אם ברצונך לשנות את הסיסמה של משתמש בדומיין, השתמש בפקודה:

```
net user [שם משתמש] [סיסמה חדשה] /domain
```

לדוגמה, שינוי סיסמת המשתמש "johndoe" לדומיין ל-"newpassword456":

```
net user johndoe newpassword456 /domain
```


#### 3. שיתוף תיקיות ברשת דומיינית

כאשר מדובר ברשת דומיינית, אפשר להשתמש בפקודה `net share` כדי לשתף תיקיות על מחשבים שונים בדומיין. זה שימושי במקרים של שיתוף משאבים בין מחשבים מרובים תחת הדומיין.

- **שיתוף תיקייה בדומיין**  
אם נרצה לשתף תיקייה בשם "Documents" שתהיה נגישה לכל מחשב בדומיין, נוכל להשתמש בפקודה הבאה:

```
net share Documents=C:\Documents /domain
```


#### 4. התחברות לכוננים רשתיים מתוך הדומיין עם `net use`

בהנחה שמחשב נוסף בדומיין משתף תיקייה, אפשר להתחבר אליה ולגשת אליה דרך פקודת `net use`. ניתן להתחבר עם שם המשתמש והסיסמה של הדומיין.

- **התחברות לכונן רשת מתוך הדומיין**  
על מנת להתחבר לשיתוף תיקייה במחשב אחר בדומיין, השתמש בפקודה:

```
net use [כונן:] \\[שם מחשב]\[שיתוף] /user:[שם דומיין]\[שם משתמש] [סיסמה]
```

לדוגמה, התחברות לשיתוף בשם "share" על מחשב בשם "server1" תחת הדומיין "example.com" עם שם המשתמש "johndoe" וסיסמה "password123":

```
net use R: \\server1\share /user:example\johndoe password123
```


#### 5. הצגת מצב החיבורים והסשנים בדומיין

אם ברצונך לבדוק את מצב החיבורים והסשנים ברשת הדומיינית, תוכל להשתמש בפקודת `net session`.

- **הצגת כל החיבורים לדומיין**  
להציג את כל החיבורים ופרטי המשתמשים המחוברים לשרת בדומיין, השתמש בפקודה:

```
net session
```


הפקודה `net` ב-Windows מספקת מגוון רחב של פעולות לניהול מחשבים ברשת, חיבורים רשתיים, משתמשים, קבוצות ושירותים. היא מאפשרת ביצוע פעולות כמו יצירת משתמשים, ניהול קבוצות, שיתוף תיקיות, התחברות לכוננים רשתיים, ועוד. כמו כן, בגרסאות דומיין של Windows (Active Directory), הפקודה מציעה כלים נוספים לניהול מחשבים וחשבונות משתמשים ברשת דומיינית.

הפקודה `net` היא כלי חיוני ומועיל שמסייע לניהול מערכות Windows ביעילות, הן ברשתות המקומיות והן ברשתות דומייניות.
