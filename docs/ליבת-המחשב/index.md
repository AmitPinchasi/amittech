# קורס ליבת המחשב

קורס מעמיק שלוקח אתכם למסע לתוך הליבה של המחשב. מארגון המחשב וייצוג בינארי, דרך תכנות באסמבלי 16 ו-32 ביט, שפת C, ועד בניית רכיבים של מערכת הפעלה. קורס שנותן הבנה אמיתית של מה שקורה מתחת למכסה המנוע.

## מה נלמד בקורס

- **ארגון המחשב** - איך המחשב בנוי, ייצוג מידע בינארי, טבלאות אמת ולוגיקה דיגיטלית
- **אסמבלי 16 ביט** - תכנות ישירות עם המעבד, רגיסטרים, מחסנית, מחרוזות, פסיקות וקלט/פלט
- **אסמבלי 32 ביט** - מעבר ל-Protected Mode, טבלת GDT, מנגנון ה-Paging, ה-Context Switch, וטיפול ב-Page Fault
- **שפת C** - הבסיס, פוינטרים, בקרת זרימה, מודולריות, סטראקטים, ה-Preprocessor, סקופ ודיבוג עם GDB
- **ספריית libc** - מחרוזות, קבצים, זיכרון דינמי, מספרים, אלגוריתמים וטיפול בשגיאות
- **לינוקס** - תהליכים, ה-Loader, ואיך מערכת ההפעלה מנהלת הכל מאחורי הקלעים

## פרויקטים

- **פרויקט אסמבלי 16 ביט** - כתיבת תוכנה שלמה באסמבלי, עבודה ישירה עם חומרה
- **פרויקט אסמבלי 32 ביט** - בניית רכיבי מערכת הפעלה: paging, context switch, וטיפול בפסיקות
- **פרויקט libc** - פרויקט מסכם בשפת C שמשלב את כל הנושאים שנלמדו

## למי מתאים הקורס

הקורס מתאים למי שרוצה להבין באמת איך מחשב עובד. מתאים למפתחים שרוצים להתמקצע בתכנות low-level, אבטחת מידע, פיתוח מערכות הפעלה, מערכות משובצות, או הנדסה לאחור. נדרש ידע בסיסי בתכנות.

## למה הקורס הזה

- **עומק אמיתי** - מייצוג בינארי ועד בניית רכיבי מערכת הפעלה, כולל הבנה של כל שכבה
- **תכנות באסמבלי** - כולל 16 ביט ו-32 ביט, עם הבנה מעמיקה של Protected Mode ו-Paging
- **שפת C מקצועית** - פוינטרים, ניהול זיכרון ידני, דיבוג עם GDB
- **פרקטי ומעשי** - כל מושג מלווה בקוד עובד ופרויקטים שמחברים הכל יחד
- **בסיס לקריירה** - הידע הזה הוא יתרון משמעותי בתפקידי סייבר, embedded, פיתוח מערכות הפעלה ו-reverse engineering, תחומים עם ביקוש גבוה ושכר גבוה
- **90+ שיעורים** שבנויים בצורה מדורגת עם תרגילים ופתרונות
