### תרגיל 2 - השפעת nice

1. כתבו תוכנית בC שרצה בלולאה אינסופית ומדפיסה כל שניה כמה איטרציות ביצעה:
```c
#include <stdio.h>
#include <time.h>

int main() {
    long count = 0;
    time_t start = time(NULL);
    while (1) {
        count++;
        if (time(NULL) - start >= 1) {
            printf("iterations: %ld\n", count);
            count = 0;
            start = time(NULL);
        }
    }
    return 0;
}
```
2. הריצו שני עותקים של התוכנית - אחד עם nice=0 (ברירת מחדל) ואחד עם nice=19:
```bash
./counter &
nice -n 19 ./counter &
```
3. השוו את מספר האיטרציות בשניה. כמה יותר איטרציות התוכנית עם nice=0 מבצעת?
4. עצרו את שני הprocesses (`kill %1 %2`).

### תרגיל 3 - מידע מproc

לכל process שרץ, חקרו את המידע שהקרנל חושף דרך proc:

1. הריצו `sleep 300 &` כדי ליצור process שישן 5 דקות. רשמו את הPID שלו.
2. בדקו את המצב שלו: `cat /proc/<PID>/status | grep State`. מה המצב?
3. בדקו את מידע התזמון: `cat /proc/<PID>/sched`. מצאו את הvruntime. האם הוא משתנה אם תבדקו שוב אחרי כמה שניות? למה?
4. כמה context switch-ים (voluntary ו-involuntary) היו לprocess? (חפשו nr_switches בproc/PID/sched, או nr_voluntary_switches ו-nr_involuntary_switches בproc/PID/status)
5. שלחו לprocess SIGSTOP (`kill -STOP <PID>`) ובדקו שוב את המצב. מה השתנה?
6. שלחו לו SIGCONT (`kill -CONT <PID>`) כדי להמשיך אותו, ואז `kill <PID>` כדי לסיים.
