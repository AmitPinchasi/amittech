הוסיפו route-ים למערכת שלכם של אוטנטקציה (post /login + post /register), אחרי login תחתמו jwt token למשתמשים.
הוסיפו route-ים שרק משתמשים מאומתים יכולים לגשת אליהם, או קחו את חלק מהroute-ים שלכם ודאגו שרק משתמשים מאומתים יכולים לגשת אליהם.

בפרויקט שלי אני מוסיף route-ים של login + register.
בנוסף אני דואג שרק משתמשים רשומים יוכלו להוסיף טיסות, אטרצקציות, מלונות וטיולים.
