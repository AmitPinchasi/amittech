# קורס צד שרת

קורס שמלמד לבנות שרתים ומערכות backend מאפס. מתחילים מהבנת פרוטוקול HTTP ומה זה API, ממשיכים לפיתוח עם FastAPI ומסדי נתונים, עוברים ל-DevOps וענן, ומסיימים עם פיתוח JavaScript ו-Node.js.

## מה נלמד בקורס

- **הקדמה** - רשתות בסיסי, פרוטוקול HTTP, מה זה API, ותכנון הפרויקט שילווה אתכם לאורך הקורס
- **פיתוח API** - בניית שרת עם FastAPI ו-Postman, אותנטיקציה עם JWT, מסדי נתונים רלציונים עם SQLAlchemy, מסדי נתונים לא-רלציונים, Redis, תורי הודעות עם Kafka, וסוכני AI עם LangChain
- דבאופס **DevOps על רגל אחת** - ענן AWS, Infrastructure as Code, תהליכי CI/CD, Elasticsearch, ו-Apache
- **פיתוח JavaScript** - הבסיס של JavaScript, פיתוח שרת עם Node.js, ובניית API עם Express

## הפרויקט

הקורס בנוי סביב פרויקט מרכזי שמתפתח לאורך כל הדרך. התלמידים בוחרים רעיון לשרת API (או משתמשים בתבנית "בונה טיולים") ומרחיבים אותו בכל שיעור - מוסיפים אותנטיקציה, מסדי נתונים, caching, תורי הודעות, ועוד. בסוף הקורס יש פרויקט שלם שמרשים ומציג ידע backend אמיתי.

## למי מתאים הקורס

הקורס מתאים למי שכבר יודע לתכנת ורוצה להיכנס לעולם הפיתוח backend. מתאים למי שרוצה להפוך למפתח fullstack, או להתמקצע בפיתוח שרתים ותשתיות.

## למה הקורס הזה

- **פרויקט מלווה** - לא לומדים תיאוריה יבשה, בונים פרויקט אמיתי שגדל עם כל שיעור
- **שתי שפות** - פייתון עם FastAPI וגם JavaScript עם Node.js ו-Express
- **מגוון מסדי נתונים** - SQL, NoSQL, Redis, וארכיטקטורת תורי הודעות עם Kafka
- דבאופס **DevOps מעשי** - לא רק פיתוח, אלא גם ענן, CI/CD, וניטור
- **סוכני AI** - שילוב של LangChain ו-AI agents בשרתים - נושא חם ורלוונטי
- **מהקוד לענן** - הקורס מלמד את כל הדרך, מכתיבת שורת קוד ראשונה ועד deployment בענן
