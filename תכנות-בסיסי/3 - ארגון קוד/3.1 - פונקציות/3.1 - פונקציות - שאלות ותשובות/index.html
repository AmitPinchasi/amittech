    3.1 - ОцОЋОаОДОдОЎОЋОф - ОЕОљОюОЋОф ОЋОфОЕОЋОЉОЋОф

    ОљОдОЎОБ ОцОћ ОЕОљОюОЋОф ОаОцОЋОдОЋОф ОбОю ОцОЋОаОДОдОЎОЋОф ОъОљОаОЕОЎОЮ ОЕОюОЎОъОЊОфОЎ:

    • ОцОЋОаОДОдОЎОЋОф ОЌОЎОЎОЉОЋОф ОюОДОЉОю ОцОеОъОўОеОЎОЮ?
      ОюОљ

    • ОцОЋОаОДОдОЎОЋОф ОЌОЎОЎОЉОЋОф ОюОћОЌОќОЎОе ОбОеОџ? (ОюОбОЕОЋОф return ОюОъОЕОћОЋ)
      ОюОљ, ОцОЋОаОДОдОЎОЋОф ОЎОЏОЋОюОЋОф ОюОбОЕОЋОф return ОюОЏОюОЋОЮ, ОЋОќОћ ОљОЋОъОе ОЕОћОцОЋОаОДОдОЎОћ ОцОЕОЋОў ОюОљ ОфОЌОќОЎОе ОЕОЋОЮ ОбОеОџ.

    • ОЏОфОЉОфОЎ ОцОЋОаОДОдОЎОћ ОЕОюОљ ОъОЌОќОЎОеОћ ОбОеОџ, ОЌОЎОЎОЉ ОюОбОЕОЋОф return?
      ОюОљ, ОюОљ ОЌОЎОЎОЉ ОюОбОЕОЋОф return, ОљОцОЕОе ОцОЕОЋОў ОюОЏОфОЋОЉ ОцОЋОаОДОдОЎОћ ОЉОюОЎ return. ОќОћ ОЎОћОЎОћ ОЉОЊОЎОЋОД ОљОЋОфОЋ ОћОЊОЉОе ОЏОъОЋ ОюОЏОфОЋОЉ ОцОЋОаОДОдОЎОћ ОЕОбОЋОЕОћ return ОюОЏОюОЋОЮ.

    • ОЕОљОаОЎ ОЏОЋОфОЉ ОфОЋОЏОаОћ ОбОЮ ОцОЋОаОДОдОЎОЋОф, ОДОдОф ОДОЕОћ ОюОЎ ОюОбОДОЋОЉ ОљОЌОеОЎ ОъОћ ОЕОъОЌОЕОЉ ОбОЋОЕОћ, ОъОћ ОљОаОЎ ОЎОЏОЋОю ОюОбОЕОЋОф?
      ОаОАОћ ОюОбОДОЋОЉ ОбОЮ ОћОљОдОЉОб ОЕОюОџ, ОъОъОЕ ОЊОъОЎОЎОЪ ОЕОљОфОћ ОъОфОЌОЎОю ОЉОЕОЋОеОћ 0 ОЋОбОЋОЉОе ОЕОЋОеОћ ОЕОЋОеОћ, ОЋОЉОеОњОб ОЕОљОфОћ ОаОфОДОю ОЉОДОеОЎОљОћ ОюОцОЋОаОДОдОЎОћ, ОфОЋОќОЎОќ ОљОф ОћОљОдОЉОб ОЕОюОџ ОюОЕОЋОеОћ ОћОъОфОљОЎОъОћ, ОЋОЉОеОњОб ОЕОћОцОЋОаОДОдОЎОћ ОЌОЋОќОеОф, ОфОЋОќОЎОќ ОљОф ОћОљОдОЉОб ОЕОюОџ ОюОъОДОЋОЮ ОЕОћОЎОЎОфОћ ОъОДОЋОЊОЮ.

    • ОћОњОЊОеОфОЎ ОљОф ОћОцОЋОаОДОдОЎОћ ОЕОюОЎ ОЉОДОЋОЉОЦ, ОЋОљОаОЎ ОДОЋОеОљ ОюОћ ОЉОДОЋОЉОЦ ОЋОъОЎОЕОЋОЮ ОъОћ ОћОЎОљ ОюОљ ОаОДОеОљОф, ОцОЎОЎОфОЋОЪ ОюОљ ОЎОЋОЊОб ОбОю ОљОЎОќОћ ОцОЋОаОДОдОЎОћ ОљОаОЎ ОъОЊОЉОе. ОюОъОћ?
      ОАОЎОЏОЋОЎ ОћОЏОЎ ОАОЉОЎОе ОЕОДОеОљОф ОюОцОЋОаОДОдОЎОћ ОЕОћОњОЊОеОф ОљОЌОеОЎ, ОћОДОеОЎОљОћ ОЕОюОћ. ОЉОЊОЎОЋОД ОЏОъОЋ ОЕОљОЎОЎ ОљОцОЕОе ОюОћОЕОфОъОЕ ОЉОъОЕОфОаОћ ОЕОбОЊОЎОЎОЪ ОюОљ ОћОњОЊОеОфОЮ, ОљОЋОфОЋ ОћОЊОЉОе ОДОЋОеОћ ОбОЮ ОцОЋОаОДОдОЎОЋОф. ОДОЋОЊОЮ ОфОњОЊОЎОеОЋ ОљОф ОћОцОЋОаОДОдОЎОћ, ОЋОеОД ОЉОЕОЋОеОЋОф ОљОЌОеОЏОџ ОфОЕОфОъОЕОЋ ОЉОцОЋОаОДОдОЎОћ, ОюОљ ОћОцОЋОџ.

    • ОбОЕОЎОфОЎ print, ОќОћ ОљОЋОъОе ОЕОћОцОЋОаОДОдОЎОћ ОъОЌОќОЎОеОћ ОъОЕОћОЋ?
      ОюОљ, print ОъОЊОцОЎОА ОюОъОАОџ, ОюОљ "ОъОЌОќОЎОе" ОъОЕОћОЋ. ОЏОЕОљОаОЌОаОЋ ОљОЋОъОеОЎОЮ "ОъОЌОќОЎОе" ОЉОДОЋОаОўОДОАОў ОЕОю ОцОЋОаОДОдОЎОЋОф, ОћОЏОЋОЋОаОћ ОћОЎОљ ОЕОћОцОЋОаОДОдОЎОћ ОбОЋОЕОћ return ОюОбОеОџ ОЏОюОЕОћОЋ ОюОЕОЋОеОћ ОЕОћОеОЎОдОћ ОљОЋОфОћ.

    • ОбОЕОЎОфОЎ input, ОќОћ ОљОЋОъОе ОЕОћОцОЋОаОДОдОЎОћ ОъОДОЉОюОф ОъОЕОћОЋ?
      ОюОљ, input ОъОЉОДОЕ ОДОюОў ОъОћОъОЕОфОъОЕ, ОюОљ "ОъОДОЉОю" ОъОЕОћОЋ. ОЏОЕОљОаОЌОаОЋ ОљОЋОъОеОЎОЮ "ОъОДОЉОю" ОЉОДОЋОаОўОДОАОў ОЕОю ОцОЋОаОДОдОЎОЋОф, ОћОЏОЋОЋОаОћ ОћОЎОљ ОЕОћОцОЋОаОДОдОЎОћ ОъОДОЉОюОф ОцОеОъОўОеОЎОЮ ОъОћОЕОЋОеОћ ОЕОћОеОЎОдОћ ОљОЋОфОћ.

    • ОюОъОћ ОЏОЕОљОаОЎ ОъОњОЊОЎОе ОъОЕОфОаОЎОЮ ОЉОцОЋОаОДОдОЎОћ, ОљОцОЕОе ОюОњОЕОф ОљОюОЎОћОЮ ОеОД ОЉОљОЋОфОћ ОцОЋОаОДОдОЎОћ? ОюОъОћ ОћОъОЕОфОаОЎОЮ ОюОљ ОЕОЎОфОЋОцОЎОЎОЮ ОЉОЎОЪ ОцОЋОаОДОдОЎОЋОф ОфОъОЎОЊ?
      ОЏОю ОцОЋОаОДОдОЎОћ ОљОъОЋОеОћ ОюОЉОдОб ОцОбОЋОюОћ ОЉОаОцОеОЊ, ОћОцОбОЋОюОћ ОљОъОЋОеОћ ОюОћОЎОЋОф ОюОљ ОДОЕОЋОеОћ ОюОцОЋОаОДОдОЎОЋОф ОЕОДОеОљОЋ ОюОћ. ОљОЮ ОъОЕОфОаОЎОЮ ОћОЎОЋ ОЕОЎОфОЋОцОЎОЎОЮ, ОќОћ ОћОЎОћ ОЎОЋОдОе ОћОъОЋОЪ ОЉОљОњОЎОЮ - ОЊОъОЎОЎОаОЋ ОЋОЎОЏОюОфОЮ ОюОЏОфОЋОЉ ОцОЋОаОДОдОЎОћ ОЕОъОЕОаОћ ОъОЕОфОаОЎОЮ ОЕОю ОцОЋОаОДОдОЎОћ ОљОЌОеОф, ОќОћ ОЎОЏОЋОю ОюОЎОдОЋОе ОћОъОЋОЪ ОЊОЉОеОЎОЮ ОюОљ ОдОцОЋОЎОЎОЮ. ОЉОњОюОю ОќОћ ОћОЊОеОџ ОћОЎОЌОЎОЊОћ ОЕОЉОћ ОцОЋОаОДОдОЎОЋОф ОљОъОЋОеОЋОф ОюОфОДОЕОе ОбОЮ ОцОЋОаОДОдОЎОЋОф ОљОЌОеОЋОф ОќОћ ОЉОљОъОдОбОЋОф ОћОЌОќОеОћ ОЕОю ОбОеОЏОЎОЮ ОъОцОЋОаОДОдОЎОћ, ОЋОћОбОЉОеОф ОцОеОъОўОеОЎОЮ ОюОцОЋОаОДОдОЎОћ.

    • ОљОЮ ОљОаОЎ ОЎОЏОЋОю ОюОЏОфОЋОЉ ОДОЋОЊ ОЉОюОЎ ОцОЋОаОДОдОЎОЋОф, ОюОъОћ ОЕОљОаОЎ ОЉОфОЏОюОА ОљОЕОфОъОЕ ОЉОќОћ?
      ОЎОЕ ОћОъОЋОЪ ОАОЎОЉОЋОф, ОћОаОћ ОљОЌОф ОћОАОЎОЉОЋОф ОћОљОћОЋОЉОЋОф ОбОюОЎОЎ: ОцОЋОаОДОдОЎОЋОф ОбОЋОќОеОЋОф ОюОаОЋ ОюОЏОфОЋОЉ ОфОЋОЏОаОЋОф ОцОЕОЋОўОЋОф ОЎОЋОфОе, ОЏОЕОљОаОЌОаОЋ ОъОЌОюОДОЎОЮ ОљОф ОћОфОЋОЏОаОћ ОЕОюОаОЋ ОюОфОцОДОЎОЊОЎОЮ - ОюОцОЋОаОДОдОЎОЋОф, ОЋОЉОъОЎОЋОЌОЊ ОЏОљОЕОе ОљОаОЌОаОЋ ОаОЋОфОаОЎОЮ ОЕОЮ ОюОЏОю ОцОбОЋОюОћ ОЉОфОЋОЏОаОћ (ОЕОЮ ОюОцОЋОаОДОдОЎОћ). ОќОћ ОЎОбОќОЋОе ОюОаОЋ ОюОЌОЕОЋОЉ ОбОю ОћОфОЋОЏОаОћ ОЕОюОаОЋ ОЉОдОЋОеОћ ОцОЕОЋОўОћ ОЎОЋОфОе, ОќОћ ОЎОфОЪ ОюОаОЋ ОљОф ОћОљОцОЕОеОЋОф ОюОћОЉОЎОЪ ОфОЋОЏОаОЋОф ОъОАОЋОЉОЏОЋОф ОЋОљОеОЋОЏОЋОф ОЕОю ОљОЌОеОЎОЮ, ОЋОњОЮ ОЕОюОаОЋ. ОаОАОЋ ОюОДОЌОф ОфОЋОЏОаОћ ОЕОљОфОЮ ОеОЋОдОЎОЮ ОюОЏОфОЋОЉ, ОюОљОЎОќОћ ОфОцОДОЎОЊОЎОЮ ОцОЕОЋОўОЎОЮ ОЋОъОЋОеОЏОЉОЎОЮ ОћОЎОЎОфОЮ ОъОЌОюОДОЎОЮ ОљОф ОћОфОЋОЏОаОћ?
