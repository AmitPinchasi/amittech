    8.5 - ОЉОДОеОф ОњОеОАОљОЋОф - 2 ОъОеОњ ОћОеОдОљОћ

    ОњОЎОў ОЉОљОАОћ¶

    • ОЉОцОеОД ОћОДОЋОЊОЮ ОЎОдОеОаОЋ ОДОЋОъОЎОЎОў ОбОЮ ОћОћОЋОЊОбОћ ОћОЉОљОћ: "some changes..", ОќОЋОћОЎ ОћОЋОЊОбОћ ОаОЋОеОљОЎОф. ОљОфОћ ОеОЋОдОћ ОюОЏОфОЋОЉ ОћОЋОЊОбОЋОф ОЕОъОфОЏОаОфОЎОЮ ОљОЌОеОЎОЮ ОЎОЋОЏОюОЋ ОюОћОЉОЎОЪ ОљОф ОћОЕОЎОаОЋОЎОЎОЮ ОЕОбОЕОЎОф ОЉОДОЋОЊ ОъОЉОюОЎ ОюОДОеОЋОљ ОљОЋОфОЋ, ОљОюОћ ОеОД ОъОюОДОеОЋОљ ОљОф ОћОћОЋОЊОбОћ. ОЉОаОЋОАОБ ОљОЮ ОфОЌОќОЋОе ОюОДОЋОъОЎОЎОў ОЕОбОЕОЎОф ОюОцОаОЎ ОЕОаОфОЎОЎОЮ ОЋОћОћОЋОЊОбОћ ОЕОюОћ ОфОћОЎОћ ОЊОЋОъОћ, ОЎОћОЎОћ ОюОџ ОДОЕОћ ОюОћОЉОЎОЪ ОъОћ ОќОћ ОћОДОЋОъОЎОЎОў ОћОќОћ. ОЉОЊОЎОЋОД ОЉОЕОЉОЎОю ОќОћ ОЌОЕОЋОЉ ОъОљОЋОЊ ОюОфОф ОЕОъОЋОф ОўОЋОЉОЎОЮ ОюОДОЋОъОЎОўОЎОЮ ОЕОюОаОЋ, ОЎОЕ ОЊОеОџ ОъОљОЋОЊ ОДОюОћ ОЉОњОЎОў ОюОЕОаОЋОф ОљОф ОћОћОЋОЊОбОћ ОЕОю ОћОДОЋОъОЎОў ОћОљОЌОеОЋОЪ, ОћОеОЎОдОЋ ОљОф ОћОцОДОЋОЊОћ ОћОЉОљОћ:
      git commit --amend -m "Reformated main.py"

    • ОъОбОЋОюОћ, ОЋОъОћ ОљОЮ ОаОеОдОћ ОюОЉОўОю ОюОњОъОеОЎОЎ ОљОф ОћОДОЋОъОЎОў? ОъОћ ОаОЋОЏОю ОюОбОЕОЋОф?
    • ОаОЋОЏОю ОюОћОЕОфОъОЕ ОЉОцОДОЋОЊОћ git revert ОЕОфОдОЋОе ОДОЋОъОЎОЎОў ОЌОЊОЕ ОЕОъОЋОЌОД ОљОф ОЏОю ОћОЕОЎОаОЋОЎОЎОЮ ОћОДОЋОъОЎОЎОў ОћОљОЌОеОЋОЪ, ОћОеОЎОдОЋ:
      git revert main

    • ОаОЎОфОЪ ОюОеОљОЋОф ОЕОаОЋОдОе ОДОЋОъОЎОЎОў ОЌОЊОЕ Оюmain, ОЕОцОЕОЋОў ОъОЋОЌОД ОљОф ОћОЕОЎОаОЋОЎОЎОЮ ОЕОћОДОЋОъОЎОЎОў ОћОДОЋОЊОЮ ОбОЕОћ

    ОњОЎОў ОъОеОњ' - git merge¶

    ОЉОњОЎОў ОЉОЕОюОЉ ОъОАОЋОЎОЎОЮ ОаОеОдОћ ОюОъОќОњ ОЉОЎОЪ ОћОбОаОцОЎОЮ ОЕОюОаОЋ, ОюОъОЕОю ОљОЌОеОЎ ОЕОЏОфОЉОаОЋ ОДОЋОЊ ОЉdev ОЋОЉОЊОДОаОЋ ОљОЋОфОЋ ОЋОљОаОЌОаОЋ ОЌОЋОЕОЉОЎОЮ ОЕОћОЋОљ ОъОЋОЏОЪ ОюОцОеОЋОЊОДОЕОЪ, ОаОЋОЏОю ОюОъОќОњ ОљОЋОфОЋ Оюmain.
    ОЎОЕ ОЕОаОЎ ОЊОеОЏОЎОЮ ОбОЎОДОеОЎОЋОф ОюОъОќОњ ОЉОЎОЪ ОбОаОцОЎОЮ, ОљОЌОф ОъОћОЮ ОћОЎОљ git merge
    - ОЏОЊОЎ ОюОъОќОњ ОЉОЎОЪ dev Оюmain ОбОЮ git merge ОаОеОЎОЦ ОљОф ОћОцОДОЋОЊОЋОф ОћОЉОљОЋОф:

    git checkout dev
git merge main
git commit -m "dev merged with main!"


    - ОаОЋОЏОю ОюОеОљОЋОф ОЕОаОЋОдОе ОДОЋОъОЎОЎОў ОЌОЊОЕ ОЉmain ОбОЮ ОћОЕОЎОаОЋОЎОЎОЮ ОЕОю ОћОбОаОБ dev!
    - ОаОЋОЏОю ОњОЮ ОюОћОЕОфОъОЕ ОЉgit pull ОЏОЊОЎ ОюОљОЌОЊ ОЉОЎОЪ ОћОбОаОцОЎОЮ, ОДОЋОЊОЮ ОЏОю ОаОЋОАОЎОБ ОбОЋОЊ ОЕОЎОаОЋОЎОЎОЮ Оюdev
    git checkout dev
# Adding changes..
git status
git add .
git commit -m "Added doc string to functions"

    - ОбОЏОЕОЎОЋ ОаОъОќОњ ОљОф ОћОбОаОцОЎОЮ ОбОЮ git pull, ОаОбОЕОћ ОќОљОф ОЉОљОъОдОбОЋОф ОћОцОДОЋОЊОЋОф ОћОЉОљОЋОф:
    git checkout main
git pull dev
git commit -m "pulled dev to main"


    - ОЕОЎОъОЋ ОюОЉ, git merge ОъОъОќОњ ОљОф Оћbranch ОћОаОЋОЏОЌОЎ ОбОЮ Оћbranch ОЕОдОЎОЎОаОЋ ОЋ - git pull ОъОЋОЕОџ ОљОф Оћbranch ОЕОдОЎОЎОаОЋ Оюbranch ОћОаОЋОЏОЌОЎ

    ОњОЎОў ОДОЋОаОцОюОЎОДОў - git conflict¶

    • ОћОъОЋОЪ ОцОбОъОЎОЮ ОЏОЕОљОаОЌОаОЋ ОбОЋОЕОЎОЮ ОъОЎОеОњ'ОЋОњ' ОЉОЎОЪ ОбОаОцОЎОЮ, ОДОЋОаОцОюОЎОДОўОЎОЮ ОЎОЏОЋОюОЎОЮ ОюОДОеОЋОф.
    • ОњОЎОў ОДОЋОаОцОюОЎОДОў ОДОЋОеОћ ОЏОљОЕОе ОљОЋОфОЋ ОДОЋОЉОЦ ОћОЕОфОаОћ ОЉОЕОаОЎ ОбОаОцОЎОЮ ОЕОЋОаОЎОЮ, ОљОќ ОњОЎОў ОюОљ ОъОдОюОЎОЌ ОюОъОеОњ'Оњ' ОбОаОцОЎОЮ.
    • ОфОеОљОЋ ОљОф ОћОЊОЋОњОъОћ ОћОЉОљОћ:

      # Adding main.py
git add main.py
git commit -m "added main.py"

# Changing main.py in dev branch
git branch dev
git checkout dev
git add main.py
git commit -m "changed main.py"


# Changing main.py in main
git checkout main
git add main.py
git commit -m "changed main.py"

# try merge-ing dev to main
git checkout dev
git merge main

    ОњОЎОў ОцОфОеОЋОЪ - cherry pick¶

    • ОљОќ ОљОЎОџ ОцОЋОфОеОЎОЮ ОДОЋОаОцОюОЎОДОў? ОќОЏОеОЋ, ОДОЋОаОцОюОЎОДОў ОДОЋОеОћ ОЏОљОЕОе ОљОЋОфОЋ ОДОЋОЉОЦ ОбОЉОе ОЏОъОћ ОЕОЎОаОЋОЎОЎОЮ ОЉОЕОаОЎ branch-ОЎОЮ ОЋОљОаОЌОаОЋ ОъОаОАОЎОЮ ОюОљОЌОЊ ОЉОаОЎОћОЮ, ОљОќ ОњОЎОў ОъОљОцОЕОе ОюОаОЋ ОюОцОфОЋОе ОДОЋОцОюОЎОДОўОЎОЮ ОЉОЊОеОџ ОЉОљОћ:
    • ОћОеОЎОдОЋ git status ОЏОЊОЎ ОюОеОљОЋОф ОљОф ОћОДОЉОдОЎОЮ ОЕОЎОЕ ОюОћОЮ ОДОЋОаОцОюОЎОДОў.
    • ОбОЏОЕОЎОЋ ОцОфОЌОЋ ОљОф ОћОДОЉОдОЎОЮ ОЕОЎОЕ ОюОћОЮ conflict ОЋОљОфОЮ ОфОеОљОЋ ОЕОЎОЕ ОћОъОЋОЪ ОАОЎОъОЋОаОЎОЎОЮ ОбОю ОћОДОЉОдОЎОЮ ОЕОюОћОЮ ОЕОю <<<<<<<, =======, ОЋ - >>>>>>> ОћОаОћ ОЊОЋОњОъОћ ОюОДОЋОЉОЦ ОЏОќОћ:
      <<<<<<< HEAD
Changes in dev branch
=======
Conflicting changes in main branch
>>>>>>> main
    • ОљОцОЕОе ОюОеОљОЋОф ОЕОЉОДОЋОЉОЦ ОЎОЕ ОљОф ОћОЕОЎОаОЋОЎОЎОЮ ОЕОю ОЕОаОЎ Оћbranch-ОЎОЮ ОЋОњОЎОў ОъОљОцОЕОе ОюОаОЋ ОюОЉОЌОЋОе ОљОЎОќОћ ОЕОЎОаОЋОЎ ОљОаОЌОаОЋ ОеОЋОдОЎОЮ ОЕОЎОћОЎОћ ОюОДОЋОЉОЦ ОћОАОЋОцОЎ, ОъОЌОДОЋ ОЏОю ОљОф ОЏОю ОћОЕОЋОеОЋОф ОЋОћОЕОЎОаОЋОЎОЎОЮ ОЕОљОфОЮ ОюОљ ОеОЋОдОЎОЮ ОЕОЎОћОЎОћ ОюОДОЋОЉОЦ ОћОАОЋОцОЎ ОЋОЏОЕОљОфОЮ ОъОАОЎОЎОъОЎОЮ ОюОбОеОЋОџ ОљОф ОЏОю ОћОДОЉОдОЎОЮ ОћОеОЎОдОЋ ОљОф ОћОцОДОЋОЊОћ:
      git merge --continue
    • ОљОЮ Оћmerge ОеОЦ ОбОЏОЕОЎОЋ ОЉОћОдОюОЌОћ ОаОбОЕОћ commit ОюОЕОЎОаОЋОЎОЎОЮ: 
      git commit -m "merged dev sucessfully"

    ОњОЎОў diff¶

    • ОћОъОЋОЪ ОцОбОъОЎОЮ ОЉОњОЎОў ОаОеОдОћ ОюОеОљОЋОф ОљОф ОћОЕОЎОаОЋОЎОЎОЮ ОЕОДОеОЋ ОЉОЎОЪ commit-ОЎОЮ, ОбОаОцОЎОЮ ОЋОњОЮ ОЕОЎОаОЋОЎОЎОЮ ОЕОљОаОЌОаОЋ ОбОЋОЕОЎОЮ ОбОю ОћОъОЌОЕОЉ (ОюОцОаОЎ ОДОЋОъОЎОЎОў). ОаОЋОЏОю ОюОбОЕОЋОф ОќОљОф ОЉОљОъОдОбОЋОф ОћОцОДОЋОЊОћ git diff, ОаОеОљОћ ОЊОЋОњОъОљОЋОф:
    • ОћОеОЎОдОЋ ОљОф ОћОцОДОЋОЊОћ ОћОЉОљОћ ОЏОЊОЎ ОюОеОљОЋОф ОљОф ОћОДОЋОЊ ОЕОЕОаОЎОаОЋ ОъОљОќ ОћОДОЋОъОЎОЎОў ОћОљОЌОеОЋОЪ: 
      git diff
    • ОћОеОЎОдОЋ ОљОф ОћОцОДОЋОЊОћ ОћОЉОљОћ ОЏОЊОЎ ОюОеОљОЋОф ОљОф ОћОЕОЎОаОЋОЎОЎОЮ ОЕОю ОћОДОЉОдОЎОЮ ОЕОбОЕОЎОаОЋ ОюОћОЮ git add:
      git diff --staged
    • ОћОеОЎОдОЋ ОљОф ОћОцОДОЋОЊОћ ОћОЉОљОћ ОЏОЊОЎ ОюОеОљОЋОф ОљОф ОћОћОЉОЊОюОЎОЮ ОЉОЎОЪ ОЕОаОЎ ОДОЋОъОЎОЎОўОЎОЮ:
      git diff commit1 commit2
    • ОћОеОЎОдОЋ ОљОф ОћОцОДОЋОЊОћ ОћОЉОљОћ ОЏОЊОЎ ОюОеОљОЋОф ОљОф ОћОћОЉОЊОюОЎОЮ ОЉОЎОЪ ОЕОаОЎ ОбОаОцОЎОЮ:
      git diff branch1 branch2

    ОњОЎОў ОеОЎ-ОЉОЎОЎОА - git rebase¶

    • ОњОЎОў ОеОЎ-ОЉОЎОЎОА ОќОћ ОбОЋОЊ ОЊОеОџ ОюОъОќОњ ОЉОЎОЪ ОбОаОцОЎОЮ ОЏОъОЋ git merge ОЋ - git rebase. ОъОљОЌОЋОеОЎ ОћОДОюОбОЎОЮ ОќОћ ОбОЋОЉОЊ ОЉОдОЋОеОћ ОДОдОф ОЕОЋОаОћ.
    • ОЏОљОЕОе ОљОаОЌОаОЋ ОбОЋОЕОЎОЮ git merge, ОњОЎОў ОЎОЋОдОе ОДОЋОъОЎОЎОў ОЌОЊОЕ ОЉbranch ОЕОъОЏОЎОю ОљОф ОЏОю ОћОЕОЎОаОЋОЎОЎОЮ ОЕОю Оћbranch ОЕОљОЋОфОЋ ОљОаОЌОаОЋ ОъОъОќОњОЎОЮ. ОћОЏОю ОЉОДОЋОъОЎОЎОў ОљОЌОЊ.
    • ОюОбОЋОъОф ОќОљОф ОЕОљОаОЌОаОЋ ОбОЋОЕОЎОЮ git rebase, ОњОЎОў ОЎОЋОдОе ОДОЋОъОЎОЎОў ОЌОЊОЕ ОЉОЕОЉОЎОю ОЏОю ОДОЋОъОЎОЎОў ОЕОаОЋОдОе ОЉbranch ОЕОљОЋОфОЋ ОљОаОЌОаОЋ ОъОъОќОњОЎОЮ, ОЏОю ОЕОЎОЏОЋОюОЎОЮ ОюОћОЎОЋОЋОдОе ОъОАОцОе ОДОЋОъОЎОўОЎОЮ ОЌОЊОЕОЎОЮ Оюbranch ОЕОљОюОЎОЋ ОљОаОЌОаОЋ ОъОъОќОњОЎОЮ.
      ОЊОЋОњОъОћ:
      git add .
git commit -m "init project"
git branch dev
git checkout dev
# A
git add .
git commit -m "A"
# B
git add .
git commit -m "B"
git rebase main

    • ОЏОљОЕОе ОљОаОЌОаОЋ ОъОеОЎОдОЎОЮ ОљОф ОћОцОДОЋОЊОћ git init ОаОЋОдОе ОюОаОЋ ОфОДОЎОЎОћ ОЌОЊОЕОћ ОЉОцОеОЋОЎОДОў ОЉОЕОЮ .git, ОќОЋОћОЎ ОћОфОДОЎОЎОћ ОЕОю ОњОЎОў ОЕОъОЏОЎОюОћ ОљОф ОЏОю ОћОЕОЎОаОЋОЎОЎОЮ, ОћОбОаОцОЎОЮ, ОЋОљОф ОЏОю ОћОњОЎОў ОЕОЎОдОеОаОЋ. - ОљОю ОфОъОЌОДОЋ ОљОЋОфОћ!
