    עמית טק - קורסי סייבר ומחשבים בחינם.

    חומרי לימוד חינמים בנושאי תכנות, IT, סייבר, מחקר ו-low level - בעברית, עם המון תרגול.

    הצטרפו לקהילה שלנו בדיסקורד

    מה נלמד היום?

    תכנות

    תכנות בסיסי קורס

    פייתון מהבסיס ועד נושאים מתקדמים, תכנות מונחה עצמים, מודולים, אלגוריתמיקה וקוד נקי
    פיתוח עם AI - AI-Assisted Development קורס

    כל כלי ה-AI שמתכנת צריך להכיר, בדגש על פיתוח אג'נטי עם Claude Code - הנדסת פרומפט, זרימות עבודה, ניהול הקשר, MCP וסוכנים, ובנייה מעל ה-API
    פיתוח צד שרת - Backend קורס

    פיתוח API עם FastAPI, מסדי נתונים, DevOps, AWS, ופיתוח JavaScript עם Node.js
    פיתוח צד לקוח - Frontend קורס

    פיתוח frontend עם React ו-Next.js
    ארכיטקטורת תוכנה קורס

    SOLID, תבניות עיצוב, Clean Architecture, מיקרוסרביסים, ועיצוב מערכות גדולות בקנה-מידה

    רשתות ומערכות הפעלה

    ווינדוס בסיסי קורס

    באצ', פאוורשל, ניהול מערכת, כלי Sysinternals
    לינוקס בסיסי קורס

    טרמינל, באש, הרשאות, סקריפטים, שירותים וניהול מערכת לינוקס
    רשתות קורס

    מודל OSI, פרוטוקולים, שכבת האפליקציה, האינטרנט, רשתות מנוהלות ואבטחת רשת

    אבטחת מידע וסייבר

    פריצת אתרים קורס

    פיתוח אתרים, חולשות צד לקוח וצד שרת, הגנות והתמודדות
    בדיקת חדירות - Penetration Testing קורס

    תקיפה חיצונית ופנימית, הסלמת הרשאות, תקיפת דומיין מתקדמת
    פריצת אתרים מתקדם קורס

    חולשות מתקדמות, התמודדות מול הגנות

    מחקר ו-Low Level

    ליבת המחשב - Low Level קורס

    ארגון המחשב, אסמבלי 16 ו-32 ביט, שפת C, ספריית libc, הקרנל של לינוקס, ארכיטקטורת מעבדים, ופיתוח מערכת הפעלה
    מחקר חולשות - Binary Exploitation קורס

    ניצול בינארי מאפס - באגים לוגיים, גלישות חוצץ, shellcode, עקיפת NX ו-ROP, מחרוזות פורמט, ניצול heap, ניצול קרנל, ומציאת חולשות אמיתיות עם fuzzing
    תכנות GPU ו-CUDA קורס

    ארכיטקטורת GPU לעומק, מודל CUDA, זיכרון ו-shared memory, PTX ו-SASS, ספריות ביצועים, ומודל Roofline עם אופטימיזציה ופרופיילינג של kernels אמיתיים
